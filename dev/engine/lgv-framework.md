# 城投债信用分析框架 — LGV（Local Government Vehicle）

**版本**: v0.7.0-release | **日期**: 2026-07-10
**定位**: 第八个"准行业"分析框架 · 轨道A基本面分析的特殊行业模板 · 与external-support-framework.md联动
**核心问题**: 城投平台陷入困境时，地方政府有没有能力+意愿救助？平台自身有没有独立存活能力？

---

## 一、为什么城投债需要独立分析框架

### 1.1 市场体量与现有缺口

城投债是中国信用债市场最大的单一板块，占比超过35%，但当前引擎的7个行业金字塔全部基于产业债逻辑构建，无法直接应用于城投分析。

| 指标 | 数据 | 对引擎的影响 |
|------|------|-------------|
| 城投债存量规模 | ~15万亿元（2026年中） | 覆盖35%+信用债市场——这是当前引擎的最大盲区 |
| 城投平台数量 | 约3,000家（其中约2,000家有存续债券） | 发行人数量远超任何一个产业行业 |
| 城投债交易量 | 日均成交额占信用债总成交的40%以上 | 市场参与者的核心交易品种 |
| 从业者审计结论 | "城投债缺失是致命短板" | 必须作为P0优先级填补 |

### 1.2 城投债 vs 产业债：两种完全不同的分析范式

城投债的信用基础不是企业自身的经营现金流——城投平台没有稳定的经营性收入和利润。城投的信用来自于"地方政府信用+平台对政府的战略价值"。这是两种完全不同的分析范式。

| 维度 | 产业债 | 城投债 |
|------|--------|--------|
| **信用基础** | 企业自身经营现金流 | 地方政府信用 + 平台自身现金流（注意：大多数平台经营性现金流为负） |
| **核心分析对象** | 企业的行业位置、竞争力、财务状况 | 地方政府的财政实力、债务负担、支持意愿 + 平台的职能定位、资产质量 |
| **违约逻辑** | 经营失败→现金流枯竭→违约 | 政府不再兜底 + 平台自身无法造血→违约 |
| **偿债来源** | 经营活动产生的现金流 | 政府注资/资产处置/土地出让收入/再融资（借新还旧） |
| **关键信号** | 财务指标恶化、市场份额流失、技术路线淘汰 | 资产无偿划转、政府支持文件撤回、融资政策突变、区域传染 |
| **数据可得性** | 企业财报可获取，行业数据较规范 | 省级城投披露较充分，区县级城投数据严重不足——需诚实标注此差距 |
| **分析确定性** | 相对可量化（收入/成本/现金流可审计） | 定性判断 > 定量计算——政府支持意愿无法量化，是最不可测的变量 |

### 1.3 核心悖论：城投分析必须在不确定性中做判断

城投分析面临一个根本性的矛盾：

> **城投的信用质量高度依赖政府支持，但政府支持意愿恰恰是最不可量化的变量。** 产业债的财务比率（如利息覆盖倍数、资产负债率）在城投分析中意义有限——因为城投的偿债能力不来自经营，而是来自政府协调和再融资。这意味着城投分析天然需要更多的定性判断和信号拼图，而非精确计算。

**诚实声明**：本框架旨在建立一个系统化的分析结构，但无法消除城投分析中固有的不确定性。以下所有评分/分级仅供参考，不构成精确预测。

## 二、城投分析双轨框架：总体设计

### 2.1 为什么是"双轨"——借鉴外部支持框架的二元逻辑

外部支持框架（external-support-framework.md）提出了"支持能力 vs 支持意愿"的二元分析。城投债是该框架最典型、最复杂、最核心的应用场景。

城投分析将政府端分析作为**第一轨**（权重50-60%），平台端分析作为**第二轨**（权重40-50%）。两轨的权重可根据平台的具体特征调整——对纯公益性平台（如市政建设类），政府轨权重应占60-70%；对经营性平台（如公用事业运营类），平台轨权重可上调至50-60%。

```
城投信用质量    =    政府信用评估    +    平台自身信用评估
（综合判断）           （50-60%）           （40-50%）
                          │                      │
                    ┌─────┴──────┐        ┌─────┴──────┐
                    │ 财政实力    │        │ 职能定位    │
                    │ 债务负担    │        │ 资产质量    │
                    │ 支持意愿    │        │ 债务结构    │
                    │ 资源调动    │        │ 现金流分析  │
                    │              │        │ 对外担保    │
                    └─────────────┘        └─────────────┘
                          │                      │
                          └──────────┬───────────┘
                                     ▼
                          城投信用质量综合判定
                          （分级 + 预警 + 展望）
```

### 2.2 两轨的关系：不是简单相加，而是"政府定上限，平台定下限"

```
最终信用等级上限 = 地方政府信用等级（平台不能高于政府本身）
最终信用等级下限 = 平台独立信用质量（如果政府不救助，平台能撑多久？）

实际信用等级落在两个边界之间的某个位置，取决于政府支持意愿的强度。
```

**核心逻辑**：
- 政府信用评估决定**天花板**——再好的平台，如果所在政府财力薄弱，信用等级也上不去
- 平台自身信用评估决定**地板**——政府虽强，但如果平台自身资产质量差、债务结构差，信用等级也会受影响
- 支持意愿决定**两轨之间的实际落点**——政府有意愿救助时，信用等级靠近政府端；意愿弱时，靠近平台端

## 三、第一轨：地方政府信用评估（权重50-60%）

### 3.1 评估框架：四维模型

地方政府信用评估的四维模型与external-support-framework.md 4.1节一致，但增加城投专属的细化指标和阈值。

| 维度 | 权重 | 核心指标 | 城投分析中的特殊关注 |
|------|------|---------|-------------------|
| **F1 财政实力** | 30% | 一般公共预算收入及增速、财政自给率、人均一般公共预算收入、税收收入占比 | 土地出让收入占比——土地财政依赖度越高的区域，地产下行周期中财政脆弱性越高 |
| **F2 债务负担** | 30% | 政府显性债务率、隐性债务估算、综合债务率、城投有息负债/综合财力 | **这是城投分析的核心定量指标**——城投有息负债/综合财力比率直接反映地方政府对城投平台的"实际负担" |
| **F3 经济基础** | 20% | GDP总量及增速、产业结构、人均GDP、人口趋势（净流入/流出） | 产业结构中的房地产依赖度——土地出让收入是城投偿债的重要资金来源 |
| **F4 资源调动能力** | 20% | 上级转移支付收入及依赖度、政府性基金收入（含土地出让）、金融机构本外币存款余额、区域内上市公司/国企资产规模 | 金融资源丰富度影响"协调银行续贷"的能力——区域存款余额是最硬指标 |

### 3.2 省/地市/区县三级财政数据辨析

**这是城投分析中最容易出错的地方**——不同层级政府使用不同的财政数据口径。

| 层级 | 应使用的数据口径 | 数据来源 | 常见错误 |
|------|-----------------|---------|---------|
| **省级** | 省本级财政数据（省级城投由省财政厅直接支持） | 财政年鉴—省本级栏目 | 错误使用全省汇总数据（包含下级政府），高估省级政府的直接支持能力 |
| **地市级** | 市本级财政数据（市本级一般公共预算收入） | 各市财政局预决算报告 | 错误使用全市汇总数据（含下辖区县），高估地市级政府的直接支持能力 |
| **区县级** | 区县级财政数据 | 区县财政局预决算报告（部分区县不公开） | 数据不可获取时，使用省级汇总数据中该县的占比估算——精度低，需标注"估算" |
| **开发区/新区** | 管委会财政数据 | 开发区财政统计（规范性参差不齐） | 开发区的财政数据独立性差异大——部分区域与所在行政区合署办公，数据不可分 |

**诚实标注**：区县级城投的财政数据质量是城投分析中最大的数据瓶颈。县级预决算报告的公开率约70%，且公开时间普遍滞后（财政年度结束后6-12个月）。分析区县级城投时，财政数据的滞后性和不完整性应作为风险因素在分析结论中明确标注。

### 3.3 支持意愿评估（城投专属）

支持意愿是城投分析中最关键也最不可量化的维度。与external-support-framework.md第五章通用框架不同，城投的分析要额外关注以下专属信号。

| 评估维度 | 强支持意愿信号 | 弱支持意愿信号 | 数据来源 |
|---------|---------------|---------------|---------|
| **平台职能定位** | 承担区域内垄断性公共职能（供水/供气/公共交通/保障房/土地储备） | 从事竞争性业务（房地产开发/贸易/酒店经营/金融投资） | 募集说明书/企业官网/政府授权文件 |
| **平台不可替代性** | 区域内唯一/主要平台，承担重点项目建设 | 同层级存在3家以上同类平台，或平台正在被整合/合并 | 评级报告/Wind平台分类/Wind数据中该区域城投数量 |
| **历史支持记录** | 近3年有政府注资/资产划入/财政补贴/隐性担保记录 | 近3年无任何形式支持，或政府曾有拖欠应拨付款项行为 | 企业年报——资本公积变动/其他收益/营业外收入 |
| **违约先例（反信号）** | 本地区/本省无城投/国企违约先例 | 本省已有城投非标违约/债券技术性违约/展期案例 | 新闻数据库/Wind违约统计/政信产品违约信息 |
| **政府隐性担保程度** | 重点项目纳入省级以上规划，偿债资金涉及财政预算安排 | 政府官员公开表态"平台自负盈亏"，或出台限制平台融资的政策 | 政府工作报告/金融监管局文件/公开发言记录 |
| **人员/治理关联度** | 平台高管由政府任命/兼任，与财政/国资部门人员交叉任职 | 平台市场化招聘高管，政府人员逐步退出董事会 | 企查查高管信息/募集说明书管理层披露 |
| **资产安全** | 核心资产（土地/股权/收费权）在平台名下，未被划转 | 近1年内核心资产被无偿划转至其他平台/国资运营公司 | 企业公告/工商变更记录/新闻检索 |

**核心判断**：政府支持意愿的评估不是静态评分，而是动态信号监测。一次资产划转、一次官员表态、一次政策变化都可能彻底改变支持意愿。**城投分析必须建立"信号跟踪清单"——不能只做一次性评估。**

### 3.4 省级政府支持能力分级（城投视角）

基于公开财政数据和城投领域的实际经验，以下分级供参考。**评级分析日的时间锚点为2026年7月**——每年应随财政数据更新重新评估。

| 等级 | 典型省份 | 综合评分 | 一般公共预算收入典型值 | 城投有息负债/综合财力 | 关键特征 |
|------|---------|---------|---------------------|---------------------|---------|
| **S1 极强** | 上海、北京、广东、江苏、浙江、福建、深圳（计划单列） | 2.5-3.0/3.0 | >7000亿（省级），>1000亿（深） | <150% | 财政自给率>70%，金融资源充裕，平台战略重要性明确，无违约先例 |
| **S2 强** | 山东、四川、湖北、安徽、河南、湖南、河北、江西、重庆、陕西 | 2.0-2.5/3.0 | 2500-7000亿 | 150-250% | 财政自给率40-70%，有化债能力，需细分到地市级评估 |
| **S3 中等** | 山西、天津、辽宁、广西、云南、内蒙古、新疆、甘肃、吉林、黑龙江 | 1.0-2.0/3.0 | 1000-2500亿 | 200-400% | 财政自给率<40%，债务率偏高（部分>300%），需关注中央转移支付规模 |
| **S4 弱** | 贵州、青海、宁夏、黑龙江（部分地区）、甘肃（部分地区） | 0.0-1.0/3.0 | <1000亿 | >400% | 财政自给率<30%，转移支付依赖度>50%，有非标违约先例 |

**重要说明**：
- 省级政府的支持能力不直接等同于其辖内所有城投的信用水平——政府对不同平台的救助顺序不同，核心平台可能获得超出省级平均水平的大力支持
- 天津、云南、内蒙古等省份近年化债力度大，需关注最新数据——建议半年更新一次评级
- 资源型省份（山西、陕西、内蒙古）受大宗商品价格周期影响显著，应在煤价/资源品价格周期的不同阶段分别评估

### 3.5 地市级/区县级政府的额外考量

地市级和区县级政府的财政实力分化比省级更大——最富的地级市（如苏州）一般公共预算收入超过2000亿，最弱的地级市不足50亿。

| 层级 | 强支持能力特征 | 弱支持能力特征 | 分析所需数据 |
|------|--------------|--------------|-------------|
| **地市级** | GDP>5000亿，一般公共预算收入>300亿，财政自给率>60%，债务率<150%，人口净流入 | GDP<1000亿，一般公共预算收入<50亿，财政自给率<30%，债务率>300%，人口持续流出 | 市本级财政决算报告/统计年鉴/省转移支付分配表 |
| **区县级** | GDP>500亿，一般公共预算收入>30亿，位于强省/核心城市周边，有产业支撑 | GDP<100亿，一般公共预算收入<10亿，以农业为主，远离经济中心 | 预决算报告（部分公开）/市级汇总数据/统计年鉴 |

**诚实声明**：区县级城投的财政数据质量是当前框架下最大的数据约束。全国超过2000个区县，大部分不能及时、完整地披露财政数据。分析区县级城投时，应依赖更高级别政府的官方数据（如省对县转移支付分配明细）以及替代指标（如区域内用电量、税收收入占比等间接信号）。

## 四、第二轨：平台自身信用评估（权重40-50%）

### 4.1 评估框架：五维模型

城投平台自身评估不是"企业分析"而是"资产+债务+职能"的组合分析——因为城投平台的经营行为与普通企业完全不同。

| 维度 | 权重 | 核心指标 | 评价逻辑 |
|------|------|---------|---------|
| **P1 职能定位** | 25% | 业务属性（纯公益/准公益/经营性）、政府授权文件、区域内平台数量及分工、是否是核心平台 | 职能定位决定了政府救助的优先顺序——纯公益类平台获支持的概率远高于经营性平台 |
| **P2 资产质量** | 25% | 有效资产占比（可变现土地/房地产/股权/收费权 vs 无效资产如市政道路/公园）、土地资产变现能力（土地性质/位置/市场条件）、应收政府款项账龄与回收确定性 | 资产质量决定平台在不依赖政府的情况下能变现多少偿债资源——土地资产的"水分"是最大风险 |
| **P3 债务结构** | 20% | 有息负债/总资产比率、短债占比、非标占比、债券融资占比、融资渠道多样性、一年内到期债务/货币资金 | 债务结构决定流动性风险——非标占比高=融资渠道收窄的标志；短债占比高=滚动压力大 |
| **P4 现金流** | 20% | 经营性现金流（通常为负或微弱正——属正常）、筹资性现金流规模及趋势、收到/支付其他与经营活动有关的现金（政府往来款）、投资性现金流（是否仍在扩张）、现金短债比 | 城投平台的分析逻辑不同于产业债：经营性现金流通常为负，筹资性现金流是主要偿债来源——关注的是"还能借到钱吗？"而非"赚钱了吗？" |
| **P5 对外担保** | 10% | 对外担保余额/净资产比率、被担保对象的信用质量、区域内互保链条长度、是否存在代偿风险 | 城投互保链是区域金融风险的传导器——一家城投违约可能通过互保链传染整个区域 |

### 4.2 P1 职能定位详解

职能定位是城投平台信用的**前提条件**——它决定了政府支持的逻辑基础。

| 职能类型 | 典型业务 | 政府支持确定性 | 平台分析逻辑 | 举例 |
|---------|---------|--------------|-------------|------|
| **纯公益型** | 市政道路/桥梁/公园/保障房/土地储备/棚改 | 最高——业务本身无现金流，完全依赖政府 | 核心看政府信用——平台自身分析价值有限 | 大部分省会城市的基础设施投融资类平台 |
| **准公益型** | 供水/供气/供热/公共交通/污水垃圾处理 | 较高——业务有现金流但不足以偿债，需政府补贴 | 需要看：经营性现金流对利息的覆盖程度+政府补贴是否稳定+收费定价是否受管制 | 北京自来水集团/各市燃气公司类平台 |
| **经营型** | 房地产开发/贸易/酒店/园区运营/产业投资 | 最低——业务以营利为目的，政府可不兜底 | 按产业逻辑分析——财务指标权重应上调至50%以上 | 部分开发区运营平台/持有型物业平台 |
| **混合型** | 上述多类业务并行 | 取决于公益业务占比 | 需分业务板块分析——公益部分看政府，经营部分看自身 | 大多数省级/省会平台属于此类 |

**核心判断**：市场上大多数城投平台是混合型或准公益型。完全纯公益或完全经营性平台反而是少数。职能定位分析的关键是**判断公益业务占比**——占比越高，政府对平台的救助必要性越大。

### 4.3 P2 资产质量详解：城投资产的"真假虚实"

城投平台的资产质量是最容易被误读的维度。以下是对不同类型资产的定性评估标准：

| 资产类型 | 在资产负债表中的列示 | 可变现性 | 对信用质量的贡献 | 需警惕的信号 |
|---------|-------------------|---------|----------------|-------------|
| **土地资产**（存货/无形资产） | 存货中的"开发成本"或无形资产中的"土地使用权" | **高度依赖位置和市场行情**——一线/强二线城市土地变现性强，三四线及以下城市土地变现周期长 | 核心资产——但需区分土地性质（划拨地 vs 出让地），划拨地的处置需政府审批，变现受限 | 土地账面价值明显高于市场价、大量土地位于人口流出区域、土地闲置超过3年 |
| **应收政府款项**（其他应收款/应收账款） | 其他应收款——对财政局的应收款项 | 回收取决于地方政府的财政安排——通常回收周期长但减记风险较低 | 城投最主要的"资产"之一——需关注账龄和政府确认函 | 应收款账龄>3年占比高、无政府确认函、应收款超过土地资产的2倍 |
| **公益性资产**（固定资产/在建工程） | 固定资产/在建工程——如市政道路、管网、绿化 | **几乎不可变现**——公益性资产没有市场交易价值 | 对信用质量贡献极小——属于"无效资产"，需做减值调整 | 公益性资产占资产总额>50%（这在很多城投中常见——需做调整后才能进行财务分析） |
| **持有型物业**（投资性房地产） | 投资性房地产 | 取决于物业位置和运营状态——可出租有现金流 | 中等——需要看出租率、租金水平、是否已抵押 | 以公允价值计量的物业估值连年上升但出租率下降、大量物业已抵押且处于二押/三押状态 |
| **金融资产**（长期股权投资/可供出售金融资产） | 对区域内其他国企/金融机构的股权投资 | 取决于被投资企业质量——上市公司股权流动性好，非上市股权变现难 | 质量分化大——高质量金融资产是重要的偿债后备资源 | 持有多家经营不善的小型城商行/农商行股权、金融资产占净资产>50% |
| **在建工程** | 在建工程——基础设施建设投入 | 完工前不可变现，完工后转入公益或准公益资产 | 低-中——需关注项目是否纳入政府预算、回款安排是否明确 | 在建工程持续增长但完工项目迟迟不结转、无对应政府回款安排 |

**核心分析工具：有效资产率**

```
有效资产率 = (可变现资产账面价值合计) / 总资产

有效资产 = 土地资产（出让地） + 持有型物业 + 高质量金融资产 + 有回收保障的应收款项
无效资产 = 公益性资产（道路/管网/绿地） + 划拨土地 + 长期挂账应收款 + 在建工程（未纳入预算部分）

有效资产率阈值：
  >60% → 资产质量优
  40-60% → 资产质量中等
  20-40% → 资产质量弱
  <20% → 资产质量极弱——平台几乎没有可自主偿债的资源
```

### 4.4 P3 债务结构详解

城投平台的债务结构分析重点关注三个维度：**期限分布、融资渠道、非标占比**。

| 指标 | 安全 | 关注 | 危险 | 数据来源 |
|------|------|------|------|---------|
| **非标融资占比** | <10% | 10-30% | >30%（融资渠道收窄的明确信号） | 年报附注——长期应付款（融资租赁/信托等） |
| **短债占比**（一年内到期/总有息负债） | <30% | 30-50% | >50% | 年报——短期借款+一年内到期非流动负债/总有息负债 |
| **债券融资占比** | >50% | 20-50% | <20%（过度依赖银行贷款和非标） | Wind/募集说明书——各融资渠道余额 |
| **银行授信使用率** | <50% | 50-80% | >80%（融资弹性已基本用尽） | 募集说明书——银行授信额度及已使用额度 |
| **单一融资渠道集中度** | 渠道多样，分散 | 依赖2-3种渠道 | 依赖单一渠道（如全额依赖债券市场） | 综合判断 |
| **隐性债务置换进度** | 大部分已纳入政府隐性债务系统，可置换 | 部分已纳入，部分自担 | 少部分或未纳入隐性债务系统 | 企业公告/地方政府债务信息公开 |

**核心信号——非标占比的指示意义**：
> 非标融资（融资租赁/信托/定向融资计划）通常成本高、期限短、监管不透明。当城投平台从债券市场向非标市场迁移，这往往是因为债券市场融资渠道已收窄（被监管限制或市场认可度下降）。**非标占比的上升是城投流动性压力的前置信号。** 非标违约也是城投违约的最常见形式——历史上非标违约的案例远多于债券违约。

### 4.5 P4 现金流分析的特殊逻辑

**这是城投分析中最容易被误读的财务维度。** 产业债的现金流逻辑是"经营性现金流为正才是健康的"——城投完全相反。

```
城投的标准现金流画像（正常状态）：
  ┌ 经营性现金流：微弱或为负（基础设施建设投入 > 政府回款收入）
  ├ 投资性现金流：大额为负（持续建设新项目）
  └ 筹资性现金流：大额为正（借新还旧 + 新增融资）
  
  综合看：经营性 + 投资性 持续为负 → 依赖筹资性现金流维持资金链

城投的恶化信号不是"经营性现金流为负"而是：
  ✅ 经营性现金流持续大幅恶化（政府回款大幅减少、补贴取消）
  ✅ 筹资性现金流由正转负（融资渠道枯竭——这是最危险的信号）
  ✅ 收到其他与经营活动有关的现金（政府对平台的拨付资金）大幅下降
  ✅ 现金短债比持续下降至<0.5x
```

| 现金流指标 | 分析方法 | 城投正常值 | 危险信号 |
|-----------|---------|-----------|---------|
| **经营性现金流** | 关注趋势而非绝对值——正常为负或微弱正 | -10亿至+10亿（视平台规模） | 持续>3年大幅为负，且趋势加剧 |
| **筹资性现金流** | **最关键指标**——是否还能"借到钱" | 大额正数（持续有融资流入） | 连续2个季度由正转负，或绝对值大幅下降>50% |
| **收到其他与经营活动有关的现金** | 反映政府对平台的资金拨付力度 | 通常占经营性现金流入的30-60% | 连续2年下降或无规律 |
| **现金短债比** | 短期流动性压力的直接衡量 | >1.0x | <0.5x，且无新增融资落地 |
| **利息覆盖倍数**（EBITDA/利息） | 城投普遍<1.0x（正常），需区分有无政府补贴 | 含补贴>1.0x，不含补贴<1.0x | 含补贴后仍<0.8x |

### 4.6 P5 对外担保风险

城投互保是中国信用市场特有的风险传导机制。**区域内一家核心城投的违约可以通过互保链快速传播。**

| 指标 | 安全 | 关注 | 危险 |
|------|------|------|------|
| **对外担保余额/净资产** | <30% | 30-60% | >60% |
| **被担保对象中同级平台占比** | <30% | 30-60% | >60%（区域内互保链条密集） |
| **被担保对象信用状况** | 被担保方均为AA+级以上 | 部分被担保方为AA级以下 | 被担保方已有非标违约/关注类/负面展望 |
| **担保集中度** | 单一被担保方占比<20% | 单一被担保方占比20-50% | 单一被担保方占比>50% |
| **反担保措施** | 有足值反担保 | 有部分反担保 | 无反担保 |

## 五、城投信用质量分级

### 5.1 分级标准

基于双轨评估的综合结果，城投信用质量分为四个等级。

| 等级 | 代码 | 综合评分 | 特征 | 典型区域+平台 |
|------|------|---------|------|-------------|
| **优质** | A | 7.0-10.0 | 强财政+核心平台+无违约先例+资产质量优+融资渠道多元 | 上海/北京/深圳/广东/江苏/浙江的省级/地市级核心平台 \\
| **中等** | B | 5.0-7.0 | 中等财政+重要平台+资产质量中等+可借新还旧 | 山东/福建/湖北/四川/安徽的地市级平台 \\
| **关注** | C | 3.0-5.0 | 弱财政或平台定位弱+非标占比较高或资产质量弱+融资渠道收窄 | 东北三省/贵州/云南/甘肃/部分区县级平台 |
| **风险** | D | 0-3.0 | 弱财政+边缘平台+已有违约传闻+融资渠道几乎关闭+非标占>30% | 历史上发生过非标违约的区县级平台/被整合中的边缘平台 |

### 5.2 分级使用规则

| 规则 | 内容 |
|------|------|
| **政府信用定上限** | 任何城投平台的信用等级不能超过其所属地方政府（省/地市/区县）的信用等级 |
| **平台自身定下限** | 如果政府不救助，平台独立信用等级大致取决于其有效资产率和现金短债比 |
| **支持意愿调落点** | 政府支持意愿强时，等级向上限靠拢（但不超上限）；支持意愿弱时，等级向下限靠拢 |
| **非标占比调减** | 非标占比>30%的平台，在综合评分基础上降低1-2分（对应一个子级） |
| **区域传染扣减** | 所在省份已有城投非标违约的，在综合评分基础上扣减0.5-1.0分；已有债券违约的，扣减1.0-2.0分 |

## 六、城投特有的预警信号

### 6.1 预警信号全景图

城投的预警信号与产业债完全不同。产业债的预警信号集中在财务恶化（收入下滑/毛利率下降/负债率上升），而城投的预警信号集中在"政府与平台之间的关系变化"。

| 信号类别 | 具体信号 | 危险等级 | 窗口期（从信号到信用事件） | 数据来源 |
|---------|---------|---------|-------------------------|---------|
| **资产划转类** | **核心资产被无偿划转至其他平台/国资运营公司** | 🔴 极高 | 4-12个月（参考：永煤2020/华晨2020） | 企业公告/国资委国有产权变更登记 |
| | 土地资产被政府收回但补偿不到位 | 🟠 高 | 6-18个月 | 政府公告/土地储备中心公示 |
| | 重要子公司股权被划出 | 🟠 高 | 6-12个月 | 工商变更/企业公告 |
| | 公立/收费权变动（如收费期限缩短/收费标准被管制） | 🟡 中 | 12-24个月 | 政府文件/行业监管公告 |
| **融资渠道类** | **债券发行利率跳升（超过同区域/同评级中位数100bp以上）** | 🟠 高 | 3-12个月 | Wind债券发行数据 |
| | **债券取消发行（非市场环境原因）** | 🔴 极高 | 1-6个月 | 中国货币网/上清所公告 |
| | 银行授信被收回或缩减 | 🟠 高 | 3-12个月 | 募集说明书更新数据 |
| | 非标融资占比连续2季度上升 | 🟡 中 | 6-18个月 | 财务报表附注逐期对比 |
| | 信托/租赁机构集中到期但续作困难 | 🟠 高 | 3-6个月 | 市场调研/渠道信息 |
| **政府支持类** | **安慰函/支持函被撤回或到期不再续签** | 🔴 极高 | 1-6个月 | 政府公告/企业公告 |
| | 财政补贴大幅减少或取消 | 🟠 高 | 6-18个月 | 企业年报——其他收益/营业外收入 |
| | 政府注资承诺未兑现 | 🟠 高 | 3-12个月 | 企业公告/国资委会议 |
| | 政府回款速度明显放缓（应收政府款项账龄拉长） | 🟡 中 | 12-24个月 | 年报——其他应收款账龄分析 |
| **平台整合类** | **小平台被合并入大平台——小平台债权人可能被"遗忘"** | 🟠 高 | 3-12个月 | 国资委公告/企业合并公告 |
| | 区域内多个平台整合（定位重新划分） | 🟡 中 | 6-18个月 | 政府工作报告/国资改革方案 |
| | 平台法定代表人/高管由政府背景→市场化招聘 | 🟡 中 | 12-24个月 | 工商变更/企业公告 |
| **区域传染类** | **隔壁县/市城投非标违约** | 🟠 高 | 1-6个月 | 新闻检索/政信产品违约统计 |
| | 所在省份其他城投债券出现技术性违约/展期 | 🔴 极高 | 1-3个月 | Wind/中国货币网/新闻 |
| | 区域内互保链中的一家平台被下调评级 | 🟡 中 | 3-12个月 | 评级机构公告 |
| **人员变动类** | 财政/国资背景高管集中离职 | 🟡 中 | 6-18个月 | 企业公告/工商信息 |
| | 平台被政府列入"转型/退出"名单 | 🟠 高 | 12-24个月 | 政府文件/国资改革方案 |

### 6.2 信号等级触发与行动规则

| 触发条件 | 行动要求 | 对评级的影响 |
|---------|---------|------------|
| 出现1个🔴 极高危险信号 | 立即启动全面重新评估——触发观察名单"负面观察" | 评级可能下调1-3个子级 |
| 出现2个以上🟠 高危险信号 | 启动专项评估（评估相关维度）——触发展望调整为"负面" | 评级可能下调0-2个子级 |
| 出现1个🟡 中危险信号+其他负面信号 | 纳入持续监控清单——密切跟踪后续变化 | 暂不调整评级，但标注"关注" |
| 同一区域（地市级）内出现2家以上城投的非标违约 | 触发"区域传染预警"——该区域所有城投平台评级扣减1子级 | 批量下调该区域平台评级 |

### 6.3 城投政策环境的三级影响传导

城投分析必须纳入政策环境变化。以下是城投政策的**三级传导链条**：

```
第一级：中央政策导向
  ├── 隐性债务化解（"化债"）→ 影响所有城投的融资空间
  ├── 平台转型要求（"剥离政府融资职能"）→ 影响平台定位和政府支持意愿
  ├── 新增融资监管（"15号文"等）→ 影响城投新增融资能力
  └── 专项债规模/用途 → 影响城投项目的资金来源

        │
        ▼

第二级：省级政策执行
  ├── 化债方案（规模、节奏、工具）→ 影响辖内城投的存量债务处置
  ├── 平台整合方案 → 影响辖内城投的个体定位
  ├── 财政资源分配 → 影响政府救助的能力和意愿
  └── 金融监管（银行/信托/租赁）→ 影响辖内城投的融资可得性

        │
        ▼

第三级：地市/区县落地
  ├── 具体平台的职能定位/转型路径 → 直接影响个体平台信用
  ├── 政府支持的实际力度（注资/补贴/回款） → 直接影响现金流
  └── 区域内债务化解的实际进展 → 直接影响市场对该区域城投的信用定价
```

**核心判断**：城投政策的变化比财务数据更致命。2023-2026年的化债政策（特殊再融资债/一揽子化债方案）显著改善了部分区域的流动性风险，但也使得"隐性债务是否纳入置换"成为政府支持意愿的分水岭——**纳入隐性债务系统的债务大概率获得化解，未纳入的则面临更大的不确定性**。

## 七、城投债的LGD考量

城投债的违约后回收路径与产业债完全不同。**产业债的违约回收依赖于破产清算/重整，而城投违约后的回收依赖于政府协调和债务重组。**

### 7.1 城投vs产业债的LGD差异

| 维度 | 产业债 | 城投债 |
|------|--------|--------|
| **法律框架** | 适用《企业破产法》——有明确的清算/重整程序 | **城投平台不适用《破产法》**——没有破产清算的先例 |
| **回收路径** | 司法重整/破产清算/庭外和解 | 政府协调/债务重组/展期降息/政策性银行介入 |
| **时间周期** | 1-3年（破产重整程序） | 2-5年甚至更长（政府协调周期长） |
| **回收率范围** | 10-50%（有抵押/担保的回收率更高） | **0-40%**（非标违约历史案例，债券违约案例极少） |
| **回收率的关键变量** | 抵押物价值/清算时的资产折价 | 地方政府的协调意愿+平台剩余资产价值 |
| **持有人的谈判地位** | 有明确的法律权利——可行使债权人权利 | 法律地位不明——城投非《破产法》主体；政府协调中政治>法律 |

### 7.2 历史非标违约回收经验

中国城投市场尚未出现大规模债券违约——截至2026年中，城投债券实质性违约的案例仍然极为稀少。但非标违约（信托/融资租赁/定向融资计划）已经发生较多，以下数据来自非标违约的观察：

| 区域 | 违约类型 | 典型回收率范围 | 回收时间 | 关键驱动因素 |
|------|---------|--------------|---------|-------------|
| 贵州 | 非标（信托/租赁） | 10-30% | 3-5年 | 省级政府化债推进力度、中央转移支付规模 |
| 云南 | 非标（信托） | 20-40% | 2-4年 | 省级资源协调、优质资产处置 |
| 内蒙古 | 非标（定向融资） | 5-20% | 3-5年 | 区域经济基础弱、资产变现困难 |
| 东北 | 非标（信托/租赁） | 10-30% | 3-4年 | 国有资产处置进度、政府协调意愿 |
| 西部弱财区域 | 非标（多类型） | 0-15% | 4-6年且部分尚未解决 | 财政极度紧张、核心资产已空 |

**诚实声明**：由于城投债券实质性违约案例极少，上述LGD数据主要基于非标违约的观察，且样本量有限。城投债券违约的实际回收率可能高于或低于非标违约——债券的公开性和市场影响更大，政府协调的激励也更强。**当前框架下，城投债的LGD评估无法做到产业债那样的精确度，只能给出参考范围。**

### 7.3 城投违约后的回收路径分析

```
回收路径优先顺序（从最可能到最不可能）：

路径一：政府协调 + 展期降息（最常见）
  适用条件：政府有协调意愿 + 平台尚有融资潜力
  做法：与债权人协商延长债券期限、降低票面利率
  回收率预期：面值或接近面值（债权人不损失本金）
  案例：部分省级城投的非标债务展期安排

路径二：资产处置 + 部分兑付
  适用条件：平台尚有可变现的有效资产
  做法：加速处置土地/持有型物业/优质股权
  回收率预期：30-60%（取决于资产质量和市场条件）
  案例：部分信托产品逾期后的资产处置兑付

路径三：政府注资 + 债务承接
  适用条件：平台对政府战略重要 + 政府有财政空间
  做法：政府安排专项财政资金/资产注入用于偿债
  回收率预期：面值或接近面值
  案例：部分纳入"一揽子化债"的隐性债务

路径四：不了了之（最坏情况）
  适用条件：政府意愿弱 + 平台无有效资产 + 区域经济极弱
  做法：债权人通过法律途径但执行困难
  回收率预期：0-10%
  案例：部分弱区域非标违约后久拖不决
```

## 八、城投债作为第八个"准行业"的集成设计

### 8.1 十维行业评分

城投债作为金融引擎中第八个分析模块，通过十维行业评分（D1-D10与industry-framework.md第二章一致）定位为**政策驱动型"准行业"**。

| 维度 | 评分 | 评分理由 |
|------|------|---------|
| D1 市场容量 | 5 | 城投债存量~15万亿，占信用债市场35%以上 |
| D2 增量空间 | 3 | "化债"政策下增量受限，存量置换为主——政策方向是"控增量、化存量" |
| D3 政策支持力度 | 5 | 化债政策（特殊再融资债券/一揽子化债方案）是城投市场的核心支撑 |
| D4 政策波动性 | 5 | 城投政策是所有行业中变化最频繁、方向最反覆的——融资监管政策几乎每季度都有新文件 |
| D5 资本可持续性 | 2 | 融资渠道持续收窄——非标被大幅压缩、银行贷款受限、债券审核趋严 |
| D6 民生关联度 | 5 | 城市基础设施建设/公用事业运营——直接涉及社会稳定和基本公共服务 |
| D7 外部依赖度 | 1 | 城投业务完全在国内——无外部依赖 |
| D8 供应链权力集中度 | 4 | 城投平台在区域内具有一定的垄断地位（多为区域内基础设施建设的唯一/主要承接方） |
| D9 产业生命周期 | 2 | 城投行业处于"转型退出期"——政策明确要求剥离政府融资职能，平台面临转型或退出 |
| D10 周期波动性 | 4 | 高度依赖土地市场和房地产市场周期——土地出让收入波动直接影响城投信用 |

### 8.2 类型判定

| 判断条件 | 结果 |
|---------|------|
| D3=5 >=4, D4=5 >=3 | → **政策驱动型** |
| D9=2 <=2 | → 生命周期处于衰退/转型期 |
| D6=5, D8=4 | → 民生关联度高+准垄断属性 |

**结论**：城投属于**政策驱动型"准行业"**——最重因子是政策变化。但需要注意形态差异：与光伏/半导体等政策驱动型"看得见的手推动增长"不同，城投是"看得见的手在做风险化解"——政策方向是控制风险而非促进增长。

### 8.3 城投金字塔权重设计

| 层 | 权重 | 说明 | 与产业政策驱动型权重差异 |
|------|------|------|------------------------|
| **L1 政策/宏观** | 35% | 与光伏等行业一致——政策是城投的最重因子 | 完全一致 |
| **L2 政府/地方** | 25% | 城投特有的层级——地方政府的财政实力和债务负担 | 无对应——城投专属层 |
| **L3 平台审计** | 20% | 平台自身的职能定位、资产质量、债务结构 | 比产业债的L3更注重资产质量和债务结构 |
| **L4 财务/偿债** | 20% | 现金流趋势（尤其是筹资性现金流）、对外担保、隐性债务置换进度 | 与产业债不同——城投财务分析的核心是筹资性现金流，而非经营性现金流 |

### 8.4 与外部支持框架的联动

本框架与external-support-framework.md的关系：

| 本框架（lgv-framework.md） | external-support-framework.md | 关系 |
|---------------------------|------------------------------|------|
| 第一轨：政府信用评估（3.1-3.5） | 第四章：政府支持能力评估（4.1-4.2） | 本框架是城投专属化版本——细化到区县级评估，增加土地财政依赖度分析 |
| 第一轨：支持意愿（3.3） | 第五章：支持意愿评估（5.1-5.3） | 本框架增加了"平台整合类"和"区域传染类"信号——城投特有的支持意愿变化信号 |
| 第二轨整体 | 第六章：上调规则（6.1-6.5） | 本框架将外部支持上调的逻辑内化为城投评估的第一轨——城投的"外部支持"不是加分项而是基础项 |
| 预警信号（第六章） | 第七章：陷阱信号（7.1-7.3） | 本框架增加了城投特有信号——资产划转、平台整合、区域传染 |

**核心差异**：在通用外部支持框架中，外部支持是"在上调到独立信用质量之上"的加分项。在城投框架中，政府支持是城投信用质量的**基础**——没有政府支持的城投平台本身就缺乏存在的合理性。**城投分析不从"独立信用质量"开始，而从"政府+平台"的双轨评估开始。**

### 8.5 在多利益相关者视角中的集成

在multi-stakeholder.md第六章框架中，新增城投专属视角：

| 角色 | 城投分析的特别关注 | 常规分析的差异 |
|------|------------------|--------------|
| **M0 信贷审批（银行）** | 政府债务率/城投有息负债/综合财力比率——第二还款来源（政府信用）比第一还款来源（平台现金流）更重要 | 城投贷后管理的重点不是财务指标跟踪，而是**政策跟踪+资产划转监测** |
| **M1 债券投资** | 城投利差的三层定价逻辑：政府信用层+平台个体层+区域传染层 | 城投的条款保护（担保/增信）往往弱于产业债——交叉违约条款普遍缺失 |
| **M2 债券承销** | 城投的发债窗口受政策影响极大——政策收紧时批文可能中止 | 城投债承销的关注点不是"企业质地"而是"地方政府协调能力" |
| **M3 市场交易** | 城投的流动性分层严重——强区域城投（江浙/广东/福建）流动性好，弱区域城投流动性极差 | 城投债的信用事件（非标违约/区域传染）传播路径不同于产业债——区域而非行业是定价的决定因素 |
| **M4 组合风控** | 城投债的集中度风险不仅仅是单一发债主体的集中度，更是**区域的集中度**——持有过多同一省份的城投债=间接持有地方政府的信用风险 | 城投的尾部风险是"区域系统性风险"而非"个体违约风险" |
| **M5 企业融资（平台融资部门）** | 融资渠道管理——维持债券市场可进入性是最关键的信用支撑 | 城投的融资策略主要是"政策应对"而非"投资者沟通"——跟上政策节奏比取悦投资人更重要 |

### 8.6 在展望与监控框架中的集成

在outlook-monitoring-framework.md中新增城投专属的监控触发条件：

**正面展望触发信号（城投专属）**：
| 信号 | 数据来源 |
|------|---------|
| 省级政府出台明确化债方案且资金到位 | 地方政府工作报告/财政厅文件 |
| 平台核心资产被注入（土地/股权/收费权）而非划出 | 企业公告/国资委公告 |
| 政府加大注资力度（资本公积大幅增加） | 企业年报 |
| 区域内城投非标违约历史被化解（如展期协议达成） | 新闻/政信产品信息 |
| 平台被整合为核心平台（非边缘化） | 政府文件/国资改革方案 |

**负面展望触发信号（城投专属）**：

| 财政补贴/注资大幅减少 | 企业年报 |
| 政府高管退出平台董事会/管理层 | 工商变更 |

## 九、数据质量与框架局限性

### 9.1 城投数据质量的分层评估

这是本框架最重要的诚实声明。城投数据的质量在不同层级、不同区域之间差距极大，**分析结论的置信度直接受数据质量影响**。

| 数据层级 | 数据可得性 | 时效性 | 可信度 | 分析置信度影响 |
|---------|-----------|-------|-------|--------------|
| **中央/省级政府财政数据** | ✅ 公开透明——财政部统一发布 | 年度（滞后6-12个月） | 高 | 对省级城投的政府端评估信心较强 |
| **地市级政府财政数据** | ✅ 大部分公开——地市财政局官网发布 | 年度（滞后6-12个月） | 中-高 | 地市级城投政府端评估有信心，但需关注数据口径 |
| **省会城市/计划单列市数据** | ✅ 公开透明程度接近省级 | 年度（滞后6个月） | 高 | 对强二线城市城投的评估信心较强 |
| **区县级政府财政数据** | ⚠️ 公开率约70%，规范性参差不齐 | 年度（滞后6-12个月，部分不公开） | 中-低 | **区县级城投的政府端评估置信度明显低于省级/地市级** |
| **省级城投平台财务报表** | ✅ 公开（有发债义务的平台） | 年度/半年度 | 中-高 | 省级平台的财务分析有数据基础 |
| **地市级城投平台财务报表** | ✅ 大部分公开（有存续债的平台） | 年度/半年度 | 中 | 地市级平台财务数据基本可用 |
| **区县级城投平台财务报表** | ⚠️ 部分公开——尤其是非发债平台 | 年度（部分平台不按时披露） | 低-中 | **区县级平台财务数据的可靠性和及时性是最大瓶颈** |
| **土地市场数据** | ✅ 自然资源部公开数据 | 月度 | 高 | 可用于评估土地资产变现能力 |
| **非标融资数据** | ❌ 非公开——从财务报表附注推断 | 年度（附注披露） | 低 | **非标融资规模是估算的，实际值可能显著高于财报披露** |
| **对外担保数据** | ⚠️ 发债平台有披露义务，但可能不完整 | 年度 | 中 | 部分平台的对外担保规模可能被低估 |
| **历史违约数据** | ⚠️ 债券违约数据透明，非标违约数据不完整 | — | 中（债券）/ 低（非标） | 非标违约的前置参考价值有限 |

### 9.2 框架的根本局限

| 局限 | 说明 | 缓解措施 |
|------|------|---------|
| **支持意愿不可量化** | 政府是否救助城投是政治决策而非财务决策——无法通过公开数据精确预测 | 使用信号驱动的定性判断，标注"置信度：低/中/高" |
| **隐性债务真实规模不可知** | 城投有息负债中只有部分被纳入政府的隐性债务系统，未被纳入的部分偿债前景更不可测 | 使用"估算+范围标注"——不给出精确数字，标注"高/中/低"估算 |
| **区县级数据严重不足** | 全国~2000个区县，大部分不能及时披露完整财政和平台数据 | 区县级城投分析的置信度天然低于省级/地市级——在分析结论中明确标注"区县级数据局限" |
| **案例验证基础薄弱** | 城投债券实质性违约极少——没有足够的历史案例来验证框架的有效性 | 使用非标违约案例作为替代参考，但标注"非标≠债券" |
| **政策变化不可预测** | 城投政策历史上曾多次突转（如2018年的"剥离融资平台政府融资职能"） | 建立政策跟踪机制——定期扫描中央/地方城投相关政策变化，而非仅在分析时点做一次性评估 |
| **区域传染效应的非线性** | 一个省份的区域城投信用恶化可能非线性传播——可能在某个临界点后急剧加速 | 建立区域传染"临界信号"清单——当某区域触发多个信号时，触发批量降级 |

### 9.3 框架适用性说明

| 适用性 | 说明 |
|-------|------|
| **本框架适用于** | 有存续债券的城投平台（各级别有财务披露义务的平台） |
| **本框架部分适用于** | 非发债平台（无公开财务数据——需依赖政府端评估+替代数据） |
| **本框架不适用于** | 已完成转型且不再承担政府融资职能的"类产业国企"（应使用产业债分析框架） |
| **框架最佳使用场景** | 地市级及以上城投平台、区县级核心平台 |
| **框架置信度较低场景** | 弱区域/弱财政的区县级边缘平台——此类平台的数据质量和分析确定性均显著偏低 |
