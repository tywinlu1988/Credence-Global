# 定量分析方法论 — 固定收益信用分析引擎

**版本**: v0.3.0 | **日期**: 2026-07-08
**定位**: 轨道B（市场定价）的深化量化层 · 双轨框架中的量化分析规范

---

## 一、定量分析的定位

### 1.1 在双轨框架中的角色

双轨分析框架中，轨道B（市场定价信号）原本定义为四级信号体系（平静/关注/异常/危机），此处定量分析是**轨道B的深化层**——在四级分类基础上，对每类信号做精确定量剖析。

```
轨道B（原四级信号）              定量分析（深化层）
   ├── 信用利差                    ├── L0.1 市场数据对标
   ├── 波动率                     ├── L0.2 统计信号检测
   ├── 资金流向                    ├── L0.3 量化因子模型
   └── 评级迁移                    └── 压力测试与情景分析
```

| 做什么 | 不做什么 |
|--------|---------|
| 从市场公开数据中提取量化信号 | 替代基本面分析（轨道A） |
| 将市场信号转化为结构化评分 | 预测违约精确日期 |
| 检测异常波动和定价偏差 | 提供财务审计结论 |
| 为交叉对撞矩阵提供量化输入 | 替代外部评级（外部评级本身是输入之一） |
| 量化定价与基本面的偏离度 | 给出交易建议（仅提供分析框架） |

### 1.2 定量分析的三个层次

| 层次 | 名称 | 描述 | 数据需求 | 计算复杂度 |
|------|------|------|----------|-----------|
| **L0.1** | 市场数据对标 | 将可获取的市场原始数据做清洗、对齐、对比——"是什么" | 低：公开市场数据 | 低：基础统计 |
| **L0.2** | 统计信号检测 | 在L0.1基础上做统计建模，检测异常——"哪里有问题" | 中：需要历史序列 | 中：统计检验 |
| **L0.3** | 量化因子模型 | 多因子分解和压力测试，解释定价——"为什么、如果...会怎样" | 高：需要面板数据和基准 | 高：回归/模拟 |

**实现优先级：L0.1 > L0.2 > L0.3**。在公开数据约束下，L0.1和L0.2步是大多数分析的核心，L0.3仅在付费数据补全后才具备完整可行性。

### 1.3 数据可得性约束（诚实声明）

本引擎**不以Wind/Bloomberg等付费终端为必要条件**。以下是在零付费终端环境下能做什么、不能做什么的硬约束清单：

| 指标 | 能否获取 | 来源/替代方案 | 质量 |
|------|---------|--------------|------|
| 到期收益率(YTM) | 可获取 | 交易所行情/中国货币网/和讯债券 | 较高——非活跃券可能偏离 |
| Z-spread | **不可获取** | 需要专业终端的现金流建模引擎 | 不可用——只能近似 |
| OAS | **不可获取** | 需要含权定价模型 | 不可用——只能近似 |
| 修正久期 | 部分可获取 | 根据票息/期限/收益率近似计算（McCaulay/1+YTM） | 中——不含含权调整 |
| 凸性 | **不可获取** | 需要精确现金流定价 | 不可用 |
| 历史收益率序列 | 可获取（有限） | 交易所公开行情，非活跃券缺失 | 中——数据稀疏 |
| Bid-ask spread | **不可获取** | 中国债市不透明，非专业终端无此数据 | 不可用——用换手率近似 |
| 隐含波动率 | **不可获取** | 中国信用债/可转债无标准化期权市场 | 不可用——用历史波动率 |
| 股票价格/波动率 | 可获取 | 免费行情API / 财经网站 | 较高 |
| 成交量/换手率 | 部分可获取 | 交易所公开数据，但流动性极低的券数据稀疏 | 中 |
| 信用评级迁移 | 可获取 | 评级公司公告 | 较高——但有滞后性 |
| 资金流向/北向资金 | 可获取 | 交易所/港交所每日披露 | 较高——但精细度不足 |

**核心结论：在不依赖付费终端的前提下，我们可以做基于公开市场数据的趋势分析、异常检测、相对价值排序，但不能做需要精确现金流贴现模型和期权定价模型的精细分析。**

## 二、信用利差分析

### 2.1 利差指标定义

在无法获取Z-spread和OAS的硬约束下，本引擎使用以下利差近似体系：

| 指标 | 公式 | 适用场景 | 局限性 |
|------|------|---------|--------|
| **YTM利差** | YTM_债券 - YTM_同期限国开债 | 通用——最基础的信用利差近似 | 未考虑期限结构差异；含权债券偏差大 |
| **YTM-基准利率近似Z-spread** | 同上 + 线性插值修正 | 当待偿期限偏离基准期限时 | 效果中等——方向正确但量级有偏 |
| **行业利差** | YTM利差 - 同评级行业均值 | 跨行业对比时，剔除行业贝塔 | 需要足够样本计算行业均值 |
| **个券-行业偏离度** | 个券YTM利差 - 行业YTM利差中位数 | 识别行业内高于/低于同行的个券 | 样本不足时中位数不稳定 |

**Z-spread的替代处理规则：**

```
当需要Z-spread但不可获取时：
  1. 首选：YTM - 同期限国开债收益率（线性插值）
  2. 标注："Z-spread不可获取，使用YTM利差近似——分析误差可能随久期和含权特性扩大"
  3. 对含权债券单独标注："含权定价依赖OAS模型，无法精确计算"
  4. 在结论中使用排序而非绝对值——"利差水平高于80%同行"优于"利差为235bp"
```

### 2.2 利差三维对比框架

每个分析对象的信用利差必须在三个维度上展开对比：

```
三维利差分析
  ├── 横向：同行业/同评级企业的利差分布
  │    ├── 目标：识别相对定价——哪些企业被市场高估或低估
  │    ├── 输出：个券在行业利差分布中的分位数位置
  │    └── 实例：光伏5只可转债的行业内部对比（见2.5）
  │
  ├── 纵向：企业自身历史利差分位数
  │    ├── 目标：判断当前定价相对自身历史是高还是低
  │    ├── 窗口：6个月 / 1年 / 3年（依数据可获取性）
  │    ├── 分位数区间：0-20%压缩 / 20-80%常态 / 80-100%扩张
  │    └── 规则：连续3个月处于80%以上分位数 → 标记为"持续利差扩张"
  │
  └── 交叉：评级间利差补偿
       ├── 目标：判断市场对不同评级的风险溢价是否合理
       ├── 指标：AAA-BBB利差差、BBB-B利差差
       ├── 正常范围：AAA-BBB利差差在80-150bp（历史中位数）
       └── 异常场景：
            ├── 评级间利差差收窄→市场不区分评级，潜在低估风险
            └── 评级间利差差急剧扩大→市场恐慌，流动性溢价主导
```

### 2.3 利差异常检测规则

| 异常类型 | 定量定义 | 信号等级 | 可能含义 |
|---------|---------|---------|---------|
| **突跳（Jump）** | 单日利差变化 > 50bp | 🔴 异常 | 突发信用事件/评级调整/价格错误 |
| **持续走阔（Trend）** | 连续20+交易日利差累计扩大 > 50bp | 🟡 关注 | 信用质量渐进恶化/行业逆风 |
| **与基本面背离** | 基本面评分稳定但利差走阔 >1个标准差 | 🟠 分歧 | 市场定价与基本面脱节→分歧信号 |
| **与行业背离** | 行业利差稳定个券利差独立走阔 >30bp | 🟠 分歧 | 企业个性化风险暴露 |
| **流动性压缩** | 成交量骤降+利差走阔同时发生 | 🔴 异常 | 流动性危机前兆 |

**异常检测处理流程：**

```
1. 从L0.1获取利差时间序列
2. 计算历史均值和标准差（周期=60交易日）
3. 检测条件：
   a) 单日变化 > Mean + 3σ → 突跳异常
   b) 20日累计变化 > Mean + 2σ → 持续异常
   c) 利差分位数 > 90%持续15日 → 高水位预警
4. 交叉验证：同时检查成交量/换手率是否异常
5. 输出：异常信号 + 可能的解释假设 + 建议行动
```


### 2.3.1 利差统计量的置信区间规范【统计修正 v0.3.1】

**Bootstrap置信区间用于利差均值估计：**
```
利差均值 95% Bootstrap置信区间
├── 方法：非参数Bootstrap，重抽样1000次
├── 计算步骤：
│   1. 从原始利差序列（N个观测值）有放回抽样N次 → 得到Bootstrap样本
│   2. 计算该样本的利差均值
│   3. 重复步骤1-2共计1000次 → 得到1000个Bootstrap均值
│   4. 取这1000个均值的2.5%和97.5%分位数 → 95%置信区间
├── 输出："利差均值: 185bp [95% CI: 172bp–198bp]"
└── 前提：N≥30才可执行Bootstrap；N<30时标注"样本量不足，Bootstrap不可靠，使用t分布近似"
```

**利差异常检测规则修订（替代原50bp主观阈值）：**
```
异常类型：突跳（Jump）
旧规则：单日利差变化 > 50bp
新规则：单日利差变化 > 历史滚动均值(60d)的95% Bootstrap置信区间上界
        → 当前值 > μ̂_boot_97.5% → 标记为异常
        → 标注："Bootstrap 95% CI上界 = XX bp，当前变化超出该范围"

异常类型：持续走阔（Trend）
旧规则：连续20+交易日利差累计扩大 > 50bp
新规则：连续20日累计变化 > 历史滚动均值(60d)的95% Bootstrap置信区间上界
        → 标注方法同上，标注"经Bootstrap CI校正"

异常类型：与基本面背离
旧规则：基本面评分稳定但利差走阔 >1个标准差
新规则：基本面评分稳定但利差走阔超出滚动均值的95% Bootstrap置信区间
```

**置信区间可视化规范：**
- 所有利差时序图必须附带95%置信区间带（阴影区域）
- 标注当前值是否在置信区间内
- 连续5个交易日超出区间上界 → 升级信号等级

### 2.3.2 多重假设检验校正（FDR）【统计修正 v0.3.1】

**问题背景：** 同时监控N个主体×K种检测规则 = 大量并行假设检验，假阳性率随检验次数增加而膨胀。例如，同时监控100只债券的3种异常类型 = 300个并行检验，即使所有债券都正常，也会产生约15个假阳性信号（300×0.05）。

```
FDR校正规范（Benjamini-Hochberg程序）：
├── 设定FDR控制水平：q = 0.05（即允许5%的显著信号为假阳性）
├── 适用场景：
│   ├── 利差异常检测：同时检测N只债券的M种异常类型
│   ├── 波动率异常检测：同时检测N只股票的K种波动率模式
│   └── 任意涉及多重比较的场景
├── BH程序步骤：
│   1. 对所有m个并行检验的p值排序：p₁ ≤ p₂ ≤ ... ≤ pₘ
│   2. 找到最大的k，使得 pₖ ≤ (k/m) × q
│   3. 拒绝H₀对应的前k个检验 → 这些是"经FDR校正后显著"的信号
│   4. 其余m-k个检验虽原始p<0.05但经校正后不显著 → 标注"校正后不显著"
├── 输出规范：
│   ├── "今日触发预警数：15个"
│   ├── "经FDR校正后的显著信号数：8个"
│   ├── "未校正的原始信号数：15个"
│   └── "假阳性风险提示：今天触发的15个预警中，统计上约0-7个可能是假阳性"
└── 标注规则：
    ├── 校正后仍显著的信号 → 标注 "✅ FDR校正后显著"
    ├── 校正后不显著的信号 → 标注 "⚠️ 原始p<0.05但FDR校正后不显著"
    └── 全局标注："FDR校正：Benjamini-Hochberg程序，q=0.05"
```

### 2.4 可转债利差分析（特殊处理）

可转债的信用利差分析与纯债不同，因其含权属性：

| 方面 | 可转债处理方式 | 原因 |
|------|--------------|------|
| YTM计算 | 标准YTM + 转股期权价值分离 | 纯债部分和转股权需要分开定价 |
| 信用利差标尺 | 纯债价值 vs 同评级信用债 | 剔除转股期权溢价后的债底分析 |
| 到期收益率 | 通常为负——纯债部分通常显著折价 | 转股期权价值抵消了信用风险溢价 |
| 更关注的指标 | 转股溢价率 + 纯债溢价率 | 信用风险在可转债中的表现更隐蔽 |

### 2.5 实例：光伏5只可转债利差对标分析

**分析日期：** 2026-07-01 | **数据源：** 交易所公开行情 + 国开债到期收益率曲线

**样本池：** 光伏行业5只可转债

| 可转债 | 对应正股 | 价格(元) | 到期收益率(%) | YTM利差(bp) | 转股溢价率(%) | 评级 |
|--------|---------|---------|-------------|-----------|------------|------|
| 隆22转债 | 隆基绿能 | 105.80 | 0.52 | 95 | 74 | AAA |
| 通22转债 | 通威股份 | 98.20 | 2.15 | 258 | 45 | AAA |
| 天23转债 | 天合光能 | 101.50 | 1.80 | 223 | 52 | AA |
| 晶科转债 | 晶科能源 | 103.40 | 1.35 | 178 | 38 | AA |
| 晶澳转债 | 晶澳科技 | 96.80 | 2.45 | 288 | 61 | AA |

**三维分析结果：**

| 维度 | 分析 | 发现 |
|------|------|------|
| **横向对比** | 行业利差中位数 = 223bp（通22/天23附近） | 隆22利差最低(95bp)——AAA评级+龙头溢价 |
| | | 晶澳利差最高(288bp)——市场对其信用质量有额外担忧 |
| | 隆22 vs 晶澳利差差 = 193bp | 同一行业中极端分化——验证5.5分评级区分度 |
| **纵向对比** | 隆22 YTM利差95bp → 历史分位数约55% | 接近历史均值，非异常 |
| | 通22 YTM利差258bp → 历史分位数约78% | 高于历史均值，与2025年亏损预期一致 |
| **交叉对比** | AAA利差中位数≈95-258bp（样本少） | AAA内部高度分化——评级信号粗糙 |
| | AA利差中位数≈178-288bp | AA内部同样分化——企业个性大于评级共性 |

**核心结论：**
- 隆22的利差显著低于同行——AAA评级+龙头地位提供了"安全溢价"，但74%的转股溢价率说明纯债部分估值非常保守
- 晶澳利差最高——市场对其盈利能力下滑和负债率上升已有定价
- 同行业内利差差异达193bp——**基本面差异驱动**，而非系统性市场情绪

**数据质量标注：**
- YTM利差计算基于收盘净价，非活跃券可能有1-2日滞后
- Z-spread不可获取，假设期限匹配的误差在±15bp以内
- 可转债YTM为负时，利差分析的实际意义下降——此时应重点关注转股溢价率

## 三、波动率分析

### 3.1 波动率指标与可得性

| 波动率类型 | 定义 | 数据可得性 | 对信用分析的意义 |
|-----------|------|-----------|-----------------|
| **历史波动率(HV)** | 股价/收益率过去N日收益率的标准差（年化） | 上市公司可获取（免费行情） | 反映企业价值的不确定性 |
| **隐含波动率(IV)** | 从期权市场价格反推 | ❌ 不可获取——中国信用债/CDS期权市场不存在 | N/A |
| **已实现波动率(RV)** | 高频交易数据计算的日度波动率 | 仅高流动性股票可获取（分钟级行情付费） | 使用日度HV代替 |
| **特质波动率** | 剥离市场因子后的残差波动 | 可从免费数据回推测算 | 反映企业自身风险 |

**硬约束：** 非上市公司无股价数据，波动率分析完全不可用。这是轨道B在非上市企业上的核心局限之一。

### 3.2 历史波动率计算规范

```
基础参数：
├── 数据源：免费行情API / 财经网站（新浪财经、东方财富等）
├── 价格类型：收盘价（复权）
├── 收益率计算：ln(P_t / P_{t-1}) × 100%
├── 年化因子：√252（交易日数）
└── 计算窗口：20日 / 60日 / 120日（同时输出）

HV公式：
  HV = σ(r) × √252
  其中 σ(r) = 窗口期内日收益率的标准差

输出格式：
  HV_20d: 35.2%  （短期——反映最近一个月的不确定性）
  HV_60d: 28.7%  （中期——反映近一个季度的不确定性）  
  HV_120d: 32.1% （长期——反映近半年的不确定性）
  HV_Trend: "上升" | "下降" | "稳定"（通过20日移动平均判断）
```

### 3.3 波动率分解框架

在数据约束下，使用简化分解方式——通过单因子回归近似拆解系统性波动和特质波动：

```
Step 1: 获取行业指数/市场指数收益率
  市场代理：沪深300 (000300.SH)
  行业代理：光伏产业指数 (931151.CSI) / 中证行业指数

Step 2: 滚动回归（窗口=120交易日）
  R_i = α + β × R_m + ε
  其中 R_i = 个股日收益率
       R_m = 指数日收益率
       ε = 残差（特质收益）
       β = 系统性因子暴露

Step 3: 波动率分解
  总波动率(V_total) = β² × V_m + V_ε
  系统性占比 = β² × V_m / V_total × 100%
  特质占比 = V_ε / V_total × 100%

Step 4: 输出
  ├── 系统性波动占比：65%（行业风险主导）
  ├── 特质波动占比：35%（企业自身风险）
  ├── 系统性波动趋势（与行业对比）："高于行业均值，且差距在扩大"
  └── 特质波动趋势："上升——反映企业个性化风险增加"
```

**局限性标注：**
- 简化单因子模型未考虑行业β的时变性
- 120日窗口期内假设β稳定——在波动率市中这个假设较弱
- 非上市公司此分析不可用

### 3.4 波动率突变与预警信号

| 波动率模式 | 检测方法 | 信用含义 | 建议行动 |
|-----------|---------|---------|---------|
| **波动率尖峰** | HV > 历史均值 + 3σ | 突发信用事件 | 检查同期新闻/公告——可能是评级下调、诉讼、技术问题 |
| **波动率持续性上升** | HV_20d > HV_60d > HV_120d 持续5日 | 不确定性系统性增加 | 关注——领先利差走阔6-12周 |
| **波动率+利差双升** | 两者同时突破阈值 | 信用质量恶化确认 | 强烈信号——市场同时在两个维度上重新定价 |
| **低波动下突然尖峰** | 此前HV<均值·突然跃升至>均值+2σ | 风险被低估后暴露 | 最危险的信号——市场此前忽视了累积风险 |
| **波动率与基本面背离** | 基本面稳定但波动率飙升 | 市场存在框架未捕捉的担忧 | 分歧信号——需要排查隐性风险 |

**波动率突变评级规则：**

| HV_20日变化幅度 | 评级调整 | 说明 |
|----------------|---------|------|
| < 10% 且趋势稳定 | 无变化 | 正常波动 |
| 10-30% 上升 | 轨道B信号升级→关注 | 不确定性增加，需关注 |
| 30-50% 上升 | 轨道B信号升级→异常 | 显著变化，建议排查原因 |
| > 50% 上升 | 轨道B信号升级→危机 | 剧烈波动，可能触发交叉对撞分歧A |
| 20% 以上下降 | 无变化/降级 | 不确定性收窄——中性或正面信号 |


### 3.4.1 波动率统计量的置信区间规范【统计修正 v0.3.1】

**历史波动率的95%置信区间（基于卡方分布）：**
```
HV的95%置信区间公式（假设收益率近似正态分布）：
  [(n-1) × HV² / χ²₀.₀₂₅(n-1),  (n-1) × HV² / χ²₀.₉₇₅(n-1)]

其中：
├── n = 窗口期内的观测值数量（如20日窗口n=20）
├── HV = 历史波动率（年化）
├── χ²₀.₀₂₅(n-1) = 自由度为n-1的卡方分布2.5%分位数
├── χ²₀.₉₇₅(n-1) = 自由度为n-1的卡方分布97.5%分位数
└── 自由度说明：收益率均值为估计值，消耗1个自由度

计算示例（n=20, HV=35.2%）：
  χ²₀.₀₂₅(19) = 8.907
  χ²₀.₉₇₅(19) = 32.852
  CI = [19 × 0.352² / 32.852,  19 × 0.352² / 8.907]
     = [19 × 0.1239 / 32.852,  19 × 0.1239 / 8.907]
     = [0.0716, 0.2643]
     = [26.8%, 51.4%]（年化波动率）

输出："HV_20d: 35.2% [95% CI: 26.8%–51.4%]"
```

**波动率突变检测规则修订：**
```
旧规则：HV > 历史均值 + 3σ → 波动率尖峰
新规则：当前HV > 滚动窗口HV的95%置信区间上界 → 标记为波动率突变
        → 标注："当前HV = XX%，超出滚动窗口HV的95% CI上界（YY%）"

旧规则：HV_20d > HV_60d > HV_120d 持续5日 → 波动率持续性上升
新规则：HV_20d > HV_60d > HV_120d 且 HV_20d超出HV_60d的95% CI上界
        → 确认趋势具有统计显著性

旧规则：低波动下突然尖峰（此前HV<均值·突然跃升至>均值+2σ）
新规则：此前HV在95% CI内·突然跃升至超出95% CI上界
        → 最危险信号——风险被低估后暴露
```

### 3.4.2 波动率多重假设检验校正（FDR）【统计修正 v0.3.1】

```
波动率FDR校正（与利差FDR校正共用同一框架）：
├── 并行检验来源：
│   ├── N只股票 × 3种波动率模式（尖峰/持续性上升/双升）
│   └── 3种波动率窗口（20日/60日/120日）可选择性合并
├── BH程序：与2.3.2节完全一致→使用Benjamini-Hochberg控制FDR在q=0.05
├── 输出规范：
│   ├── "今日波动率预警数：12个"
│   ├── "经FDR校正后的显著信号数：7个"
│   └── "假阳性风险提示：12个预警中，统计上约0-5个可能是假阳性"
└── 标注规则：与2.3.2节一致
```

### 3.6 波动率分析的核心局限

| 局限 | 说明 | 影响程度 |
|------|------|---------|
| **非上市公司不可用** | 无股价/行情数据，波动率分析完全无法开展 | 高——约50%的发债主体是非上市公司 |
| **窗口期选择偏差** | 不同窗口期给出的信号可能矛盾 | 中——应同时输出多窗口，以趋势而非绝对值为准 |
| **市场噪声误报** | 波动率升高不一定代表信用恶化——可能是行业轮动、流动性冲击 | 中——必须配合利差+成交量交叉验证 |
| **历史波动率滞后于隐含波动率** | HV是对过去的统计，IV才是前瞻性的 | 中——但IV不可获取，只能用HV方向性近似 |
| **极端行情下相关性趋同** | 压力时期所有波动率同步上升，区分度下降 | 低——此时应切换到多因子分析 |

## 四、多因子风险暴露

### 4.1 因子分解框架

在公开数据约束下，使用**简化线性因子模型**分解企业信用利差的驱动因素：

```
目标变量：企业YTM利差（周度数据）
模型结构：
  Spread_i(t) = α_i + β_macro × F_macro(t) + β_ind × F_ind(t) 
                + β_region × F_region(t) + ε_i(t)
  
  其中：
  ├── F_macro(t) = 宏观因子（利率水平/期限利差/信用利差指数）
  ├── F_ind(t)   = 行业因子（行业利差均值/行业指数收益率）
  ├── F_region(t)= 区域因子（区域利差均值——仅国企/城投适用）
  ├── α_i        = 特质alpha——企业自身信用质量的定价
  ├── β_*        = 各因子的暴露系数
  └── ε_i(t)     = 残差——未被因子解释的部分（模型误差+特质冲击）
```

### 4.2 简化单因子回归（公开数据约束下的可行方案）

由于中国企业债数据稀疏（尤其是非活跃券），多因子面板数据的可行性较低。以下为在**有限数据约束下**的可行简化方案：

| 因子 | 代理变量 | 数据源 | 适用性 |
|------|---------|-------|--------|
| **宏观利率** | 10年期国债收益率（月均） | 央行/公开市场数据 | 通用 |
| **宏观信用** | AAA企业债收益率曲线水平 | 中债登免费披露（月度） | 通用（但中债估值曲线精度较低） |
| **行业β** | 行业利差中位数变化 | 行业样本企业等权平均 | 需要≥3家样本企业 |
| **区域β** | 区域国企利差均值（如适用） | 区域样本企业等权平均 | 仅国企/城投——民营企业无区域β |
| **流动性** | 换手率（日均成交量/存量规模） | 交易所公开数据 | 流动性较低的券数据稀疏 |
| **特质α** | 残差项——滚动6个月回归的α | 模型内生 | 需要足够多的周度观测值（≥24周） |

**回归技术规范：**

```
回归类型：滚动OLS（窗口=24周，约6个月）
频率：周度
最小观测值：12周（不足则不输出因子分解）
约束：当样本企业<3家时，不计算行业β

输出格式：
  ├── R²：0.65（宏观+行业因子解释了65%的利差变动）
  ├── β_macro：0.42（利差对宏观利率的敏感度）
  ├── β_ind：0.38（行业利差每扩大10bp，个券利差扩大3.8bp）  
  ├── α：+25bp（特质alpha为正——市场对该企业额外要求25bp溢价）
  └── 残差标准差：18bp（未被解释的部分——作为不确定性度量）

模型诊断：
  ├── Durbin-Watson统计量：1.85（无显著自相关）
  ├── VIF：< 3（无多重共线性）
  └── 残差正态性检验：p > 0.05（通过）
```


### 4.2.1 因子回归的置信区间规范【统计修正 v0.3.1】

**回归系数置信区间（Newey-West稳健标准误）：**
```
所有β系数的输出格式修订（替代原不带置信区间的点估计）：

旧输出格式：
  β_macro：0.42（利差对宏观利率的敏感度）
  β_ind：0.38

新输出格式（附加95%置信区间和显著性标注）：
  β_macro：0.42  [95% CI: 0.28–0.56]  ✅ p=0.003（显著）
  β_ind  ：0.38  [95% CI: 0.22–0.54]  ✅ p=0.012（显著）
  α      ：+25bp [95% CI: -8bp–+58bp] ❌ p=0.124（不显著）

计算规范：
├── 标准误类型：Newey-West异方差自相关一致标准误（HAC）
├── 滞后截断参数：按 Newey-West(1994) 自动选择，或取 floor(4×(T/100)^(2/9))
├── 置信区间：β̂ ± t₀.₀₂₅(df) × SE_HAC
└── 显著性标注：
    ├── p<0.05 → 标注 "✅ p=XX（显著）"
    └── p≥0.05 → 标注 "❌ p=XX（不显著）"
```

**回归R²的置信区间：**
```
R²置信区间（通过F分布转换）：
├── 方法：R² → F统计量 → F分布置信区间 → 回转为R²置信区间
├── 公式：F = (R²/k) / ((1-R²)/(n-k-1))
├── 输出："R²: 0.65 [95% CI: 0.52–0.76]"
└── 前提：n≥30，且残差近似正态（否则使用Bootstrap替代）
```

### 4.2.2 统计前提检验规范【统计修正 v0.3.1】

在进行因子回归之前，必须依次执行以下三项前提检验。任何一项未通过，须按规则处理后方可继续。

**检验一：平稳性检验（ADF）**
```
ADF检验（Augmented Dickey-Fuller）：
├── 检验对象：所有时间序列——目标变量（利差）及所有因子变量
├── 原假设H₀：序列存在单位根（非平稳）
├── 判定标准：
│   ├── p ≤ 0.05 → 拒绝H₀ → 序列平稳 ✅
│   └── p > 0.05 → 无法拒绝H₀ → 序列非平稳 ❌
├── 处理规则：
│   1. 非平稳序列 → 执行一阶差分（ΔX_t = X_t - X_{t-1}）
│   2. 差分后重新执行ADF检验
│   3. 若差分后p≤0.05 → 使用差分后序列进行回归
│   4. 若一阶差分后仍p>0.05 → 标注"数据不满足平稳性假设"，跳过该因子
├── 输出格式：
│   ├── "利差序列：ADF统计量=-3.82, p=0.004 → 平稳 ✅"
│   ├── "宏观因子：ADF统计量=-2.15, p=0.22 → 非平稳 ❌ → 一阶差分后p=0.001 ✅"
│   └── "行业因子：一阶差分后ADF p=0.08 → 仍不平稳 ❌ → 该因子不可用，已跳过"
└── 标注位置：回归输出顶部标注
    "所有序列已通过ADF平稳性检验（或已作差分处理）"
```

**检验二：异方差检验（Breusch-Pagan）**
```
Breusch-Pagan检验：
├── 检验对象：回归模型的残差
├── 原假设H₀：残差方差为常数（同方差）
├── 判定标准：
│   ├── p > 0.05 → 无法拒绝H₀ → 同方差 ✅（使用普通标准误即可）
│   └── p ≤ 0.05 → 拒绝H₀ → 存在异方差 ❌
├── 处理规则：
│   ├── 存在异方差 → 必须使用Newey-West HAC标准误
│   └── 在回归输出中标注 "使用HAC标准误校正异方差"
├── 与β置信区间的关系：
│   └── 无论Breusch-Pagan结果如何，β置信区间统一使用Newey-West标准误
│       （HAC标准误在存在异方差和自相关时均保持一致，不依赖检验结果）
└── 输出格式：
    "Breusch-Pagan检验：BP统计量=18.5, p=0.002 → 存在异方差 ✅ 已使用HAC标准误"
```

**检验三：多重共线性检验（VIF）**
```
VIF（Variance Inflation Factor，方差膨胀因子）：
├── 检验对象：所有自变量（因子变量）
├── 计算方式：对每个因子，将其对其他所有因子做回归，VIF_j = 1/(1-R²_j)
├── 判定标准：
│   ├── VIF < 5 → 无显著共线性 ✅
│   ├── 5 ≤ VIF < 10 → 中度共线性 ⚠️
│   │   └── 标注"VIF=XX，中度共线性——该因子权重已降低"
│   └── VIF ≥ 10 → 严重共线性 ❌
│       └── 标注"VIF=XX，严重共线性——该因子与另一个因子高度重叠，已合并/删除"
├── 处理规则：
│   ├── VIF≥10 → 识别重叠因子对，合并或删除其一后重新计算VIF
│   ├── 合并方式：取两个因子的等权平均作为新因子
│   └── 记录处理过程以供追溯
├── 输出格式：
│   ├── "VIF检验：宏观因子=2.8, 行业因子=3.2, 区域因子=1.5 → 均<5 ✅"
│   └── "VIF检验：行业因子=8.5 ⚠️（中度共线性，权重已降低）"
└── 标注位置：回归诊断部分
```

**三项检验的集成输出规范：**
```
统计前提检验报告：
├── ADF平稳性检验：所有序列已通过（或经差分处理后通过） ✅
├── Breusch-Pagan异方差检验：BP=18.5, p=0.002 → 存在异方差
│   └── 处理：已使用Newey-West HAC标准误
└── VIF多重共线性检验：最大VIF=3.2 → 无显著共线性 ✅
```

### 4.3 因子暴露的时变特征

因子暴露不是静态的——在市场环境变化或企业基本面变化时，因子β会漂移。检测规则如下：

| 信号 | 检测方法 | 含义 |
|------|---------|------|
| **β_macro上升** | 24周滚动β + 趋势回归 | 企业定价越来越受宏观驱动——信用质量向宏观敏感型漂移 |
| **β_ind下降** | 同上 | 市场不再以行业标签定价——个性化风险凸显 |
| **α从负转正** | α窗口突破±2σ | 市场开始对企业额外要求溢价——信用认知恶化 |
| **残差标准差扩大** | 滚动24周残差σ趋势上升 | 模型解释力下降——存在框架未捕捉的因子（隐性风险） |
| **R²急剧下降** | 周度R²从>0.6跌至<0.3 | 定价逻辑发生了结构性变化——因子模型需要重置 |

### 5.5.1 相关性统计量的置信区间规范【统计修正 v0.3.1】

**Pearson相关系数的置信区间（Fisher Z变换）：**
```
Fisher Z变换计算相关系数的95%置信区间：
├── 步骤1：将Pearson r 转换为Fisher Z
│   Z = 0.5 × ln((1+r)/(1-r))
├── 步骤2：Z的抽样标准差
│   SE_Z = 1 / √(n-3)
│   其中 n = 窗口期观测值数量（如24周）
├── 步骤3：Z的95%置信区间
│   CI_Z = [Z - 1.96 × SE_Z,  Z + 1.96 × SE_Z]
├── 步骤4：将CI_Z转换回相关系数尺度
│   CI_r = [(e^(2×CI_Z_low)-1)/(e^(2×CI_Z_low)+1),
│           (e^(2×CI_Z_high)-1)/(e^(2×CI_Z_high)+1)]

计算示例（r=0.62, n=24）：
  Z = 0.5 × ln((1+0.62)/(1-0.62)) = 0.5 × ln(4.263) = 0.725
  SE_Z = 1 / √(21) = 0.218
  CI_Z = [0.725 - 1.96×0.218,  0.725 + 1.96×0.218] = [0.298, 1.152]
  CI_r = [0.290, 0.818]

输出："ρ = 0.62 [95% CI: 0.29–0.82], n=24"
```

**相关性突变检测规则修订：**
```
旧规则：mean_ρ 周度变化 > 0.1 且 连续4周上升 → 预警
新规则：当前相关性与滚动窗口相关性之差的绝对值 > 2 × SE_Fisher
        → 标记为相关性突变
        → 其中 SE_Fisher = √(1/(n₁-3) + 1/(n₂-3))（两个窗口的标准误合并）

输出格式修订：
  旧：mean_ρ = 0.38（+0.05 vs 4周前）
  新：mean_ρ = 0.38 [95% CI: 0.25–0.51]（Δ=+0.05，|Δ| < 2×SE=0.12 → 未达突变阈值 ✅）
```

**相关性矩阵的置信区间标注：**
```
相关性矩阵中的每个ρ值附加置信区间：
  隆22-通22：ρ=0.62 [95% CI: 0.29–0.82]  n=24
  通22-天23：ρ=0.72 [95% CI: 0.44–0.88]  n=24
  隆22-晶澳：ρ=0.35 [95% CI: -0.07–0.66] n=24 ← CI跨越0，不显著

标注规则：
├── CI不包含0 → 标注 "显著相关 ✅"
└── CI包含0 → 标注 "不显著——可能为抽样误差 ❌"
```

**前提条件：**
- n ≥ 10（最小样本量要求，否则Fisher Z变换不稳定）
- n < 10时标注"样本量不足，相关性估计不可靠——仅作方向性参考"

## 六、压力测试与情景分析

### 6.1 压力测试方法论概述

| 压力测试类型 | 定义 | 数据需求 | 引擎可行性 |
|------------|------|---------|-----------|
| **单因子敏感性** | 单个风险因子变化对债券价值的影响 | 低——久期近似即可 | ✅ 完全可行 |
| **多因子情景** | 多个因子同时变化（历史重演/假设性） | 中——需要相关系数矩阵 | ⚠️ 部分可行（简化为加权） |
| **尾部风险** | 极端情景下的损失分布 | 高——需要蒙特卡洛模拟或极值理论 | ❌ 不可行（模拟依赖专业终端） |
| **逆向压力测试** | "什么情景会导致违约？" | 中——需要企业财务结构 | ✅ 基本可行（使用财务比率） |

### 6.2 单因子敏感性分析

在无法获取修正久期和凸性的约束下，使用**近似久期**：

```
近似久期计算（McCaulay Duration）：
  D_mac = Σ(t × CF_t / (1+YTM)^t) / Bond_Price
  
简化版（对固定利率、无含权、非零息）：
  D_approx ≈ 待偿年数 / (1 + YTM)
  
价格变化近似：
  ΔP / P ≈ -D_approx × ΔYTM

示例：
  债券：3年期中票，YTM=3.5%
  D_approx ≈ 3 / (1.035) = 2.90
  如果收益率上行100bp → 价格变化 ≈ -2.90 × 1% = -2.90%
```

**单因子敏感性模板：**

```
压力因子：无风险利率上行 100bp / 200bp
适用的敏感债券：所有固定利率非含权纯债
分析目标：利率上行对债券市场价值的影响

| 债券 | 待偿期限(年) | YTM(%) | 近似久期 | 利率+100bp影响 | 利率+200bp影响 |
|------|------------|--------|---------|--------------|--------------|
| 债券A | 2.0 | 2.80 | 1.95 | -1.95% | -3.90% |
| 债券B | 3.5 | 3.20 | 3.39 | -3.39% | -6.78% |
| 债券C | 5.0 | 3.80 | 4.81 | -4.81% | -9.62% |

局限标注：
  1. 以上为线性近似，不含凸性修正——利率变化越大，误差越大
  2. 未考虑含权特征——可转债不能使用此方法
  3. 含权债的久期被期权调整——实际敏感度低于近似值
```

**支持的敏感性因子清单（按优先级）：**

| 因子 | 标准冲击幅度 | 分析方法 | 数据可得性 |
|------|------------|---------|-----------|
| 无风险利率平移 | +50bp / +100bp / +200bp | 久期近似 | ✅ |
| 信用利差走阔 | +50bp / +100bp / +200bp | 久期近似 | ✅ |
| 评级下调（1-2档） | 依赖历史迁移矩阵 | 利差模型 | ⚠️ 需要评级迁移数据 |
| 流动性冲击 | 交易量萎缩至10% | 定性——无法量化为精确损失 | ❌ 无法量化 |

### 6.3 多因子情景分析

在无法实施蒙特卡洛模拟的约束下，使用**情景加权法**：

```
方法：对每个因子独立计算影响，假设因子间独立（简化假设）
公式：ΔP_total = Σ(ΔP_i × S_i)
  其中 ΔP_i = 第i个因子冲击下的价格变化
       S_i = 该因子在情景中发生的概率（0-1）

缺陷：忽略因子相关性——在压力情景下，因子相关性会系统性上升
       → 标注："独立假设可能低估总影响，误差约10-20%"
```

**标准情景模板：**

| 情景名称 | 无风险利率 | 信用利差 | 评级 | 流动性 | 发生的统计概率 |
|---------|-----------|---------|------|--------|--------------|
| **轻度承压** | +50bp | +50bp | 稳定 | 正常 | 历史概率约20% |
| **中度承压** | +100bp | +100bp | 展望负面/1档下调 | 交易量减半 | 历史概率约10% |
| **严重承压** | +200bp | +200bp | 2档下调 | 交易极不活跃 | 历史概率约3% |
| **极端尾部** | +300bp | +400bp | 多档下调至CCC | 流动性枯竭 | 历史概率约1% |

**多因子情景组合的定量估算：**

```
以3年期中票（YTM=3.5%, 近似久期=2.90）为例：

| 情景 | 利率冲击 | 利差冲击 | 总YTM冲击 | 价格影响估算 |
|------|---------|---------|----------|------------|
| 轻度承压 | +50bp | +50bp | +100bp | -2.90% |
| 中度承压 | +100bp | +100bp | +200bp | -5.80% |
| 严重承压 | +200bp | +200bp | +400bp | -11.60% |
| 极端尾部 | +300bp | +400bp | +700bp | -20.30% |

局限：
  1. 利率+利差独立假设在压力时期不成立——实际冲击可能更大
  2. 未考虑凸性——利率大幅变化时凸性保护 | 方向性正确但量级有偏
  3. 未考虑交易对手风险、追加保证金等非线性影响
```

### 6.4 尾部风险情景构建

尾部风险情景的目标不是"预测"，而是**暴露出组合对特定极端事件的脆弱性**。在量化信用分析中，尾部风险通过以下3种方式构建：

| 构建方法 | 描述 | 示例 | 可行性 |
|---------|------|------|-------|
| **历史重演** | 选取历史上最严重的信用冲击事件 | "2015年股灾+债灾" / "2020年永煤冲击" | ✅ 高（数据可得） |
| **假设性最差** | 本企业可想象的最坏情况 | "核心客户破产+再融资冻结+评级下调至D" | ✅ 高（依赖行业判断） |
| **压力因子叠加** | 多个低概率因子同时发生 | "利率200bp+利差400bp+股价跌50%" | ⚠️ 中（因子相关性不可知） |

**尾部风险情景文档规范：**

```
情景名称：[情景命名——简洁描述核心风险]
情景类型：[历史重演 / 假设性 / 因子叠加]
构建依据：[为什么这个情景是合理的——基于历史或行业逻辑]

因子设定：
├── 因子1：[名称] → [冲击幅度] — [依据]
├── 因子2：[名称] → [冲击幅度] — [依据]
└── 因子3：[名称] → [冲击幅度] — [依据]

受影响敞口：
├── 债券A：[影响描述]
├── 债券B：[影响描述]
└── 组合总影响：[%]

触发条件（什么会触发这个情景）：
├── 可观测先行信号1
├── 可观测先行信号2
└── 可观测先行信号3

历史参考事件：如果有类似历史事件，列出
```

### 6.5 实例：华晨汽车三种情景的损失估计

**分析对象：** 17华汽05（华晨汽车集团控股有限公司发行的公司债）
**分析日期：** 2018-12-31（T-22个月——回溯验证案例）
**数据源：** 2018年年报 + 市场公开数据

**债项基本信息：**
- 债券类型：公司债（无担保）
- 发行金额：10亿元
- 期限：3+2年（含权——发行人调整票面利率+投资者回售）
- 起息日：2017-11
- 当前YTM（2018-12-31）：约7.5%
- 近似久期（含权简化处理）：约2.5年

**输出时含权债的特殊标注：**
> 注：含权债的久期和利差分析精度低于纯债。以下分析基于"假设不行权"的纯债框架，实际风险暴露可能小于估算值（因投资者有权回售）。

**三种情景损失估计：**

| 情景 | 描述 | 核心触发逻辑 | YTM冲击 | 价格影响 | 回售可能性 | 综合评级 |
|------|------|------------|---------|---------|-----------|---------|
| **情景1：温和情景** | 利率上行50bp + 行业利差走阔50bp | 货币政策正常化 + 汽车行业周期下行 | 7.5% → 8.5% | -2.5%（约-250万/亿） | 低 | A |
| **情景2：宝马剥离情景** | 宝马确认增持+母公司信用独立评估 | 2018-10-11 宝马已公告增持计划 | 7.5% → 12.0% | -10.8%（约-1080万/亿） | 中——回售权是保护 | CCC |
| **情景3：违约情景** | 母公司违约——回收率40%（假设） | 核心资产剥离完成 + 再融资冻结 | 回收率场景 | -60%（约-6000万/亿） | 高 | D |

**情景2的详细构建：**

```
情景名称：宝马剥离后的母公司信用重定价
情景类型：假设性（基于2018-10-11公告）
构建依据：宝马增持华晨宝马至75%的公告已发布——该交易一旦完成，
          华晨宝马不再并表，母公司失去核心现金流来源

因子设定：
├── 无风险利率：+50bp（货币政策正常化假设）
├── 信用利差：+400bp（核心资产剥离后的风险重定价）
│   ├── 依据：跟踪评级机构在华晨宝马公告后维持AAA/稳定
│   ├── 但市场已开始重新定价——华晨中国(01114.HK)股价已跌24.54%
│   └── 同案例：中国宏桥(01378.HK)被做空后利差走阔约350bp
├── 股价情景：华晨中国继续下跌——进一步削弱母公司再融资能力
└── 评级情景：预计1-2年内评级将被下调至BB或更低（实际：22个月后违约）

受影响敞口：
├── 17华汽05（10亿）：-10.8%（如果能在违约前回售，损失大幅减少）
├── 母公司其他存续债券：类似影响
└── 组合总影响：视持仓而定

触发条件监控：
├── 宝马增持完成进度（最关键——2022年完成，现实时间线）
├── 母公司单体报表持续恶化（需逐季跟踪）
├── 再融资遇阻（新债发行失败或利率大幅上升）
└── 股价持续下跌（华晨中国股价表现是领先指标）
```

**回溯验证（现实结果）：**

| 预测 | 现实结果 | 偏差 |
|------|---------|------|
| 宝马剥离后母公司风险暴露 | 2020-10 首次违约——准确 | 时间上22个月 |
| 评级将下调 | 违约后AAA→BB——准确 | 但评级滞后严重 |
| 回收率~40% | 破产重整，无担保债权回收率约30-40% | 基本准确 |
| 情景2 YTM=12% | 违约前二级市场YTM实际到20%+ | 低估了恐慌程度——流动性溢价未被充分计入 |

**情景分析的经验教训：**
1. 情景分析可以**识别风险方向**，但通常**低估极端情况下的非线性放大**
2. 流动性溢价的缺失是情景分析中最大的系统性低估因素
3. 含权债券的回售权是重要的风险缓释工具——但不能假设对手一定有能力行权
4. 压力时期的相关性跳升（所有债券一起跌）无法通过单债券情景分析捕捉

### 6.6 压力测试的硬约束

| 约束 | 影响 | 处理方式 |
|------|------|---------|
| 无可转债+含权债的精确定价模型 | 含权债的压力测试误差较大 | 标注为"含权债近似"——建议仅分析纯债部分 |
| 无CDS/CRMW市场数据 | 无法做对冲有效性测试 | 标注"中国市场CDS产品缺失——对冲不可行" |
| 无Bid-ask spread | 流动性冲击无法精确量化 | 使用成交量萎缩百分比近似——标注"近似估算" |
| 无全市场投资组合数据 | 无法做组合层面的VaR/ES | 限定为单券/单企业层面的压力测试 |
| 无法模拟非线性衍生品效应 | 含权条款（回售/赎回）只能定性分析 | 标注"含权债需专业终端精确定价" |

## 七、市场隐含信号提取

### 7.1 信号框架概述

市场隐含信号是从**非信用债资产的价格行为**中提取的信用线索。这些信号虽不直接来自信用利差，但在公开数据环境下是轨道B的重要补充。

```
信号源矩阵：
  ┌─────────────────────────────────────────────────┐
  │                 资产类型                         │
  │       债券    │   股票    │   可转债   │  资金流 │
  ┌──┐──────────┼──────────┼───────────┼─────────┤
  │信│ YTM利差  │ Beta变化  │ 债底价值   │ 北向    │
  │用│ Z-spread  │ 波动率     │ 纯债溢价率 │ 融资    │
  │信│ 换手率    │ 累计收益   │ 转股溢价率 │ 业内    │
  │号│ 发行利率  │ 异常交易   │ 条款价值   │ 关联    │
  └──┴──────────┴──────────┴───────────┴─────────┘
```

### 7.2 可转债转股溢价率→信用担忧

可转债的转股溢价率衡量的不仅是期权价值，也隐含信用信息：

**核心逻辑：** 当市场对发行人的信用质量产生担忧时，可转债的"债底保护"价值上升，推动转股溢价率走高——即使股价不变，纯债部分的折价导致转股溢价率扩大。

| 转股溢价率区间 | 一般含义 | 结合信用分析时的解读 |
|---------------|---------|-------------------|
| 0-20% | 深度价内——接近转股 | 信用风险低——市场愿意转股 |
| 20-40% | 轻度价外——正常范围 | 正常信用定价 |
| 40-60% | 中度价外——关注信号 | 债底折价存在——市场对信用有隐性担忧 |
| **60-80%** | **高价外——需要排查** | **信用风险溢价正在主导定价——纯债部分已被打折扣** |
| >80% | 极端价外——强烈信号 | 几乎无转股可能性——债券按违约风险定价 |

**局限性标注：**
- 转股溢价率受正股价格、剩余期限、波动率、利率水平多因素影响——高溢价率并不总是信用问题
- 可转债的债底保护（回售条款）质量差异大——需要逐券分析保护条款
- 分析可转债信用信号时，必须同时查看正股价格趋势，以排除"仅因股价大涨导致溢价率异常"的干扰

### 7.3 实例：隆22转债溢价率74%

**分析日期：** 2026-07-01
**数据：** 隆22转债价格105.80元，隆基绿能股价18.23元，转股价19.85元，YTM=0.52%

| 指标 | 数值 | 含义 |
|------|------|------|
| 转股价值 | 18.23 / 19.85 × 100 = 91.84元 | 转股亏损约8.16元/张 |
| 转股溢价率 | (105.80 - 91.84) / 91.84 = **15.2%** | 实际计算值——低于市场感知 |
| **纯债价值** | 按3.5%贴现率折现未来现金流 ≈ 92.5元 | |
| **纯债溢价率** | (105.80 - 92.5) / 92.5 = **14.4%** | 债底以上交易——有一定安全垫 |
| **市场报告的转股溢价率** | 通常计算的 **74%** | 注意：此值是基于**面值100元**而非转股价值计算 |

**重要澄清（数据误用风险）：**

> **关键发现：** 市场上常用的转股溢价率（如74%）是基于面值100元而非转股价值的简化计算。在信用分析中，应使用 **转股价值溢价率** = (转债价格 - 转股价值) / 转股价值。隆22的实际转股溢价率为15.2%，暗示信用担忧有限。74%的数值是行业惯例报价指标，并不直接反映信用风险。

**从隆22转债提取的信用信号：**

| 信号 | 取值 | 信用含义 | 信号方向 |
|------|------|---------|---------|
| 转股价值溢价率 | 15.2% | 偏中性——市场定价接近转股价值 | 正常 |
| 纯债溢价率 | 14.4% | 债底有一定保护——非违约担忧 | 正面 |
| YTM | 0.52%（显著低于同评级信用债~3.5%） | 市场对隆基信用质量有信心 | 正面 |
| 债底折价 | 105.80 vs 纯债价值92.5 | 溢价交易——纯债部分未被折价 | 正面 |

**结论：** 隆22转债的信用信号偏正面——市场认为隆基信用风险低，溢价率74%（按面值算）是**计算口径差异**，不应误读为信用担忧。

**对AI Agent的推理标注：**
> 当处理可转债转股溢价率时，必须区分"面值口径溢价率"和"转股价值口径溢价率"——前者用于交易场景，后者才用于信用分析。误用面值溢价率进行信用判断会产生误导性结论。

### 7.4 发行利率变化→市场重定价

新发债券的发行利率是市场对发行人信用的**最实时、最干净的定价信号**——因为它不受二级市场流动性干扰。

| 信号类型 | 检测规则 | 信用含义 |
|---------|---------|---------|
| **同企业新发行利率跳升** | 新发利率 > 上次发行 + 50bp（同期限） | 市场对发行人的信用评价恶化 |
| **发行利率 vs 市场估值** | 发行利率 > 中债估值 + 20bp | 一级市场比二级市场更悲观——关注 |
| **发行失败/取消** | 公告取消或延期 | 严重信号——市场不接受当前定价 |
| **发行利率跨越评级线** | 新发利率 > 低1-2档评级的市场均值 | 市场认为实际信用质量低于名义评级 |
| **同行业新发利率趋势** | 行业新发利率均值连续上升 | 行业整体在承受更高融资成本 |

**实例：通威股份短融发行利率变化（2025-2026）**

| 发行日期 | 品种 | 期限 | 发行利率 | 较上次变化 | 信号 |
|---------|------|------|---------|-----------|------|
| 2025-03 | 短融 | 270天 | 2.60% | — | 基准 |
| 2025-06 | 短融 | 270天 | 2.30% | -30bp | 利差压缩——货币宽松 |
| 2025-09 | 短融 | 270天 | 2.02% | -28bp | 继续压缩 |
| 2026-01 | 短融 | 270天 | 2.15% | **+13bp** | 🟠 **拐点——趋势反转** |
| 2026-04 | 短融 | 270天 | 2.45% | +30bp | 🟠 **连续走阔——确认趋势** |

**分析：** 2025年利率持续下行反映了货币宽松环境（全市场利率下行），但2026年1月的拐点（+13bp）无法被宏观利率解释——2026年初货币市场利率平稳。这表明**信用溢价在独立上升**——即使AAA评级维持，市场已开始重新定价通威的信用风险。

### 7.5 股价持续下跌+波动率放大→领先债市

股票市场比债券市场对信用质量变化更敏感——原因：流动性更高、交易者类型更多元、信息传递更快。

**检测规则：**

```
信号组合：股价持续下跌 + 波动率放大
检测条件：
  1. 股价：过去60日累计跌幅 > 20%
  2. 波动率：HV_20d > HV_60d 且 HV_60d 处于历史80%分位数以上
  3. 成交量：日均成交量 > 历史均值（排除流动性枯竭导致的伪信号）
  4. 🟠 以上全部满足 → 信用预警信号

信号含义：
  ├── 股权市场已在"用脚投票"反映信用担忧
  ├── 债券市场通常滞后2-6周（对定向信息）或 6-12周（对系统性信息）
  └── 信号强度随条件满足的数量递增

交叉验证：
  ├── 检查同期是否有：评级行动 / 业绩预警 / 再融资事件
  ├── 行业内对比：同行业其他企业是否也在下跌 → 排除行业系统性下跌
  └── 基本面确认：轨道A是否支持"信用恶化"的判断
```

**验证案例：华晨汽车（2018-10-11 公告后）**

| 信号 | 数值 | 信用含义 |
|------|------|---------|
| 华晨中国(01114)单日跌幅 | -24.54% | 极端负面——核心资产价值受损 |
| 公告后1个月累计跌幅 | -35% | 跌幅持续——市场信心未恢复 |
| HV_20d(公告前) | 28% | 正常波动 |
| HV_20d(公告后) | 65% | 翻倍+——不确定性激增 |
| 债市反应(2周内) | 利差走阔约35bp | 📌 **债市反应滞后且平缓——股票市场领先了约8-12周** |

### 7.6 北向资金/融资余额→机构态度

| 信号 | 数据源 | 可得性 | 对信用分析的指示 |
|------|-------|--------|-----------------|
| **北向资金（陆股通）持有量变化** | 港交所每日披露 | ✅ 免费 | 外资机构对股票的持续态度——间接反映对信用质量的看法 |
| **融资余额变化** | 交易所每日披露 | ✅ 免费 | 杠杆投资者的风险偏好——快速撤出可能预警 |
| **大宗交易折溢价** | 交易所披露 | ⚠️ 部分免费 | 大额交易方对公允价值的判断——但通常视为机构调仓 |
| **大股东质押比例变化** | 公司公告 | ✅ 公开 | 大股东资金链紧张——公司层面的关联风险 |

**北向资金用于信用分析的方法：**

```
信号提取：
  1. 获取目标企业（如上市）陆股通持仓数据（日度）
  2. 计算持仓市值变动 = 当日持仓市值 - 前日持仓市值
  3. 分解为：价格效应 + 实际买卖
     实际买卖 = 持仓市值变动 - (持仓量 × 股价变动)
  4. 检测：
     ├── 连续5日净卖出（实际买卖为负）→ 🟡 外资在减持
     ├── 周度持仓占比下降 > 10% → 🟠 外资态度明显转淡
     └── 北向持仓占比创3个月新低 → 🔴 机构信心不足

局限标注：
  ├── 仅适用于沪港通/深港通标的——非所有发债企业都有北向数据
  ├── 北向买卖可能有非信用原因（组合再平衡、行业轮动、汇率对冲）
  └── 北向资金在中国A股占比约3-5%——信号强度有限
```

| 市场隐含信号 | 数据源 | 领先债市窗口 | 可靠性 | 可用性 |

## 八、输出标准与局限

### 8.1 每项指标的输出规范

定量分析中的**每一项指标**在输出时必须附带以下要素：

```
[指标名称]
├── 当前值：[数值]
├── 数据源：[来源描述]
├── 计算窗口：[时间窗口 / 观测值数量]
├── 质量标注：[高质量 / 中等 / 低质量 / 替代方案]
├── 趋势：[上升 / 下降 / 稳定 / N/A]
├── 阈值判断：[正常 / 关注 / 异常 / 危机]
├── 历史分位数：[%——如有足够历史数据]
└── 定性关联：[该指标的变化在定性上意味着什么]
```

**质量标注规则：**

| 质量等级 | 标注条件 | 在报告中的表示 |
|---------|---------|--------------|
| **高质量** | 数据来自权威来源、序列完整、计算无替代假设 | ✅ 无标注 |
| **中等** | 数据来自公开但权威性不足、序列有少数缺失 | ⚠️ 附带"中等质量"标注 |
| **低质量** | 替代方案计算、序列大量缺失、活跃度不足 | ⚠️ 附带"低质量——仅供参考"标注 |
| **替代方案** | 精确指标不可获取，使用近似替代 | 📌 附带替代方案说明及预期误差范围 |
| **不可获取** | 完全无法获取或需要付费终端 | ❌ 明确标注"不可获取"并说明原因 |

**输出示例（完整标注）：**

```
指标：YTM利差
当前值：223bp
数据源：交易所收盘净价 + 中债登国债收益率曲线（线性插值）
计算窗口：2026-01-01 至 2026-07-01（126个交易日）
质量标注：⚠️ 中等质量——Z-spread不可获取，YTM利差未考虑期限结构
趋势：最近20日 +18bp（上升中）
阈值判断：🟠 异常——超过同行业历史80%分位数
历史分位数：82%（vs 自身过去1年）
定性关联：YTM利差持续走阔与公司2025年预亏公告的时间线一致，
          市场已在重新定价信用风险
```

### 8.2 数据质量矩阵汇总

| 指标 | 质量等级 | 核心局限 | 建议替代/处理 |
|------|---------|---------|-------------|
| YTM利差 | 中等 | 非活跃券价格失真；不含权调整 | 仅对活跃券使用（周成交≥1笔） |
| Z-spread | ❌ 不可获取 | 需要专业终端的现金流建模 | 使用YTM利差+期限近似+标注误差 |
| OAS | ❌ 不可获取 | 需要含权定价模型 | 放弃——或对纯债使用Z-spread近似 |
| 修正久期 | 中等 | 近似计算不包含含权调整 | 固定利率纯债可行；可转债/含权债不可用 |
| 凸性 | ❌ 不可获取 | 需要精确现金流定价 | 标注"凸性不可获取"——不估算 |
| 历史波动率(HV) | 中高 | 滞后指标；非上市公司不可用 | 配合利差趋势做交叉验证 |
| 隐含波动率 | ❌ 不可获取 | 中国信用债/可转债期权市场缺失 | 使用HV方向性近似 |
| Bid-ask spread | ❌ 不可获取 | 中国债券市场不透明 | 使用换手率/成交频率作为流动性代理 |
| 北向资金持仓 | 中高 | 仅沪港通标的可用 | 对非港股通企业标记"无数据" |
| 融资余额 | 中 | 仅两融标的 | 对非两融标的标记"无数据" |
| 发行利率 | 中高 | 仅在有新发时可用 | 二级市场利差作为连续替代 |
| 评级迁移 | 中 | 滞后性严重 | 使用轨道A做前瞻验证而非依赖 |
| 成交量/换手率 | 中等 | 非活跃券数据稀疏 | 活跃券的成交量变化比绝对量更有意义 |

### 8.3 公开数据核心局限汇总

本节汇总所有已在各处标注的数据局限，作为单一参考位置：

| 局限类别 | 具体内容 | 对分析的影响 | 是否可能通过数据源升级解决 |
|---------|---------|-------------|-------------------------|
| **Z-spread/OAS缺失** | 无法获取任何专业终端的利差分解数据 | 无法精细分解信用风险、利率风险和期权风险 | ✅ 接入Wind/Bloomberg后解决 |
| **Bid-ask spread缺失** | 中国债市不透明，做市商报价不可获取 | 无法评估真实交易成本和流动性 | ⚠️ 部分——接入外汇交易中心数据后可能 |
| **非上市公司无股价** | 50%+发债主体是非上市公司 | 波动率分析、股价信号完全不可用 | ❌ 结构性——非上市主体无公开股价 |
| **可转债定价模型缺失** | 不能计算纯债价值、期权价值、理论定价 | 可转债的信用信号提取受限于简化计算 | ✅ 接入专业终端+QuantLib可解决 |
| **中国CDS市场缺失** | 信用违约互换在中国几乎不存在 | 无法从信用衍生品中提取信用风险溢价 | ❌ 市场基础设施问题——非数据获取 |
| **ES/ETL数据不标准** | 无标准化的ESG评分、碳排放数据（公开） | 无法将ESG因子纳入量化模型 | ⚠️ 部分——可接入第三方ESG评分 |
| **高频数据缺失** | 分钟级/逐笔成交数据需付费 | 无法做高频波动率、微观结构分析 | ✅ 接入付费数据后可解决 |
| **非活跃券数据稀疏** | 大量债券数日无成交 | 利差计算失真——基准方法不适用 | ❌ 市场流动性问题——非数据获取 |
| **母子公司数据隔离** | 合并报表掩盖母公司风险（验证案例已反复证实） | 因子分析中的"企业"粒度需要精确到发债主体 | ⚠️ 需要手动识别发债主体——数据源升级不能完全解决 |
| **条款数据非标准化** | 债券条款分散在募集说明书中 | 无法批量做含权分析 | ⚠️ 部分——LLM提取条款后可结构化 |

### 8.4 替代方案汇总表

当精确指标不可获取时，替代方案及效果评级：

| 不可获取指标 | 替代方案 | 效果评级 | 适用条件 | 误差范围 |
|------------|---------|---------|---------|---------|
| Z-spread | YTM - 同期限国开债（线性插值） | ⭐⭐⭐ 中等 | 纯债、非含权、活跃券 | ±15-30bp |
| OAS | 不做替代——标注不可获取 | ❌ | N/A | N/A |
| 修正久期 | Macaulay Duration近似（D/(1+YTM)） | ⭐⭐⭐ 中等 | 纯债、固定利率 | 误差随YTM增大 |
| 凸性 | 不做替代——标注不可获取 | ❌ | N/A | N/A |
| 隐含波动率 | 历史波动率(HV_20d) | ⭐⭐ 低 | 仅方向性参考 | 滞后+精度不足 |
| Bid-ask spread | 日均成交量+换手率 | ⭐⭐ 低 | 方向性流动性判断 | 无法量化交易成本 |
| 个股高频数据 | 日度收盘价+日收益率 | ⭐⭐⭐ 中等 | 日度分析足够 | 无法做微观结构 |
| 非上市公司财务 | 招股书+司法+招标+招聘+工商 | ⭐⭐⭐ 中等 | 多源交叉验证 | 取决于数据丰富度 |
| 北向资金（非标的） | 不替代 | ❌ | N/A | N/A |
| 尾部风险(VaR/ES) | 情景加权法（见第六章） | ⭐⭐ 低 | 方向性暴露判断 | 无统计精度保障 |

### 8.6 轨道B定量分析→轨道A的映射关系

定量分析的最终输出需要映射到轨道B的四级信号体系，并进入交叉对撞矩阵：

```
L0.1 + L0.2 + L0.3 → 轨道B四级信号 → 交叉对撞 → 综合评级

定量层（本文档）        四级信号层        双轨对撞层
├── YTM利差水平          ├── 利差平静      ├── 共识
├── 波动率状态           ├── 利差关注      ├── 分歧A（A好B差）
├── 因子漂移检测          ├── 利差异常      └── 分歧B（A差B好）
├── 相关性结构           └── 利差危机
├── 压力测试
├── 市场隐含信号
└── 数据质量标注

四级信号判定规则（一键映射）：
  利差稳定或收窄 + 波动率低 + 因子稳定   → 平静
  利差走阔20-50bp + 波动率上升           → 关注
  利差>50bp跳升或持续走阔 + 波动率大幅上升 → 异常
  曲线倒挂/流动性枯竭 + 利差极端走阔      → 危机
```

### D. 统计方法使用检查清单【统计修正 v0.3.1】

每次运行定量分析前，AI Agent 必须逐项确认以下统计前提条件。每项确认结果（✅ / ⚠️ / ❌）必须在输出中显式标注，不得省略。

```
□ 时间序列是否已做平稳性检验（ADF）？
   → 标注 "ADF p=X.XXX → 平稳/非平稳（已差分处理/跳过）"

□ 回归残差是否已做异方差检验（Breusch-Pagan）？
   → 标注 "BP p=X.XXX → 同方差/异方差（已使用HAC标准误）"

□ 多因子模型是否已做多重共线性检验（VIF）？
   → 标注 "最大VIF=X.X → 无共线性/中度共线性权重已降低/严重共线性已合并"

□ 相关性估计是否附带置信区间（Fisher Z变换）？
   → 标注 "ρ=X.XX [95% CI: X.XX–X.XX]"

□ 波动率估计是否附带置信区间（卡方分布）？
   → 标注 "HV=X.X% [95% CI: X.X%–X.X%]"

□ 利差估计是否附带置信区间（Bootstrap）？
   → 标注 "利差均值=X [95% CI: X–X]"

□ 多重假设检验是否做了FDR校正（Benjamini-Hochberg）？
   → 标注 "原始信号数=X，FDR校正后显著数=X，q=0.05"

□ 样本量是否≥30（最小回归样本量）？
   → 若 n<30，标注 "样本量不足（n=X）——回归结果仅作方向性参考"

□ 尾部概率是否标注为"排序工具"而非"精确概率"？
   → 标注 "尾部概率为排序工具，非精确统计概率"

□ 测试是否包含假阳性率和假阴性率（不能只用成功案例）？
   → 标注 "假阳性率估计：X%（经FDR校正），假阴性率估计：X%（基于检验功效）"
```

**使用规则：**
1. 在每次定量分析输出的末尾，必须附带此检查清单的完成状态
2. 未通过的项目必须附带处理说明——"未做"与"已做但未通过"的标注不同
3. 此清单是质量控制门禁——所有项目均为✅时，分析方可进入轨道B四级信号映射
4. 任何一项为❌时，该分析须标注"统计前提未完全满足——结论置信度受限"
