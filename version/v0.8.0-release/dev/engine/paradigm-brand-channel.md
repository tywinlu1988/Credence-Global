**版本**: v0.8.0-release | **范式版本**: v1.0.0 | **日期**: 2026-07-10 | **状态**: 初版发布

## 一、范式定义与适用范围

### 1.1 什么是"品牌+渠道型"行业

品牌+渠道型行业的核心特征是企业通过**品牌认知度**建立消费者心智壁垒，通过**渠道网络**实现产品触达，从而将品牌溢价转化为可持续的财务回报。这类行业的信用逻辑与政策驱动型、技术壁垒型有本质区别。

**典型行业：**
- 白酒、乳制品、调味品、软饮料、休闲食品、预制菜
- 日化用品、化妆品、家清产品
- 品牌服装、运动鞋服
- 家电品牌（格力/美的/海尔等成熟品牌）

**引擎映射：** 食品饮料、纺织服装在 `industry-framework.md` 中归类为 **品牌+渠道型** 范式。

### 1.2 判定条件

一个行业被归入"品牌+渠道型"需同时满足以下条件：

| 条件 | 判定标准 | 说明 |
|---|---|---|
| **B1 品牌依赖度** | 消费者在购买决策中品牌认知权重 > 50% | 消费者愿意为品牌支付溢价 |
| **B2 渠道广度** | 产品通过多级分销网络触达终端，非直营为主 | 经销/分销体系是核心资产 |
| **B3 转换成本** | 消费者转向竞品的心理/习惯成本高 | 品牌忠诚度可测量 |
| **B4 定价权** | 企业拥有主动提价能力（非完全价格接受者） | 毛利率水平稳定或持续提升 |
| **B5 产品同质化** | 产品本身的技术差异有限，差异化来自品牌 | 非技术驱动型行业 |

> **排除规则：** 凡产品技术差异是消费者决策的首要因素（如半导体、药品、高端装备），不适用本范式。

## 二、范式核心

### 2.1 最重因子：品牌定价权

**品牌定价权是品牌+渠道型行业信用分析的起点和终点。**

品牌定价权不是定性判断，而是可以通过以下指标量化的：

| 指标 | 计算公式/数据来源 | 强信号 | 弱信号 | 危险信号 |
|---|---|---|---|---|
| 毛利率水平 | (收入-营业成本)/收入 | > 60% | 20-60% | < 20% |
| 毛利率稳定性 | 近5年毛利率标准差 | < 3pp波动 | 3-8pp波动 | > 8pp波动或持续下降 |
| 提价频率与幅度 | 年度均价变动 | 每年提价2-5%且销量不降 | 3年提价一次 | 被迫降价促销 |
| 消费者NPS | 第三方调研 | > 60 | 30-60 | < 30 |
| 复购率 | 会员/销售数据 | > 50% | 20-50% | < 20% |
| 市占率趋势 | 行业数据 | 持续提升 | 维持稳定 | 持续下降 |

**毛利率是品牌力的直接映射。** 在品牌+渠道型行业中：
- **90%+毛利率** = 超强品牌定价权（如贵州茅台90%+，品牌即品类）
- **60-90%毛利率** = 强品牌护城河（如高端化妆品、奢侈食品）
- **30-60%毛利率** = 品牌有溢价但竞争激烈（如乳业、调味品龙头）
- **< 30%毛利率** = 品牌力弱或商品化严重（如基础食材、白牌产品）

### 2.2 次重因子：渠道控制力

渠道控制力决定品牌溢价能否转化为实际收入和现金流。

**为什么渠道控制力比产品组合更重要？**
在品牌+渠道型行业，产品可以被模仿（如蒙牛vs伊利的产品线高度重叠），但渠道网络是长期积累的、不可复制的核心资产。伊利500万+终端网络的建立需要数十年的投入和运营积累，竞争对手无法短期复制。因此渠道控制力的权重（25%）高于产品组合（20%）。

**品牌定价权与渠道控制力的协同效应：**
最强的品牌+渠道型企业呈现"正循环"：品牌力强→经销商愿意预付款→渠道利润丰厚→经销商忠诚度高→终端覆盖密度提升→消费者触达增加→品牌认知强化→品牌力更强。反之，品牌力弱的企业陷入"负循环"：品牌力弱→经销商要求赊账→应收账款恶化→渠道利润薄→经销商不稳定→终端覆盖下降→品牌认知弱化。

**渠道评估三维模型：**

**维度一：经销网络的广度和深度**
- 广度：覆盖城市/县域数量、终端网点数量、KA渠道渗透率
- 深度：层级结构（省代→市代→县代→终端）、每个层级的利润分配
- 线上覆盖：电商平台渗透率、直播/社交电商布局、O2O履约能力

**维度二：渠道依赖度与风险分散**
- 单一渠道收入占比（> 50%为危险）
- 前五大经销商收入占比（> 30%为关注）
- 线上 vs 线下比例（极端依赖单一渠道）

**维度三：账期博弈能力**
- 应收账款周转天数（越短→渠道话语权越强）
- 预收账款/合同负债（越多→渠道愿意提前付款→品牌力强）
- 经销商保证金制度（有保证金→经销商被锁定）

> **关键洞察：** 在品牌+渠道型行业，**预收账款（合同负债）是品牌力的最直接财务证明。** 消费者/经销商愿意提前付款 = 品牌稀缺 = 渠道控制力强。

### 2.3 特有风险：品牌危机

品牌+渠道型行业拥有其他行业几乎不存在的特殊风险——**品牌价值的突然归零**。

**品牌危机类型与影响路径：**

| 危机类型 | 典型案例 | 影响路径 | 可恢复性 |
|---|---|---|---|
| **食品安全丑闻** | 三聚氰胺（三鹿）、丙二醇（麦趣尔） | 消费者信任归零→终端拒购→渠道退货→收入断崖→供应商挤兑 | 极低（品牌彻底死亡） |
| **创始人/实控人丑闻** | 权健事件、董事长被调查 | 品牌形象崩塌→渠道观望→终端动销停滞 | 低 |
| **核心品牌被替代** | 品类消费趋势变化 | 新品类崛起→老品牌被边缘化→渠道转移资源 | 低-中 |
| **经销商大规模解约** | 渠道政策失误、经销商利益受损 | 渠道网络崩塌→产品无法触达终端→收入崩溃 | 低-中 |
| **舆情危机** | 广告翻车、价值观争议 | 消费者抵制→品牌声誉受损 | 中-高（取决于事件性质） |

### 2.4 产品生命周期风险

品牌+渠道型企业面临"品类过时"的独特风险——当消费趋势变化导致整个品类被替代时，品牌本身的价值也随品类消亡。

**为什么品类过时是品牌+渠道型的特有风险？**
在光伏/半导体等产业债中，技术路线可以被迭代（如TOPCon替代PERC），但市场对"能源"或"计算"的需求本身不会消失。而在消费品领域，整个品类可能被消费趋势淘汰——碳酸饮料让位于无糖茶、方便面让位于外卖、传统调味品让位于复合调味料。当品类消亡时，即使该品类的龙头品牌拥有最强的渠道网络，也无法阻挡品类的结构性衰退。

**品类生命周期阶段判断：**

| 阶段 | 特征 | 典型品类 | 对信用分析的影响 |
|---|---|---|---|
| 导入期 | 增速 > 20%，渗透率 < 10% | 预制菜(2020-2023)、植物蛋白饮料 | 高成长但确定性低 |
| 成长期 | 增速 10-20%，渗透率快速提升 | 高端白酒(2016-2021)、无糖茶 | 最佳信用窗口期 |
| 成熟期 | 增速 0-10%，渗透率见顶 | 白酒整体、乳业、酱油 | 信用分化加剧，龙头受益 |
| 衰退期 | 负增长，品类被替代 | 碳酸饮料(长期趋势)、方便面(结构性下滑) | 需一票否决预警 |

## 三、金字塔权重

### 3.1 标准四层金字塔

| 层级 | 权重 | 核心指标 | 一票否决 |
|---|---|---|---|
| **L1 品牌护城河** | 35% | 品牌认知度·定价权(毛利率稳定性)·市占率趋势·消费者NPS/复购率·品牌历史与声誉积累 | 核心品牌被替代 / 重大食品安全事件导致品牌价值归零 |
| **L2 渠道控制力** | 25% | 经销商网络广度与深度·渠道依赖度·应收账款周转·预收/合同负债·渠道库存健康度 | 经销商大规模解约（单年 > 30%经销商流失） |
| **L3 产品组合** | 20% | 产品线宽度·新品迭代速度·单品依赖度·品类生命周期阶段·SKU健康度 | 单一产品收入占比 > 70% 且该品类进入衰退期 |
| **L4 财务质量** | 20% | 经营现金流/净利润·有息负债率·现金储备·ROIC·营运资金效率(DSO/DIO/DPO/CCC)·债务到期排程 | 经营性现金流连续3年无法覆盖利息支出 |

### 3.2 与传统范式的关键差异

| 对比维度 | 品牌+渠道型 | 政策驱动型(光伏) | 技术壁垒型(半导体) | 存量博弈型(NEV) |
|---|---|---|---|---|
| **最重层** | L1品牌35% | L1政策35% | L1地缘政治30-35% | L1生存位势25% |
| **次重层** | L2渠道25% | L2技术30% | L2技术25-30% | L2技术20% |
| **财务层权重** | 20% | 15% | 5-10% | 15-17% |
| **产业政策层** | 无（政策影响间接） | 有（L1核心） | 有（L4辅助） | 有（L5辅助） |
| **品牌危机否决** | 有（特有风险） | 无 | 无 | 无 |
| **食品安全否决** | 有 | 无 | 无 | 无 |
| **技术淘汰否决** | 不适用（品类替代） | 有 | 有 | 有 |

**财务层权重提升的逻辑：**
- 品牌型企业通常现金流强劲（预收模式、高毛利率），财务分析有实际意义
- 优秀的品牌企业可以长期维持低负债甚至零有息负债（如茅台）
- 财务质量直接反映品牌力和渠道力的"变现"效率
- 不同于光伏/半导体行业（财务层权重5-15%），品牌型企业的财务数据更可信、更具区分度

### 3.3 权重调整规则

| 调整条件 | 调整内容 | 适用场景 |
|---|---|---|
| 品类处于导入/成长期 | L3产品组合权重提升至25%，L1降至30% | 渗透率快速提升阶段，产品迭代和品类选择更重要 |
| 行业处于消费降级周期 | L1品牌权重降至30%，L4财务提升至25% | 消费者对价格更敏感，现金流韧性更重要 |
| 线上渠道占比 > 50% | L2渠道拆分出"数字化渠道"子层 | 电商渠道逻辑与线下经销差异大 |
| 出口占比 > 30% | 增加5%外部环境层（贸易政策/汇率） | 品牌出海面临额外的地缘/汇率风险 |

## 四、一票否决详解

### 4.1 一票否决条件总表

| 条件 | 触发标准 | 本质风险 | 可否豁免 |
|---|---|---|---|
| **V1 重大食品安全事件** | 被国家市场监管总局通报/处罚，或引发全国性媒体负面报道 | 消费者信任归零 | 不可豁免（品牌彻底死亡） |
| **V2 核心品牌被替代** | 核心品类/品牌连续4个季度收入同比降幅 > 30%且无替代品牌接力 | 产品性死亡 | 不可豁免 |
| **V3 经销商大规模解约** | 单年经销商流失率 > 30% 或 前五大经销商中超过2家终止合作 | 渠道性死亡 | 分析师委员会裁定 |
| **V4 核心产品过时** | 单一产品收入占比 > 70%且该品类行业增速连续3年为负 | 品类性死亡 | 分析师委员会裁定 |
| **V5 控股股东/实际控制人严重失信** | 实控人被列入失信被执行人/被立案调查/涉及重大法律纠纷 | 治理性死亡 | 分析师委员会裁定 |
| **V6 审计否定/无法表示意见** | 年度审计报告被出具否定意见或无法表示意见 | 信息性死亡 | 不可豁免 |
| **V7 现金流枯竭** | 经营现金流连续3年为负且现金/短期债务 < 0.5x | 现金流性死亡 | 不可豁免 |

### 4.2 一票否决触发后的处理流程

```
触发一票否决条件
    ↓
自动锁定评级上限 CCC
    ↓
分析师委员会 48 小时内召开紧急会议
    ├── 确认触发：维持 CCC 及以下，启动退出/回收程序
    └── 裁定豁免（仅 V3/V4/V5 可豁免）：记录豁免理由，评级上限提升至 B-
    ↓
通知相关利益方（银行/债券持有人）
    ↓
纳入信用事件监控清单，每月更新
```

## 五、双轨分析映射

### 5.1 轨道A：金字塔评分

使用四层金字塔加权计算综合评分：
- L1 品牌护城河 35%
- L2 渠道控制力 25%
- L3 产品组合 20%
- L4 财务质量 20%

### 5.2 轨道B：市场定价信号

品牌+渠道型行业的特有市场信号：

| 信号类型 | 观察指标 | 品牌消费行业特有解读 |
|---|---|---|
| **产品市场信号** | 终端动销率、渠道库存周转、产品保质期折价率 | 产品保质期短的行业（乳业、生鲜），渠道库存是**即时信号** |
| **渠道信号** | 经销商打款意愿、预收账款变化 | 经销商打款是品牌信心的"用脚投票" |
| **消费者信号** | 电商复购率、社交媒体口碑、百度指数/微信指数 | 社交媒体舆情是品牌危机的**最早预警** |
| **资本市场信号** | 股价波动、北向资金、信用利差 | 消费龙头通常享有估值溢价，溢价缩小即品牌信心下降 |

### 5.3 交叉对撞示例

**常见分歧：**
- 轨道A好 + 轨道B差 = 短期市场过度悲观，基本面未变（如白酒行业政策调控时的误杀）
- 轨道A差 + 轨道B好 = 市场未充分定价品牌衰退风险（如品类进入衰退期但市场仍按成长股估值）

**对撞关键追问：**
1. 品牌价值是否真实（毛利率是否可持续）？
2. 渠道库存是否健康（压货 vs 真实动销）？
3. 品类是否被替代（消费趋势是否逆转）？
4. 财务数据是否匹配品牌故事（高毛利率→高现金流→低负债是否成立）？

## 七、品牌力评估的量化方法

### 6.1 品牌力评分卡（B-Score）

品牌力评分卡用于将定性的品牌认知转化为可量化的评分。每个维度1-10分，最终加权得到品牌力B-Score。

| 维度 | 权重 | 评分标准（1-10分） | 数据来源 |
|---|---|---|---|
| **毛利率水平** | 25% | 1分= &lt;10%; 3分=10-30%; 5分=30-60%; 7分=60-90%; 10分=&gt;90% | 年报 |
| **毛利率稳定性** | 20% | 1分=年波动&gt;10pp; 5分=波动3-8pp; 10分=波动&lt;3pp | 年报·5年标准差 |
| **提价能力** | 20% | 1分=被迫降价; 5分=可跟随通胀提价; 10分=主动提价且销量不降 | 年报均价+销量数据 |
| **市占率趋势** | 15% | 1分=持续下降; 5分=维持稳定; 10分=持续提升 | 行业报告 |
| **消费者NPS** | 10% | 1分=&lt;10; 5分=30-50; 10分=&gt;70 | 第三方调研 |
| **品牌历史与声誉** | 10% | 1分=品牌成立&lt;5年; 5分=10-30年老牌; 10分=&gt;50年国民品牌 | 公开信息 |

**B-Score计算公式：** B = (毛利率×0.25) + (稳定性×0.20) + (提价能力×0.20) + (市占率×0.15) + (NPS×0.10) + (历史×0.10)

**B-Score解读：**
- B-Score &gt; 8.0: 强品牌护城河（对应L1评分8-10分）
- B-Score 5.0-8.0: 中等品牌力（对应L1评分5-7分）
- B-Score &lt; 5.0: 弱品牌力（对应L1评分1-4分，需警惕一票否决风险）

### 6.2 渠道控制力评分卡（C-Score）

| 维度 | 权重 | 评分标准 | 数据来源 |
|---|---|---|---|
| **终端覆盖密度** | 25% | 1分=&lt;1万终端; 5分=10-100万; 10分=&gt;500万 | 年报+行业数据 |
| **应收账款周转天数** | 20% | 1分=&gt;90天; 5分=30-60天; 10分=&lt;10天(预收模式) | 年报附注 |
| **预收/合同负债营收比** | 20% | 1分=&lt;1%; 5分=5-15%; 10分=&gt;20% | 年报 |
| **渠道集中度** | 15% | 1分=前五大经销&gt;50%; 5分=20-50%; 10分=&lt;20% | 年报/调研 |
| **渠道利润空间** | 10% | 1分=渠道无利润; 5分=合理; 10分=丰厚（出厂价vs终端价差异大） | 渠道调研 |
| **线上覆盖能力** | 10% | 1分=无线上的; 5分=有电商但一般; 10分=线上线下全渠道领先 | 电商平台/年报 |

### 6.3 评分卡在金字塔中的使用

品牌力B-Score和渠道控制力C-Score分别对应L1和L2的核心量化输入，与L3产品组合评估和L4财务评估共同构成四层金字塔的完整评分体系。建议在每份Type 1报告中至少使用两个评分卡进行量化评分。

## 八、验证案例

| 层级 | 权重 | 评分 | 加权 | 依据 |
|---|---|---|---|---|
| L1 品牌护城河 | 35% | **10/10** | 3.50 | 毛利率91.5%+持续提价能力+品牌即品类+金融属性 |
| L2 渠道控制力 | 25% | **9/10** | 2.25 | 经销商预付款模式+合同负债超百亿+渠道利润丰厚 |
| L3 产品组合 | 20% | **8/10** | 1.60 | 飞天茅台绝对主力+系列酒成长+但单品依赖度高 |
| L4 财务质量 | 20% | **10/10** | 2.00 | 0有息负债+净现金+经营现金流/净利润 > 1.0x+ROIC > 30% |
| **综合** | **100%** | — | **9.35** | **等价评级：AA** |

**验证结论：** 品牌+渠道型范式在茅台身上得到完美验证。四项评分均 ≥ 8分，品牌和财务达满分。茅台是品牌+渠道型企业的"理想型"——品牌不可替代、渠道控制力极强、财务质量无懈可击。

### 7.2 中性验证：伊利股份（600887）

| 层级 | 权重 | 评分 | 加权 | 依据 |
|---|---|---|---|---|
| L1 品牌护城河 | 35% | **7/10** | 2.45 | 双位数市占率领先+但毛利率仅30%+乳业竞争激烈+品牌力弱于白酒 |
| L2 渠道控制力 | 25% | **8/10** | 2.00 | 500万+终端覆盖+冷链领先+但渠道库存管理压力大 |
| L3 产品组合 | 20% | **6/10** | 1.20 | 液态奶贡献65%+新品迭代好+但品类成熟+蒙牛贴身竞争 |
| L4 财务质量 | 20% | **7/10** | 1.40 | 现金流稳健+有息负债率适中+但ROIC约15%低于茅台+冷链资本开支 |
| **综合** | **100%** | — | **7.05** | **等价评级：BBB+** |

**验证结论：** 伊利在品牌+渠道型范式下评分7.05，反映出"强渠道+中品牌"的特征。渠道控制力（8分）是核心优势，品牌护城河（7分）因毛利率仅30%+而受限。这是典型的"中性"质量消费龙头画像。

### 7.3 困境验证：ST龙大（龙大美食，002726）

| 层级 | 权重 | 评分 | 加权 | 依据 |
|---|---|---|---|---|
| L1 品牌护城河 | 35% | **2/10** | 0.70 | 品牌力弱(屠宰加工为主)+毛利率仅4.5%+无定价权+连续3年巨亏 |
| L2 渠道控制力 | 25% | **2/10** | 0.50 | 子公司停产+经销商萎缩+应收账款恶化+渠道崩塌 |
| L3 产品组合 | 20% | **3/10** | 0.60 | 产品线单一(生猪屠宰占大头)+品类处于下行周期+无新品迭代 |
| L4 财务质量 | 20% | **1/10** | 0.20 | 资产负债率85.68%+经营现金流为负+现金枯竭+7亿转债即将违约 |
| **综合** | **100%** | — | **2.00** | **等价评级：CCC** |

**验证结论：** ST龙大完美触发了范式的多个一票否决条件（V7现金流枯竭、V3渠道崩塌），综合评分仅2.00。范式有效识别了"品牌消失+渠道崩塌+财务枯竭"的三重死亡螺旋。

## 十、子行业差异化分析指南

### 9.1 白酒子行业

| 特征 | 信用分析侧重 |
|---|---|
| 高毛利率（60-90%+）| 品牌护城河权重可提升至40%，毛利率稳定性是核心 |
| 品类成熟但高端化升级 | 关注"量减价升"结构变化，需区分高端/次高端/中低档 |
| 金融属性（老酒+收藏）| 金融属性是额外的"需求蓄水池"，但监管政策可能收紧 |
| 经销商预付款模式 | 合同负债是最直接的品牌信心指标 |
| 无保质期问题 | 渠道库存压力远低于乳业/生鲜，但需关注年份酒真实库存 |

### 9.2 乳制品子行业

| 特征 | 信用分析侧重 |
|---|---|
| 毛利率30-40%（受原奶周期影响）| L4财务权重建议提升至25%（原奶成本波动大） |
| 保质期短（7-30天）| 渠道库存管理是核心运营风险，需关注产品报废率和退货率 |
| 双龙头格局+同质化 | 品牌差异化有限，渠道控制力（冷链）是核心竞争壁垒 |
| 资本开支持续（牧场+冷链）| 自由现金流评估比净利润更关键，需做capex压力测试 |
| 低温vs常温结构 | 低温奶增速快但损耗高，需评估企业冷链能力和区域布局 |

### 9.3 调味品子行业

| 特征 | 信用分析侧重 |
|---|---|
| 毛利率40-50%（品牌溢价较好）| 品牌+渠道并重，两者权重可调整为各30% |
| 消费粘性高（习惯驱动）| 转换成本高是品牌护城河的额外加分项 |
| 品类成熟但细分赛道增长 | 关注复合调味料等新品类占比，评估创新能力 |
| 餐饮渠道占比高 | 需评估餐饮渠道（B端）vs零售渠道（C端）的结构和毛利率差异 |
| 库存压力小（保质期长）| 渠道库存管理压力小于乳业，但需关注"压货"行为 |

### 9.4 软饮料子行业

| 特征 | 信用分析侧重 |
|---|---|
| 毛利率50-60%（品牌+包装溢价）| 品牌力评估中需区分"品牌忠诚度"vs"渠道便利性"的权重 |
| 产品迭代速度快 | L3产品组合权重建议提升至25%（新品成功率是关键） |
| 渠道为王（终端陈列位争夺）| 渠道控制力权重可提升至30%，终端陈列位是核心 |
| 季节性明显 | 需评估现金流是否覆盖淡季运营成本和旺季备货资金需求 |
| 线上渠道占比快速提升 | 评估企业DTC能力和电商运营效率 |

## 十一、范式边界与局限性

### 9.1 明确不适用场景

- **产品技术差异驱动的行业**（半导体、医药、高端装备）——应使用技术壁垒型范式
- **强政策周期行业**（光伏、新能源汽车OEM）——政策影响远超品牌/渠道
- **B2B纯工业品**（化工原料、钢材、大宗商品）——品牌溢价不存在
- **服务型行业**（数据中心、物流）——服务能力重于品牌

### 9.2 已知局限性

1. **品牌价值量化困难**：品牌认知度/NPS等指标缺乏标准化的公开数据源，依赖第三方调研
2. **渠道库存可视化不足**：经销商层面的真实库存数据通常不公开，只能通过应收/预收间接推断
3. **品类生命周期判定有滞后性**：消费趋势变化往往在数据确认前已经发生，需要前瞻性判断
4. **线上渠道颠覆线下**：直播电商/DTC模式的兴起可能使传统经销网络分析框架失效

### 9.3 缓释措施

| 局限性 | 缓释措施 |
|---|---|
| 品牌量化困难 | 交叉使用多个数据源（电商评分+社交媒体+百度指数+线下调研），建立品牌力评分卡 |
| 渠道库存不可见 | 关注预收账款变化 × 应收账款周转 × 保质期折价率三个信号的综合变化 |
| 品类判定滞后 | 建立季度消费趋势更新机制，跟踪Z世代消费行为变化 |
