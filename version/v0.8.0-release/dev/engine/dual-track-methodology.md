# 双轨分析方法论

**版本**: v0.8.0-release | **日期**: 2026-07-10

---

## 一、双轨并行架构

分析引擎的核心是**两条独立并行的分析轨道**，在路径末端通过交叉对撞矩阵产生最终判断。

```
输入: 行业 + 企业 + 分析日期
        │
   ┌────┴────────┐
   │              │
轨道A: 基本面   轨道B: 市场定价
(定性+评分)     (量化信号)
   │              │
 L1 (最重层)     Credit Spreads
 L2               Volatility
 L3               Fund Flows
 L4 (最轻层)      Rating Migration
   │              │
   └──────┬───────┘
          ▼
  交叉对撞矩阵
  Consensus → 互相强化
  Divergence → 最有价值的洞察
```

### 两轨设计哲学

| 特征 | 轨道A：基本面 | 轨道B：市场定价 |
|---|---|---|
| 数据来源 | 行业数据、企业财报、政策文件、招投标 | 债券收益率、股票价格、成交量、评级事件 |
| 更新频率 | 季度/年度（财报周期） | 日度/周度（持续更新） |
| 客观性 | 较高（基于公开事实） | 受市场情绪影响 |
| 滞后性 | 存在财报滞后 | 实时反映 |
| 核心价值 | 识别结构性不可持续 | 捕捉市场定价错误 |

## 二、轨道A：基本面金字塔评分

### 2.1 评分逻辑

每层独立评分（0-10分），权重由行业类型决定。

```
Composite Score = Sigma(Layer Score × Layer Weight)
Layer Score = Sigma(Indicator Score × Indicator Weight)
Indicator Score = f(Raw Value, Threshold, Direction)
```

### 2.2 标准四层金字塔评分

| 行业类型 | L1（最重） | L2 | L3 | L4（最轻） |
|---|---|---|---|---|
| **政策驱动型** | 35% 政策/宏观 | 30% 技术 | 20% 供应链 | 15% 财务 |
| **技术壁垒型** | 20% 政策 | 35% 技术/IP/注册 | 25% 运营 | 20% 财务 |
| **存量博弈型** | 25% 生存位势 | 20% 技术 | 30% 利润要塞 | 25% 财务 |
| **资产租约型** | 15% 政策 | 20% 技术 | 35% 客户/租约 | 30% 财务 |

> **权威源引用**：四层金字塔权重模板的权威定义见[行业分类与分析框架](industry-framework.md) §3.2。本表为复制件，如有差异以industry-framework.md为准。

### 2.3 特殊金字塔结构

| 行业 | 层数 | 层结构及权重 |
|---|---|---|
| **半导体** | 5层 | L1地缘政治30-35%, L2技术25-30%, L3市场15-20%, L4政策/资本10-15%, L5财务5-10% |
| **新能源车-OEM** | 5层 | L1生存位势25%, L2技术20%, L3运营20%, L4财务15%, L5政策/出口10% |
| **新能源车-供应链** | 5层 | L1利润要塞30%, L2技术20%, L3客户质量22%, L4财务17%, L5政策/出口8% |
| **医疗器械** | 5层 | L1政策15%, L2注册证22%, L3渠道25%, L4财务20%, L5外部支持8% |
| **生物医药** | 5层 | L1政策15%, L2管线25%/15%, L3 BD20%/25%, L4财务20%/15%, L5外部支持10%/- |

### 2.4 各层评分示例（光伏）

| 层 | 评分维度 | 高评分（8-10）特征 | 低评分（0-3）特征 |
|---|---|---|---|
| L1 政策 | 政策方向对齐度 | 享受国家级补贴和税收优惠 | 核心产品被政策限制 |
| L2 技术 | 效率差 vs 领先者 | 效率领先行业均值 > 2% | 产能以淘汰技术为主 |
| L3 供应链 | 客户集中度 | 多元化客户分布 | 单一客户 > 50% |
| L4 财务 | 现金/短债比 | > 1.5x | 连续负现金流 + 短债高 |

### 2.5 逐层递进原则

1. **L1必须通过才有意义进入L2。** 如果政策/宏观层面存在致命风险，下层分析无意义。
2. **L4（财务层）是验证层，不是判断层。**
   - 财务状况好但上层差 = **更危险**（可能是财务造假或周期顶部）
   - 财务状况差但上层好 = 标记风险但不颠覆评级
3. **一票否决：** 任一层面触发一票否决条件，综合评级上限锁定为CCC。

## 三、轨道B：市场定价信号

### 3.1 四级市场信号体系

| 级别 | 信用利差 | 波动率 | 资金流向 | 评级事件 |
|---|---|---|---|---|
| **平静（Calm）** | 稳定/收窄 | < 3% 日波动 | 稳定/流入 | 评级稳定 |
| **关注（Watch）** | 走阔20-50bp | 3-5% | 适度流出 | 展望负面 |
| **异常（Abnormal）** | >50bp跳升或持续走阔 | >5% 持续 | 加速流出 | 观察名单 |
| **危机（Crisis）** | 曲线倒挂/冻结 | >8% 或流动性枯竭 | 净清算流出 | 降级 |

### 3.2 轨道B评分映射

| 市场状态 | 轨道B评分 | 含义 |
|---|---|---|
| 全部四级处于平静 | 8-10 | 市场对信用质量无担忧 |
| 1-2个维度进入关注 | 5-7 | 市场出现谨慎信号 |
| 1-2个维度进入异常 | 3-4 | 市场定价反映实质性风险 |
| 任何维度进入危机 | 0-2 | 市场定价反映违约预期 |

## 四、交叉对撞矩阵

### 4.1 四象限对撞

```
                 │ 轨道B：平静          │  轨道B：异常/危机
                 │ (评分 8-10)          │   (评分 0-4)
─────────────────┼──────────────────────┼─────────────────────
轨道A：良好      │  共识                │  分歧A
评分 6-10        │  互相强化确认        │  市场在恐慌什么？
                 │  评级可信度高        │  是否过激？还是框架遗漏？
─────────────────┼──────────────────────┼─────────────────────
轨道A：较差      │  分歧B               │  共识
评分 0-5         │  市场在忽视什么？    │  互相验证确认
                 │  是框架太激进？      │  风险已全面定价
                 │  还是有隐藏风险？    │  评级可信度高
```

### 4.2 分歧处理规则

| 分歧类型 | 优先相信 | 理由 |
|---|---|---|
| **分歧A**（A好+B差） | **轨道A** | 市场可能过度反应；需检查是否有框架未捕捉的外部负面因素 |
| **分歧B**（A差+B好） | **轨道A** | 外部评级滞后是已知问题（已验证滞后17+个月）；市场可能被叙事蒙蔽 |

**核心原则：当两轨分歧时，优先相信轨道A（可审计的公开财务事实）而非轨道B（外部评级和市场价格）。**

### 4.3 偏离度评分（建议新增）

偏离度评分（0-10分）：衡量轨道A与轨道B的一致性程度。

| 偏离度 | 含义 | 操作建议 |
|---|---|---|
| 0-3 | 高度一致 | 自动输出，无需人工介入 |
| 4-6 | 中度偏离 | 自动触发复核提醒 |
| 7-10 | 严重偏离 | 必须人工分析师介入 |

## 五、现金流深度分析

> **权威源引用**：FCF分析的完整计算规格（含分级标准、庞氏检测、营运资金联动）见[财务深度分析报告](financial-deep-dive.md) §D。本节为双轨架构中的定位摘要。

### 5.1 核心公式

自由现金流（FCF）是衡量企业真实造血能力的核心指标，在L4财务层评分中作为验证性输入：

```
FCF = 经营活动现金流净额 - 资本支出
FCF/收入 = FCF / 营业收入
FCF/利息 = FCF / 利息支出
```

数据来源：年报现金流量表（经营活动现金流净额、购建固定资产/无形资产/其他长期资产支付的现金、分配股利/利润/偿付利息支付的现金）。

### 5.2 FCF生成能力分级

| FCF/收入 | FCF/利息 | 判定 | 含义 |
|---|---|---|---|
| >5% | >3x | 🟢 造血能力强 | 经营现金流充裕，可同时覆盖投资和偿债，信用风险低 |
| 0-5% | 1-3x | 🟡 维持性经营 | 仅能维持现有运营和基本偿债，扩张依赖外部融资 |
| <0 | <1x | 🟠 依赖融资输血 | 经营现金流不足，需持续融资支持——短期可承受，长期不可持续 |
| 任何值 | <0（持续>2年） | 🔴 庞氏融资嫌疑 | FCF持续为负且无法覆盖利息支出——紫光集团（2017-2019连续3年）、华晨集团（2018-2020）的共性特征 |

### 5.3 与评级映射的衔接

FCF分析结果不直接改变轨道A综合评分，但作为L4财务层的**验证性修正输入**：

- **FCF/收入 > 5% 且 FCF/利息 > 3x**：L4评分上限可在常规上限基础上+1分（信用质量强化确认）
- **FCF/收入 < 0 且 FCF/利息 < 1x**：L4评分上限锁定为4分（满分10分），即财务层无论其他指标多好，最高评分为4
- **FCF/利息持续<0超过2年**：触发"庞氏融资嫌疑"风险警告，综合评级上限锁定为B（无论其他层评分多高）

### 5.4 与营运资金分析的联动

FCF质量需结合营运资金效率（DSO/DIO/DPO/CCC）交叉验证：

| FCF特征 | 营运资金特征 | 联合判定 |
|---|---|---|
| FCF为正 | CCC正常或优化 | 真实造血，FCF可信 |
| FCF为正 | CCC恶化（DSO上升/DIO堆积） | FCF质量存疑，利润可能被营运资金占用侵蚀 |
| FCF为负 | CCC自然恶化 | 经营基本面恶化，需关注现金流跑道 |
| FCF为负 | CCC改善（靠压应付账款） | 短期粉饰，不可持续 |

## 七、EL预期损失整合

**【评级修订 v0.4.0】** 新增PD评级与LGD框架的整合层，实现EL（Expected Loss，预期损失）的量化估算。

### 7.1 核心公式

```
EL = PD × LGD × EAD

其中：
  PD  = 从PD评级映射的年化违约概率（取区间中值作为点估计）
  LGD = 从LGD五级分类映射的违约损失率
  EAD = 默认按本金 + 应计利息计算（简化处理，不考虑信用转换系数CCF）
```

### 7.2 PD到违约概率的映射

PD评级映射为年化违约概率区间，参考Moody's全球历史违约率统计数据：

| PD评级 | 对应的年化PD区间 |
|---|---|
| AAA | < 0.01% |
| AA+/AA/AA- | 0.01% - 0.05% |
| A+/A/A- | 0.05% - 0.15% |
| BBB+/BBB/BBB- | 0.15% - 0.50% |
| BB+/BB/BB- | 0.50% - 2.0% |
| B+/B/B- | 2.0% - 8.0% |
| CCC | 8.0% - 30% |
| D | > 30% |

**诚实标注：** 上述PD区间基于Moody's《Corporate Default and Recovery Rates, 1920-2023》全球统计数据。中国信用债市场违约样本不足200个（截至2025年末），未经中国市场校准。这些PD区间仅供排序参考，不可用于监管资本计算。

### 7.3 LGD损失率映射

LGD等级定义见[LGD与回收率分析框架](lgd-recovery-framework.md)第二节，损失率区间汇总如下：

| LGD等级 | 损失率区间 | 回收率区间 |
|---|---|---|
| LGD1 | < 20% | > 80% |
| LGD2 | 20% - 40% | 60% - 80% |
| LGD3 | 40% - 60% | 40% - 60% |
| LGD4 | 60% - 80% | 20% - 40% |
| LGD5 | > 80% | < 20% |

### 7.4 输出示例（通用模板）

```
评级: BB+
年化PD: 0.5%-2.0%（中值1.0%）
LGD: LGD2（20-40%，优质抵押覆盖）
EAD: 本金1亿元 + 应计利息200万元 = 1.02亿元
EL: 1.0% × 30% × 1.02亿 = 30.6万元
    （即预期损失率为0.30% × 本金）
```

### 7.5 实际案例：隆基绿能EL计算

以下用隆基绿能（分析日期：2026-07）展示完整的PD × LGD × EAD计算链。

**第一步：从评级映射PD**

隆基绿能轨道A综合评分7.50 → 评级映射为A+（7.5-7.9区间，中低风险）

| PD评级 | 对应年化PD区间 | 使用中值（点估计） |
|--------|---------------|-----------------|
| A+ | 0.05% - 0.15% | **0.10%** |

数据来源：7.2节PD映射表，基于Moody's全球违约率统计。

**第二步：评估LGD**

隆基绿能存续债券为无担保普通信用债，按LGD框架：

| LGD因子 | 评估 | 说明 |
|---------|------|------|
| 债项优先级 | 无担保（Senior Unsecured） | 无特定抵押物覆盖 |
| 行业回收率基准 | 光伏设备制造 | 重资产行业，设备有一定残值但技术迭代快 |
| 抵押物/担保 | 无 | 纯信用债 |
| LGD评级 | **LGD3** | 对应损失率区间40%-60%，使用中值50% |

数据来源：[LGD与回收率分析框架](lgd-recovery-framework.md)第二节LGD五级分类体系。

**第三步：确定EAD**

以隆基绿能2024年发行的某期中票为例：

| EAD构成 | 数值 | 说明 |
|---------|------|------|
| 本金 | 10亿元 | 发行面值 |
| 应计利息（3个月） | 1,350万元 | 票面利率约5.4% |
| **EAD合计** | **10.135亿元** | 简化计算，不考虑CCF |

**第四步：计算EL**

```
EL = PD × LGD × EAD

PD  = 0.10%（A+评级中值）
LGD = 50%（LGD3中值，无担保信用债）
EAD = 10.135亿元

EL = 0.10% × 50% × 10.135亿
   = 0.0005 × 10.135亿
   = **50.7万元**

即预期损失率为0.05% × 本金（0.10% PD × 50% LGD）
```

**第五步：对比解读——评级 vs EL**

| 分析工具 | 结论 | 精确度 |
|---------|------|--------|
| **仅看评级（A+）** | "中低风险，整体信用质量良好" | 定性描述，无法量化损失 |
| **看EL（0.05% × 本金）** | "预期损失约本金的0.05%，即每亿元敞口预期损失约5万元" | **量化精确，跨债项可比** |

**为什么EL比单纯评级更精确？**

1. **区分同一评级内的差异**：两家A+企业，一家有优质抵押（LGD2）一家无担保（LGD3），EL相差约1倍。评级本身无法反映这一差异。
2. **将PD和LGD的变动转化为一致标尺**：评级是从"亏损概率"角度排序，EL是从"预期亏损金额"角度排序。对投资组合管理者而言，后者更具可操作性。
3. **EL可直接用于组合层面**：将每个债项的EL加总，得到组合层面的预期损失，用于拨备计提和风险预算分配。

**重要限制说明**（同7.5节）：

| 限制 | 对隆基案例的具体影响 |
|------|---------------------|
| PD区间未经中国市场校准 | 中国光伏企业实际违约率可能与Moody's全球基准有差异 |
| LGD为简化估算 | 隆基绿能若发生违约，实际回收率可能偏离40%-60%区间 |
| EL是排序工具 | 50.7万元为排序参考值，不是精确预测值

### 7.6 重要限制说明

1. **PD映射未经中国市场校准：** 上述PD区间引用Moody's全球数据。中国信用债违约样本不足200个，实际违约率可能因市场结构（国企主导）、监管环境（地方政府协调）和司法实践（破产重整清偿率差异）与全球基准显著不同。
2. **LGD为简化估算：** LGD区间来自框架设定值，并非基于中国市场的历史回收率统计。详见[lgd-recovery-framework.md](lgd-recovery-framework.md)的诚实性声明。
3. **EL是排序工具而非计量工具：** 本框架的EL估算用于比较不同债项的信用风险排序，不能替代Basel II/III框架下的监管资本计量。
4. **EAD简化处理：** 不考虑表外风险暴露的信用转换系数，因为引擎当前分析对象为信用债（本金+利息敞口明确）。

### 7.7 中国市场参考

#### 7.7.1 中国信用债市场违约概况（2014-2026）

中国信用债市场违约历史短、样本少，但趋势清晰：

| 年份 | 违约主体数量 | 涉及债券规模（亿元） | 标志性事件 |
|---|---|---|---|
| 2014 | 1 | 10 | 超日债违约——中国公募债首例违约 |

**汇总（2014-2025）：** 累计违约主体约300个（含重复违约），涉及债券规模约1.2万亿元。但其中**真正意义上的公开债券违约样本不足200个**（大部分为已暴露主体的后续违约或非公开违约）。

> **统计意义声明：** 中国信用债市场的有效违约样本远低于Moody's全球违约数据库（1920-2023年超过5000个样本）。以下所有基于中国数据的PD估算均为**方向性参考**，统计置信区间极宽，不可用于监管资本计算或精确风险定价。

#### 7.7.2 中国市场PD参考估算

|---|---|---|---|

| AAA（民企） | 样本极少 | — | 民企极少获得AAA，无统计意义 |

| AA+/AA/AA-（民企） | 0.5% - 2.0% | 样本约1500主体年 | 民企AA违约率显著高于国企同评级 |

#### 7.7.3 评级迁移矩阵参考

中国评级机构（中诚信国际、联合资信、中债资信等）定期发布信用评级迁移矩阵。以下为基于公开数据的方向性参考：

| 初始评级 | 1年后维持 | 上调 | 下调 | 下调至违约 | 说明 |
|---|---|---|---|---|---|
| AAA | > 98% | < 0.5% | < 1.5% | < 0.1% | 迁移率极小，评级僵化 |
| AA+ | > 95% | < 1% | 3-5% | < 0.5% | 少数AA+下调至AA |
| AA | > 90% | 1-2% | 5-8% | 0.5-1.5% | AA级分化明显 |
| AA- | > 85% | 1-2% | 8-12% | 1-3% | 弱资质AA-违约风险上升 |
| A+ | > 80% | 1-3% | 10-15% | 2-5% | A+是近年违约密集区 |
| A | > 75% | 1-3% | 12-18% | 3-7% | 统计稳定性低 |

**迁移矩阵反映的中国市场特征：**

1. **评级上调极少**——上调比例远低于国际水平（1-3% vs 国际5-10%），反映评级分布整体向上偏移后缺乏上调空间。
2. **AAA级几乎不迁移**（维持率>98%），评级调整几乎不反映违约风险变化。
3. **AA是信用分水岭**——AA级以上违约率极低，AA-及以下违约率激增。这与国际市场上BBB为投资级分水岭的格局完全不同。
4. **2018年后矩阵右偏恶化**——去杠杆后各评级下调迁移率显著上升。

#### 7.7.4 SOE vs 非SOE违约率差异

中国信用债市场最显著的区分特征是国企与民企的违约率鸿沟：

| 比较维度 | 中央国企 | 地方国企 | 民营企业 |
|---|---|---|---|
| 2014-2025累计违约主体 | 10-15个 | 30-50个 | 200+个 |
| 占全部违约比例 | < 5% | 15-20% | 75-80% |
| AAA主体历史违约率 | ~0.02%（仅永煤1例） | ~0.1% | 样本不足 |
| 同评级违约率比较（AA） | 基准（最低） | 央企的2-3x | 央企的10-20x |
| 违约后回收率 | 较高（政府协调） | 中（地方政府主导） | 低（市场化处置为主） |
| 首次刚兑打破时间 | 2020年（永煤） | 2015年（天威） | 2014年（超日） |

**核心结论：**

1. **SOE违约风险不是零但显著低于同评级非SOE。** 永煤违约（2020年11月）证明了"国企不违约"信仰的终结，但国企违约率仍远低于相同评级的民企。

2. **同评级内部SOE/非SOE违约率差异超过1-2个评级档位。** 建议在PD映射中对SOE给予**隐性+1子级**的校准偏移——即一个AA级地方国企的PD参考区间相当于同评级民企的AA-档位。

3. **"逃废债"（恶意违约）是非SOE特有的风险。** 民营企业通过资产转移、破产重整逃避债务的案例显著多于国有企业（详见[治理与欺诈风险](governance-fraud-risk.md)）。

4. **地方政府在国企违约中的角色是关键变量。** 有协调价值的国企违约后政府介入可能性高、回收率也更高；无协调价值的国企更接近市场化处置。

#### 7.7.5 中国信用债市场特有因素

以下因素造成中国市场的PD特征与Moody's全球基准存在系统性差异：

| 因素 | 对PD影响方向 | 影响机制 | 影响程度 |
|---|---|---|---|
| **政府支持隐含上调** | 降低违约率（对国企） | 地方政府/中央政府在部分国企面临违约风险时介入协调（流动性支持/续贷协调/并购重组） | 中-高（对地方国企PD降幅可达50%） |
| **刚兑打破历史短** | 结构不稳定 | 中国公募债2014年才首例违约，市场运行不到12年，违约率仍在上升通道，尚未进入稳态 | 中（实际违约率可能仍低于长期均衡水平） |
| **"逃废债"风险** | 推高违约率（民营） | 部分企业有能力偿债但选择违约（资产转移/破产逃废债），还款意愿风险远高于国际市场 | 中-高（对民企额外提高PD 50-100%） |
| **评级虚高/区分度不足** | 同评级实际违约率离散 | 中国评级AAA+AA占比>60%（国际市场<20%），同一评级内部信用质量差异远大于国际市场 | 高（同评级违约率离差可能是国际3-5倍） |
| **增信措施效力不确定** | 视增信类型而定 | 担保/抵押法律效力存在执行风险（政府协调可能绕过担保条款；抵押物处置流程复杂） | 中（增信效力约为国际标准60-80%） |
| **区域风险分化** | 推高部分区域违约率 | 经济弱省信用风险无法通过个体分析单独评估，需叠加区域分析（详见[城投双轨框架](lgfv-framework.md)） | 中（区域分化导致PD差异3-5倍） |
| **系统性冲击事件** | 短期全市场PD同步上升 | 如永煤违约后发行冻结、2022年理财赎回潮——全市场利差飙升，所有评级PD同步上升 | 短期极高但非稳态——不作为正常时期PD调整基准 |

#### 7.7.6 中国市场校准总结

中国市场PD映射的**方向性校准建议**（非精确调整——精确需要超过2000个有效违约样本）：

| 调整维度 | 建议 | 操作提示 |
|---|---|---|
| **央企AAA** | 维持Moody's全球基准（< 0.01%） | 极少数样本支持 |
| **地方国企AA+至AA-** | 在Moody's基准上降低30-50% | 政府协调效应显著 |
| **国企A+及以下** | 接近Moody's基准 | 政府支持效力随评级下降递减 |
| **民企AAA** | 样本不足，参考AA+级Moody's基准 | 慎用AAA——中国民企AAA实际信用质量约当国际AA |
| **民企AA+至AA-** | 在Moody's基准上提高50-100% | 逃废债+高波动性 |
| **民企A+及以下** | 在Moody's基准上提高100-200% | 违约风险极高，谨慎投资 |
| **弱资质城投（绑定政府）** | 不直接适用PD映射——参考[城投双轨框架](lgfv-framework.md) | 城投信用由政府信用+平台质量双轨决定 |

> **核心诚实标注：** 上述校准建议基于不到200个有效违约样本的次级分析，统计显著性和普适性有限。建议用户在实际应用中将中国校准的PD区间视为方向参考而非精确概率。当需要精确PD用于定价时，优先采信Moody's全球基准（样本充分、统计稳健），中国市场校准仅作为"上/下方向调整"参考。

## 九、风险缓释建议框架

> **来源**：[风险管理标准审计报告](audits/risk-management-standards-audit.md) （G1 — 风险缓释建议的缺失）
> **说明**：引擎现有输出止步于"风险识别"和"风险评估"，缺少"应该怎么办"的具体指引。本框架提供的不是投资建议（"该买还是该卖"），而是在用户已决定降低风险的前提下，帮助选择最合适的缓释路径。

### 9.1 设计原则

1. **不是投资建议**：本框架不回答"该买还是该卖"，只回答"如果要降低风险，有哪些可选路径"
2. **场景匹配**：每种缓释路径标注适用场景和成本，不搞"一刀切"
3. **诚实标注**：某些风险（如系统性风险）无法通过单一组合调整来缓释——必须如实指出
4. **与现有分析衔接**：缓释建议基于引擎已完成的信用分析结论（评级、场景压力测试、集中度分析等），不重复分析

### 9.2 缓释路径菜单

| # | 缓释路径 | 适用场景 | 有效性 | 成本/代价 | 实施难度 |
|---|---|---|---|---|---|
| 1 | **缩短久期** | 信用质量不确定，利率上行风险 | ⭐⭐⭐⭐ 高 | 可能损失Carry收益（短券收益率低） | 容易——卖出长券换短券 |
| 2 | **增加担保/增信要求** | 单主体信用质量偏弱但仍有缓释空间 | ⭐⭐⭐⭐ 高 | 影响融资成本（发行人可能要求补偿） | 中——需重新谈判条款 |
| 3 | **分散到不同区域** | 区域集中度过高（单一省份敞口>20%） | ⭐⭐⭐ 中高 | 交易成本+新区域研究成本 | 中——需寻找新投资标的 |
| 4 | **分散到不同行业** | 行业集中度过高（单一行业敞口>30%） | ⭐⭐⭐ 中高 | 跨行业研究门槛 | 中——需理解新行业逻辑 |
| 5 | **购买CDS/CRMW** | 个券信用风险暴露敞口较大，但不想直接卖出 | ⭐⭐⭐ 中 | 保费成本（可参考信用利差定价） | 中——市场深度有限，需交易对手 |
| 6 | **降低单一敞口** | 单主体集中度过高（>10%组合） | ⭐⭐⭐⭐⭐ 最高 | 交易佣金+流动性冲击（如持有比例高） | 容易——在二级市场分批减持 |
| 7 | **增加质押式回购能力** | 流动性储备不足，需要提升流动性 | ⭐⭐ 中 | 抵押品占用，杠杆率上升 | 中——需开户/授信 |
| 8 | **增加利率对冲** | 久期敞口大，利率上行风险 | ⭐⭐ 中 | 对冲成本（期货/IRS） | 中——需衍生品交易能力 |
| 9 | **增加条款保护** | 新投资或存续券可变更条款 | ⭐⭐⭐ 中高 | 可能降低发行利差（保护越强票息越低） | 中等——仅适用一级市场或持有人会议 |
| 10 | **清仓/退出** | 触发一票否决条件或评分<3 | ⭐⭐⭐⭐⭐ 最高 | 交易成本+机会成本+可能影响市场信誉 | 容易（市场流动性好时） |

### 9.3 缓释路径推荐规则

基于引擎输出结果自动生成缓释建议：

| 引擎发现 | 推荐缓释路径 | 优先级排序 |
|---|---|---|
| **A. 单主体信用质量恶化**（评级降至B以下） | ① 降低单一敞口 ② 缩短久期 ③ 购买CRMW ④ 增加担保要求 | ①>③>②>④ |
| **B. 行业集中度过高**（单一行业>30%） | ① 分散到不同行业 ② 对行业高风险主体降低敞口 ③ 购买行业指数CDS（如有） | ①>②>③ |
| **C. 区域集中度过高**（高风险区域>10%） | ① 分散到不同区域 ② 降低高风险区域敞口 ③ 关注区域化债进展 | ①>②>③ |
| **D. 伪高评级占比过高**（>15%） | ① 逐只排查伪高评级券 ② 降低评分<5的券 ③ 要求增加担保 | ①>②>③ |
| **E. 流动性风险突出**（现金跑道<6个月） | ① 缩短久期 ② 增加质押能力 ③ 保持现金储备 | ②>①>③ |
| **F. Bear场景不通过**（利息覆盖<1.5x） | ① 大幅降低敞口 ② 购买CDS ③ 要求追加抵押品 | ①>③>② |
| **G. Severe场景不通过**（利息覆盖<1.0x） | ① 清仓退出 ② 或要求全额担保 ③ 缩短至超短久期 | ①（如可退出）>③（如不可退出） |
| **H. 治理风险高**（🔴治理信号触发） | ① 降低敞口至0.5%以下 ② 要求交叉违约条款 ③ 加速到期安排 | ①>②>③ |
| **I. 逃废债嫌疑**（还款意愿评分<-50） | ① 仅推荐清仓退出 ② 无可接受的替代缓释路径 | ①（必选） |

### 9.5 重要限制与诚实标注

**必须标注的缓释局限性**：

| 局限性 | 说明 |
|---|---|
| **系统性风险无法通过单一工具缓释** | 宏观衰退、流动性危机、监管政策急转——这些风险无法通过调整个券暴露来化解，只能降低整体风险预算或使用宏观对冲工具 |
| **CDS/CRMW市场深度有限** | 中国信用衍生品市场尚处发展早期，大多数个券没有对应的CDS或CRMW——对"购买保护"路径需要实际确认市场可得性 |
| **组合调整本身有成本** | 调仓需考虑交易成本（佣金+冲击成本）+ 税务成本 + 机会成本（卖出后错过可能的价格恢复） |
| **流动性约束** | 在市场压力时期，大量减仓可能导致更大的价格损失——"减仓"路径在危机中的实际效果可能远低于模型估算 |
| **缺失数据影响缓释效果判断** | 如果引擎分析本身的数据完备性偏低，缓释建议的置信度也会相应降低 |
| **委托代理问题** | 不同身份（基金经理 vs 信贷审批 vs 交易员）对同一缓释路径的成本承受力不同——M1基金经理对降低收益率的容忍度低于M0信贷审批 |

### 9.6 与交叉对撞矩阵的衔接

缓释建议的输出强度（建议力度）应与交叉对撞分歧度联动：

| 交叉对撞状态 | 缓释建议力度 | 说明 |
|---|---|---|
| **共识（A好+B好 或 A差+B差）** | 常规建议 | 信心充分，缓释建议按标准模板输出 |
| **分歧（A好+B差 或 A差+B好）** | 增强建议+方向性提示 | 存在不确定性→建议实施"不可逆的缓释路径"（如降低敞口）而非"可逆的缓释路径"（如缩短久期） |
| **数据不足** | 保守建议 | 缺少关键数据→缓释路径优先选择"信息不敏感"方案（如增加担保） |

## 十、完整推理实例：一道新能 vs 隆基绿能（5步推理）

### 背景
- 分析日期：2026-07
- 行业：光伏（政策驱动型）
- 对象：隆基绿能（上市，债券存续） vs 一道新能（未上市，IPO终止）

### 第1步：行业分类与权重确定

- 十维评分判定结果：政策驱动型（D3=5, D4=4）
- 权重模板：L1政策35%, L2技术30%, L3供应链20%, L4财务15%

### 第2步：逐层评分

#### 隆基绿能

| 层 | 关键信号 | 评分 | 权重 |
|---|---|---|---|
| L1 政策 | BC技术受政策支持，央企集采入选50.3GW | 8 | 35% |
| L2 技术 | BC量产效率24.8%，专利510件 | 8 | 30% |
| L3 供应链 | 多元化客户分布，垂直整合程度高 | 7 | 20% |
| L4 财务 | 现金储备526亿，经营性现金流为正 | 6 | 15% |

**综合 = 8×35% + 8×30% + 7×20% + 6×15% = 2.80 + 2.40 + 1.40 + 0.90 = 7.50**

#### 一道新能

| 层 | 关键信号 | 评分 | 权重 |
|---|---|---|---|
| L1 政策 | 无特殊政策优势 | 5 | 35% |
| L2 技术 | IPO终止，技术路线不明确 | 2 | 30% |
| L3 供应链 | 客户集中度高 | 2 | 20% |
| L4 财务 | 负债率86.89%，净资产可能为负 | 0 | 15% |

**综合 = 5×35% + 2×30% + 2×20% + 0×15% = 1.75 + 0.60 + 0.40 + 0 = 2.75**

**一票否决检查：** 一道新能负债率超过危险阈值，但未触发明确技术路线淘汰条件。评级上限未被锁定。但综合评分2.75→B。

### 第3步：轨道B分析

#### 隆基绿能

| 信号维度 | 状态 | 说明 |
|---|---|---|
| 信用利差 | 异常 | 市场对光伏行业整体悲观 |
| 波动率 | 关注 | 股价波动加剧 |
| 资金流向 | 关注 | 行业整体资金流出 |
| 评级事件 | 稳定 | 评级维持 |

**轨道B评分：** 4（异常/关注混合）

#### 一道新能

| 信号维度 | 状态 | 说明 |
|---|---|---|
| 信用利差 | 无数据 | 非上市，无债券流通 |
| 波动率 | 无数据 | — |
| 资金流向 | 无数据 | — |
| 评级事件 | 无数据 | — |

**轨道B评分：** 无数据（数据缺口本身就是风险信号）

### 第4步：交叉对撞

| 对比项 | 隆基绿能 | 一道新能 |
|---|---|---|
| 轨道A评分 | 7.50（A+） | 2.75（B） |
| 轨道B评分 | 4（异常） | 无数据 |
| 交叉状态 | **分歧A（A好+B差）** | 不可对撞 |
| 框架判断 | 市场在恐慌什么？全行业亏损+评级滞后的利差收窄 | 非上市无市场数据本身就是重大信号 |
| 最终评级 | A+（中置信度） | B（低置信度，数据不足） |
| 分差 | **5.5分——框架清晰区分好坏企业** | |

### 第5步：输出与完备性报告

**隆基绿能：**
- 综合评级：A+（中置信度）
- 核心发现：BC技术领先 + 央企集采第一
- 关键风险：全行业亏损 + 评级滞后的利差收窄
- 数据完备性：L2技术缺良率数据（评分±1.5不确定性）

**一道新能：**
- 综合评级：B（低置信度）
- 核心发现：IPO终止 + 极高负债率 + 净资产可能为负
- 关键风险：非上市企业无市场数据覆盖
- 数据完备性：轨道B数据完全缺失——最严重的单一缺口
