相对价值评估 · 关键发现

正偏离机会：隆22转债是唯一一只纯债价值高于面值（98.52）且转股溢价率低于20%的标的，相对价值优势显著。AAA评级+充足流动性使其成为组合核心底仓的首选。

关注信号：通22转债与晶澳转债的转股溢价率均在24-29%区间，估值处于行业中位水平。但通22的下修触发条件为15/30, 80%（较隆22的85%更宽松），若董事会启动下修程序，转股溢价率可快速压缩至10%以下。

风险区域：天23转债（35.2%）和晶能转债（42.1%）均处于深价外状态，转股溢价率显著高于行业均值（25-30%）。两者信用利差也处于高位，反映市场对正股价格持续下跌和信用风险的双重担忧。

隐含波动率启示：晶能转债隐含波动率41.3%显著高于隆22的32.5%，表明市场对晶能未来正股波动预期更高。但高波动率需要配合正股方向性判断，若正股继续下跌，高波动率无法提供保护。

七、方法论说明

四维评分方法

相对价值（权重30%）：纯债价值(40%)+转股溢价率(30%)+信用利差(30%)

条款保护（权重25%）：下修条款(40%)+回售保护(25%)+赎回条款(25%)+其他(10%)

流动性（权重25%）：日均成交额(40%)+换手率(20%)+买卖价差(25%)+大宗可行(15%)

信用质量（权重20%）：主体评级(40%)+债项评级(30%)+正股质量(30%)

评分标准：每个维度满分10分，加权求和后得到综合得分（满分10分）。排名前2位的债券（综合得分>6.5）建议标配或超配，综合得分4.5-6.5的债券建议低配或交易为主，综合得分<4.5的债券建议回避。

关键指标计算公式

纯债价值：未来各期票息和面值按同期限国债收益率+信用利差贴现的现值之和

转股溢价率：(转债价格 - 转股价值) / 转股价值 x 100%

转股价值：100 / 转股价 x 正股股价

纯债YTM：使纯债未来现金流现值等于转债纯债价值（即剔除转股权价值后）的到期收益率

信用利差：纯债YTM - 同期限国债到期收益率

隐含波动率：将转债价格分解为纯债价值+转股权价值，使用BS模型反推的波动率参数

买卖价差：(最优卖价 - 最优买价) / 中间价 x 10000（单位：bp）

冲击成本：指定交易量下，实际成交均价与下单前中间价的偏离（单位：bp）
