## Case 1: Yongmei Coal (永城煤电控股集团)

- **Default date:** 2020-11-10 (20永煤SCP003, 10亿本金)
- **Pre-default rating:** AAA/稳定 (中诚信)
- **Post-default rating:** BB (一刀砍至违约级)
- **Risk genotype:** Governance structure defect — parent company hollowed out by majority shareholder, financial fraud (虚增货币资金861亿)
- **Market belief:** "AAA国企=安全"

### T1 Analysis (2019-06-30, T-17 months)

**Data available:** 2018 annual report (published April 2019), 2019 Q1 report

| Track A Signal | Status |
|---|---|

| Attributable net profit: -11.44B (2018) | 🔴 |
| Short-term debt ratio: 59.14% | 🟡 |
| Related-party receivables: 22.5B | 🔴 |
| Shareholding structure defect | 🔴 |

| Track B Signal | Status |
|---|---|
| External rating: AAA/Stable | 🟢 (misleading) |
| SCP issuance rate: 3.24-3.40% | 🟢 (normal) |
| Coal industry: large miners profit +32.8% | 🟢 |

**Framework verdict:** 4 red + 1 amber on Track A. Track B completely green. Recommended: "Needs ongoing monitoring."

### T2 Analysis (2020-06-30, T-4.5 months)

Track A: 5 red + 0 amber (escalated). Track B: beginning to converge (amber signals emerging).

### What the framework could NOT have known at T1:
- Asset transfer plan (internal government decision, July 2020)
- True scale of cash fabrication (861B, discovered by post-default investigation)
- Exact default date (Nov 2020)

## Case 2: Tsinghua Unigroup (紫光集团)

- **Default date:** 2020-11-16 (17紫光PPN005, 13亿本金)
- **Pre-default rating:** AAA/稳定 (中诚信)
- **Restructuring:** 破产重整, 智路建广联合体接盘
- **Risk genotype:** M&A bubble — debt-fueled acquisition spree, goodwill inflated, interest coverage <2x
- **Market belief:** "国家半导体舰队, 大而不倒"

### Analysis (2019-06-30, T-17 months)

**Data available:** 2018 annual report (published April 2019)

| Layer | Key Signal | Status |
|---|---|---|
| L5 Financial | Debt ratio: 73.42% (97.59% excluding goodwill) | 🔴 |
| L5 Financial | EBITDA interest coverage: 1.61x (2017: 2.29x) | 🔴 |
| L5 Financial | Core net profit: -67.86B (vs headline -6.31B) | 🔴 |
| L5 Financial | Financial expenses = 6.5x total profit | 🔴 |
| L5 Financial | Goodwill: 543B (19.59% of assets) | 🔴 |
| L4 Policy | School-enterprise reform forcing Tsinghua divestiture | 🔴 |
| L4 Policy | Two failed equity transfers (no market takers) | 🔴 |
| L3 Market | H3C #1 in China enterprise networking | 🟢 |
| L2 Technology | YMTC not yet in mass production, 2-3 generations behind | 🟡 |
| L1 Geopolitics | ZTE/Huawei already sanctioned, YMTC equipment US-dependent | 🟡 |

**Framework verdict:** L5 extreme risk + L4 high risk → "High risk, recommend exposure reduction."

### Key insight:
Unlike Yongmei (fraud concealed the truth), Tsinghua's risk was transparent — all numbers were in the 2018 annual report. The market was simply blinded by the "national champion" narrative.

### Proposed framework refinement:
L5 weight for M&A-driven enterprises should increase from 5-10% to 15-20%. The "goodwill-adjusted debt ratio" metric proved critical.
