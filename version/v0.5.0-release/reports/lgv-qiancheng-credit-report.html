04 风险信号清单 · 按紧急度排列

从信贷经理视角出发，按风险紧急度从高到低排列，重点关注6-12个月内可能进一步恶化的风险

紧急 #1
失信被执行人持续新增——司法信用彻底破产
11次失信记录表明平台已不具备任何再融资条件。若继续新增（概率极高），将触发更多债权人起诉和资产查封，形成"起诉-失信-更多起诉"的恶性循环。
紧急 #2
再融资渠道全部关闭——流动性枯竭
银行授信冻结+债券发行暂停+融资租赁渠道中断=完全丧失外部融资能力。仅靠政府回款和资产处置维持，现金流随时可能断裂。
紧急 #3
公开债券展期——信用事件已实质发生
多只债券已展期，构成公开市场信用事件。评级已降至C档，距D档（违约）仅一步之遥。债券持有人的回收时间和回收率存在极大不确定性。
紧急 #4
隐性负债规模远超账面——实际负债率估计超80%
表面资产负债率约62%，但含大量担保、明股实债和融资租赁隐性负债。若全部纳入，真实负债率可能超80%，偿债能力远弱于账面表现。
紧急 #5
化债政策覆盖不足——区县级平台仍在"风险处置"阶段
一揽子化债政策主要惠及省级和市级平台，区县级平台受益有限。黔城产业未被纳入化债白名单，短期内看不到政策救助希望。

05 信贷建议 · 对公信贷经理视角

回避——已进入信用事件状态，不建议新增敞口

黔城产业是典型的弱资质区县级城投信用崩塌案例。CC评级对应违约级信用质量，失信被执行人11次+债券展期+再融资渠道全面关闭 = 信用修复可能性极低。信贷建议的核心原则是：不新增敞口，存量积极退出。

对存量敞口的处置建议

现状评估全面排查存量敞口 — 逐笔确认担保措施、抵押物现状和法律状态。梳理到期时间，建立风险台账
担保执行立即启动 — 如存在有效担保（土地抵押、房产抵押等），应尽快启动司法执行程序，防止其他债权人抢先查封
退出策略积极推进 — 建议通过二级市场转让、债务重组、债权折价出售等方式退出。参考贵州其他区县级城投处置案例，预期回收率约30-50%
债务重组参与债权人委员会 — 如已成立债委会，积极参与统一协调。避免单方面诉讼导致碎片化处置

密切监控的指标

失信记录持续跟踪 — 是否新增失信执行人案件
债券展期进展展期协议执行情况，是否有还款或部分还款
化债政策动态是否被纳入化债白名单或获得特殊再融资债支持
政府态度地方财政是否有任何救助信号或资产注入
其他债权人行动其他银行/信托的处置策略——是诉讼、重组还是等待
严重不建议的操作：
1. 新增任何形式的信贷敞口——纯信用、抵押、担保、贸易融资等均不建议。信用事件已发生，不存在"安全结构"
2. 参与债券二级市场买入——虽有折价（净价50-60元），但回收时间和金额存在高度不确定性
3. 接受展期协议——除非债委会统一安排且担保权益得到明确保护，否则展期仅推迟损失确认
4. 依赖政府"隐性担保"——区县级城投的政府信用背书已实质失效，化债政策覆盖不足

信用修复情景分析 · 概率极低

评估黔城产业未来1-3年信用修复的可能性，不做传统现金流压力测试（已无正常现金流可测）

情景发生概率描述对债权人的含义

地方政府资源救助 ~15% 上级政府协调资源注入（资产划拨+财政资金），帮助平台进行债务重组和部分兑付 回收率40-60%
资产处置回血 ~30% 通过处置非核心资产（土地、股权、物业等）获得部分资金用于偿债，但处置时间不确定 回收率30-50%，时间1-3年
维持现状/持续恶化 ~45% 平台继续处于风险处置阶段，无外部救助，债权回收高度依赖司法程序，回收率极低 回收率20-30%，时间3-5年+
评估：信用修复概率极低。化债政策延伸覆盖的概率仅10%，且即使发生也非短期利好。"维持现状/持续恶化"概率45%是最可能的情景。对信贷经理而言，最务实的预期是：回收率30-50%，回收时间1-3年。建议成立专项工作组，积极推动存量敞口的司法处置和债务重组。
核心逻辑总结：黔城产业已不是"放不放进款"的问题——信用事件已发生。
CC评级 + 失信11次 + 债券展期 = 信用崩塌已确认。
区域经济弱 + 区县级层 + 化债覆盖不足 = 短期内无修复希望。
结论：不新增敞口，存量积极退出。对现有敞口以回收最大化为目标，启动担保执行或参与债委会统一处置安排。
