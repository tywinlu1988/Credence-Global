永煤控股违约回溯：17个月预警区间

两个时点、同一框架 —— 验证双轨方法论能否在危机前足够长的时间窗口内，持续跟踪并给出递进式预警

事件：2020.11.10 永煤SCP003违约 时点一：2019.06.30（违约前17个月） 时点二：2020.06.30（违约前4.5个月）
验证结论：在违约前17个月（2019年6月），双轨框架即可明确识别永煤控股的信用风险。
不需要疫情、不需要评级下调、不需要市场传闻——只需要认真阅读2018年年报中母公司报表和合并报表之间的巨大裂缝。

违约前17个月的关键信号：母公司结构在2018年就已经坏了

永煤控股的结构性缺陷不是逐渐恶化的——它在2018年底就已成型。此后的17个月只是在既有轨道上加速下滑。

核心发现：永煤的信用风险不是来自煤炭行业下行、不是来自疫情冲击、不是来自资产划转——这些只是导火索。根本原因是母公司层面极度脆弱的财务结构，而这个结构在2018年年报（2019年4月公开）中已经暴露无遗。

关键洞察：这不是一个"预测"问题——这是在2019年6月已经可以看到"这个结构撑不了太久"的问题。母公司94.87%的负债率+59%短债占比+224.91亿关联方占款+盈利子公司只有61.9%持股——这个结构本身就是在"裸奔"。

双轨框架的价值不在"预测17个月后会违约"，而在"在17个月前就说清楚——这家企业不是AAA质量的"。

最独特的发现：持股结构的根本缺陷

这是传统信用分析几乎不关注的维度，但它是解释永煤困境最底层的逻辑：

盈利的煤炭子公司

永煤控股仅持股 61.9%

少数股东分走利润的38.1%

合并层面盈利，但归属母公司的比例被大幅稀释

亏损的化工子公司

永煤控股 100% 持股

亏损全部由母公司吸收

3家化工子公司2019年合计亏损约9.2亿，2020年已全部资不抵债

"盈利归别人，亏损归自己"——这个结构在2018年年报中完全公开。母公司报表中，营业总收入仅占合并报表的约1%（约5亿 vs 489.94亿），但全部应付债券（100%）由母公司承担。母公司本质上是一个"融资空壳"——收入在子公司，债务在母公司，亏损归母公司，利润归少数股东。

完整预警轨迹：从17个月前到违约日

2019年4月
2018年年报公开。母公司负债率94.87%、归母净利-11.44亿、短债占比59%、关联方占款225亿。
→ 红灯×4。轨道A首次预警。
2019年6月
双轨框架第一次扫描。A说高风险，B说AAA安全。两轨打架。
→ 判定：需持续关注。开始限制新增敞口。
2019年下半年
化工品价格继续暴跌。甲醇、醋酸、乙二醇价格跌至低位。化工板块亏损加速。
→ 化工成为失血点。
2020年1-3月
疫情爆发。煤价跌入红色区间。煤炭行业利润Q1同比-29.9%。
→ 外部环境急剧恶化。
2020年4月
2019年年报公开。归母净利-13.17亿、母公司负债率95.06%、短债占比加速恶化至72%。
→ 红灯×5。轨道A第二次预警，升级为"高风险"。
2020年7月
中债隐含评级下调（AA→AA-）。非市场化发行特征暴露（票面锁定6%）。资产划转内部启动。
→ 轨道B也开始出现负面信号。两轨首次同向。
2020年11月2日
资产划转公告发布。违约前8天。
→ 市场恐慌。但此时为时已晚。
2020年11月10日
永煤SCP003实质性违约。AAA→BB仅用1天。河南、山西、河北国企债全线抛售。
如果从2019年6月开始执行降敞口操作 → 有17个月的安全窗口来完成全部风险规避。
