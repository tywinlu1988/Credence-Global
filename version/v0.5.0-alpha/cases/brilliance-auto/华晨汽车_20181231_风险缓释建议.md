# 风险缓释建议：华晨汽车集团信用敞口

## 建议日期：2018年12月31日

---

## 一、核心风险概述

华晨汽车集团在2018年末存在**三大结构化风险**，且短期内无法通过常规风险管理手段消除：

| 核心风险 | 严重程度 | 可缓释性 |
|---------|---------|---------|
| 母公司资不抵债 + 短期债务集中到期 | **极高** | 低（属于融资结构问题） |
| 华晨宝马股权从50%降至25%，利润腰斩 | **极高** | 不可缓释（交易已签署） |
| 自主品牌持续亏损，无改善迹象 | **高** | 低（属于经营问题） |

---

## 二、风险缓释策略矩阵

### 2.1 策略优先级

按有效性排序：

| 优先级 | 策略 | 有效程度 | 可行性 | 执行时间 |
|-------|------|---------|-------|---------|
| **P0** | 立即减仓 | 极高 | 高 | 1-4周 |
| **P1** | 设置硬性集中度上限 | 高 | 高 | 立即执行 |
| **P2** | 辽宁区域敞口降风险 | 高 | 中高 | 1-3个月 |
| **P3** | 回购渠道审查 | 中高 | 高 | 1-2周 |
| **P4** | 做空华晨中国股票对冲 | 中低 | 中 | 根据市场时机 |
| **P5** | 信用衍生品对冲 | 低 | 极低 | 产品不可用 |

### 2.2 具体措施

#### P0：减仓方案

| 减仓方式 | 操作 | 影响 | 优先级 |
|---------|------|------|-------|
| **主动卖出** | 在二级市场向做市商/基金对手方卖出 | 实现止盈/止损，锁定收益或控制损失 | 最高 |
| **不续作回购** | 到期回购不再续做，自然降杠杆 | 降低名义敞口 | 高 |
| **拒绝新增** | 一级市场不参与华晨新债认购 | 防止敞口扩大 | 必须执行 |
| **利用回售条款** | 关注2021年回售窗口，提前准备行权 | 以面值退出（100元） | 中期 |

**减仓时间窗口判断：**
- 2019年Q1：**最佳窗口**。年初配置需求旺盛，保险/基金有"开门红"配置压力，流动性最好
- 2019年Q2-Q3：**一般窗口**。年报披露后可能出现负面信息，但市场情绪仍可接受
- 2019年Q4及以后：**狭窄窗口**。临近华晨2020年到期，市场关注度上升

#### P1：集中度控制硬约束

建议设置以下硬性限额：

| 限额类型 | 建议值 | 超限处理 |
|---------|-------|---------|
| 华晨单一敞口占组合净值 | **≤0.5%** | 超限须风控委员会批准，限期30天减仓 |
| 汽车行业敞口占信用组合 | **≤5%** | 当前汽车行业下行周期，整体不宜超配 |
| 辽宁省敞口占信用组合 | **≤3%** | 东北地区信用记录不佳，区域限额从严 |
| 单一集团关联敞口（华晨系所有实体） | **≤1%** | 含华晨中国股票、华晨宝马供应链敞口 |

#### P2：区域敞口降风险方案

| 步骤 | 措施 | 时间 |
|------|------|------|
| 1 | 盘点组合中所有辽宁省敞口（含城投、国企、民企） | 2周内 |
| 2 | 将沈阳机床等弱资质辽企列入观察名单 | 立即 |
| 3 | 对辽宁区域的新增敞口实行"一票否决"（仅可减仓不可增仓） | 即时生效 |
| 4 | 将辽企的信用评级内部下调一档（如外部AAA→内部AA+） | 立即 |
| 5 | 对辽宁敞口计提额外风险资本（区域风险附加因子1.2x） | 下个考核期 |

#### P3：回购渠道审查

| 审查项目 | 关注点 | 潜在风险信号 |
|---------|-------|-------------|
| 华晨债券在回购市场的折扣率 | 是否高于AAA篮子标准3% | 若折扣率上调至8%+，说明市场已不认可AAA评级 |
| 正回购对手方集中度 | 是否过度依赖单一银行/券商 | 一旦对手方收紧，再融资出现困难 |
| 回购期限 | 是否大量使用隔夜/7D短期滚动 | 期限错配加剧流动性风险 |
| 是否有GC001/R001滚动融资 | 融资集中度 | 若叠加市场资金紧张，滚动失败风险 |

#### P4：做空对冲（部分有效）

| 对冲工具 | 操作方式 | 有效性 | 注意事项 |
|---------|---------|-------|---------|
| 华晨中国(01114.HK)融券做空 | 借入股票卖出，信用恶化后买回 | **部分有效** | 港股权证流动性一般，融券成本约1%-3%/年 |
| 卖出华晨中国看涨期权 | 收取期权费对冲下行风险 | **有限** | 期权流动性差，但深度虚值期权可达一定保护 |
| 做空汽车行业ETF | 无法精确对冲华晨个体风险 | **弱** | 华晨异于行业其他标的 |

**融券做空的局限性：**
1. 华晨中国(01114.HK)是香港上市主体，非境内发债主体华晨汽车集团——两者信用质量相关但不相同
2. 股票价格反映股权价值，对母公司的债务违约敏感度有限
3. 港股融券需通过QFII/港股通，交易成本高

#### P5：信用衍生品（基本不可行，仅存档）

| 产品 | 可行性 | 原因 |
|------|-------|------|
| 银行间CRM（信用风险缓释合约） | **无** | CRM仅覆盖有限民营企业，无华晨 |
| 交易所信用保护工具 | **无** | 2018年11月刚启动试点，仅4家民企 |
| 定制CDS（OTC） | **理论可行但实际不可行** | 中国场外信用衍生品市场几乎不存在 |

## 三、预警指标体系

设计华晨信用恶化的先行预警指标体系，各指标触及阈值时自动触发响应机制：

### 3.1 先行指标（提前6-12个月）

| 指标 | 当前值（2018末） | 预警阈值 | 频率 |
|-----|----------------|---------|------|
| 母公司经营性净现金流 | -10.51亿元 | 连续两年为负（已是） | 年度 |
| 母公司短期债务/总债务 | 73% | >70%（已是） | 半年度 |
| 华晨中国股价（01114.HK） | ~8HKD | <7HKD（距年初-60%） | 每日 |
| 宝马增持对财报影响 | 2022年生效 | 模拟2020年利润下降幅度 | 事件触发 |

### 3.2 同步指标（提前3-6个月）

| 指标 | 预警阈值 | 触发行动 |
|-----|---------|---------|
| 评级展望从稳定调至负面 | 任何变动 | 立即启动减仓程序 |
| 债券二级成交收益率>8% | YTM突破8% | 强制减持50%以上 |
| 质押折扣率上调 | >5% | 审查全部回购交易 |
| 银行授信压缩 | 未使用额度<20亿 | 预备流动性应急方案 |
| 华晨中国跌破5HKD | 较当前再跌37% | 立即全面评估信用状况 |

### 3.3 滞后指标（确认恶化）

| 指标 | 含义 |
|-----|------|
| 年报中母公司净资产进一步为负 | 资不抵债深化 |
| 逾期债务（哪怕技术性） | 违约前最后信号 |
| 员工欠薪/停产 | 资金链断裂 |

## 四、应急方案

### 4.1 轻度情景（信用恶化）：缓释措施

```
目标：降低风险暴露，防止损失扩大
行动：
  1. 二级市场减仓30%-50%
  2. 暂停新投资
  3. 增加日常监测频率（周报改为日报）
```

### 4.2 中度情景（单一违约）：应急预案

```
目标：控制损失，启动回收程序
行动：
  1. 立即冻结华晨相关所有交易
  2. 启动法律追偿程序
  3. 参与债权人委员会（债委会）
  4. 与评级机构沟通确认评级影响范围
  5. 计提减值准备
  6. 通报管理层和风控委员会
  7. 评估区域传染风险——排查其他辽宁敞口
```

### 4.3 严重情景（区域连环违约）：危机应对

```
目标：保全组合，防范系统性冲击
行动：
  1. 全面排查辽宁区域所有敞口（信贷、债券、衍生品、回购）
  2. 评估对CET1比率的冲击和资本补充需求
  3. 向主管监管机构报告
  4. 启动紧急流动性安排（如有必要）
  5. 对策略投资者关系进行沟通
  6. 评估信用组合整体风险偏好的调整必要性
  7. 考虑对冲工具（国债期货多头）对冲系统性利率下行避险
```

## 五、高管层建议——总体判断摘要

尊敬的投委会/风控委员会：

站在2018年12月31日，我们建议对华晨汽车集团信用敞口采取**最高优先级的主动风险管理措施**。我们识别以下事实：

1. **母公司已技术性破产**——母公司净资产-39.46亿元，资产负债率107%，经营性现金流-10.51亿元，依赖借新还旧维持。

2. **利润来源被结构性削弱**——2018年10月宝马宣布将持股从50%提升至75%，2022年生效后华晨从合资公司获得的利润将直接减半。华晨已无任何手段阻止这一变化。

3. **评级严重滞后**——外部AAA评级与母公司的真实信用质量严重不符。评级报告依赖"合并报表"数据，而优质资产（华晨宝马）在子公司层面，母公司无法自由调动。

4. **对冲工具缺失**——中国市场缺乏有效的信用风险对冲工具，无法对华晨信用风险进行市场化转移。

5. **尾部风险不容忽视**——如果华晨违约，辽宁省其他弱资质国企（如沈阳机床）可能受到区域融资环境收紧的连锁冲击。

2018年中国AAA信用利差处于历史低位，华晨债券5.4%-6.3%的票面利率看似有吸引力，但这实质上是对一个**结构性问题**的风险补偿。我们建议：

- **最核心建议：** 实质性减仓，将华晨敞口降至组合净值的0.5%以下
- **底线建议：** 至少不再新增任何华晨多头敞口
- **如无法减仓：** 则必须缩短久期、设置硬性止损线、储备流动性预案

我们对华晨的买入/持有评级的信心为**极低**。在不存在有效对冲工具的情况下，持有华晨债券实质上是在承担无保护的信用风险。
