## 二、三身份并行评估

### 2.1 M0 信贷审批（银行分支行视角）

**评估框架**: 5步决策链：主体准入 -> 用途审查 -> 还款来源 -> 抵押担保 -> 定价条件 [来源: multi-stakeholder.md §3.1]
**核心约束**: FTP成本 + 不良率指标 + 抵押覆盖率 + 尽职免责 [来源: multi-stakeholder.md §1.1]

#### 第一步：主体准入——母公司负债率94.87%

**数据**：母公司层面资产负债率94.87%（2019年报），合并层面73.64%，两表差21.23个百分点。母公司现金/短债比仅0.1。

**分析**：
- 极端杠杆锁定。母公司负债率94.87%已经逼近资不抵债边缘。任何银行的信贷准入标准都不会接受负债率>85%的主体发放无担保信用贷款。
- 更严重的是两表之间的裂缝——母公司负债率比合并报表高出21个百分点，证明"母公司被掏空"的判断成立 [来源: holding-company-framework.md §2.2]。
- 母公司货币资金/合并货币资金比例极低，母公司账户上几乎无可用现金。

**初步判定**: 主体准入不通过。

#### 第二步：用途审查

**逻辑**：永煤控股作为控股母公司，自身无实际经营业务，融资用途主要为"补充流动资金"或"偿还存量债务"。

**问题**：
- 母公司有息负债高达764亿，短债占比约60%，每年利息支出庞大。
- 关联方占款104-225亿，资金实际被关联方占用，融资的最终用途不可追踪 [来源: holding-company-framework.md §2.9]。
- 三个化工子公司在母公司报表上已资不抵债，部分融资可能被用于填补这些子公司的窟窿。

**初步判定**: 融资用途不透明，存在资金挪用风险。

#### 第三步：还款来源——母公司自身造血能力极弱

**数据**：
- 母公司2019年归母净利润-13.17亿（2018年-11.44亿），连续两年亏损。
- 母公司自身经营性现金流（单体报表）持续为负，无造血能力。
- 还款完全依赖以下两者之一：a）核心子公司（永煤股份）分红回笼资金；b）借新还旧。

**风险分析**：
- 子公司分红没有法律强制力——母公司能否从子公司获得分红取决于子公司董事会决策 [来源: holding-company-framework.md §2.5]。
- 在母子公司资金关系出现裂痕时（关联占款规模如此之大往往意味着关系已恶化），分红回流可能中断。
- 核心子公司（永煤股份）经营虽正常，但其利润贡献是否足够覆盖母公司764亿有息负债的利息？显然不能。

**初步判定**: 第一还款来源不可靠。

#### 第四步：抵押担保——母公司层面无合格抵押物

**数据**：母公司资产负债表上的核心资产是对子公司的长期股权投资（主要是永煤股份股权）。

**分析**：
- 核心煤矿资产在法律上属于子公司（永煤股份），不在母公司层面。银行无法直接获取子公司的资产作为抵押。
- 如果抵押母公司持有的子公司股权：永煤股份是核心盈利主体，但其股权价值取决于子公司分红权，而分红权在压力情景下可能被子公司董事会限制。
- 集团层面的其他资产（化工子公司）已资不抵债，无抵押价值。

**交叉验证**：华晨案例中，核心资产同样在子公司（华晨宝马），母公司层面无有效抵押物 [来源: holding-company-framework.md §1.2]。

**初步判定**: 无可接受的合格抵押物。

#### 第五步：定价条件——如果被迫参与，最低条件

**假设银行出于某种原因（政府协调、银团联动等）必须参与**：

- 期限限制：最长1年（超短融级别），不允许中长期敞口。
- 抵押要求：必须提供足额抵押物（土地/房产/采矿权），不接受纯信用。
- 利率定价：基准利率 + 至少300bp风险溢价。原因：母公司隐含CCC级别的信用质量（外部评级AAA/稳定完全不可信），参照高风险主体的定价水平。
- 担保要求：增加永煤股份或其他核心子公司的连带责任担保（但需确认子公司担保的法律效力）。
- 交叉违约条款：必须纳入，且降低触发门槛。
- 单一银行敞口上限：建议不超过5亿元。

#### M0评估结论

| 决策项 | 结论 |
|--------|------|
| 主体准入 | ❌ 不通过（负债率94.87%超过银行内部硬上限） |
| 用途审查 | ❌ 不通过（资金被关联方占用，用途不透明） |
| 还款来源 | ❌ 不可靠（连续亏损，依赖借新还旧） |
| 抵押担保 | ❌ 母公司层面无合格抵押物 |
| 定价条件 | 仅接受：1年以内 + 足额抵押 + 利率基准+300bp + 子公司担保 + 交叉违约 |
| **最终结论** | **拒绝无担保信用贷款；如有外部压力，仅接受足额抵押贷款，敞口<5亿，期限<1年** |

### 2.2 M1 债券投资（固收基金经理视角）

**评估框架**: 四维框架（相对价值30% + 条款保护25% + 流动性20% + 事件日历25%）[来源: multi-stakeholder.md §2.1]
**核心约束**: 相对排名 + 最大回撤 + 流动性可得性 [来源: multi-stakeholder.md §1.1]

#### M1.1 相对价值评估（权重30%）

**定价偏差分析**：
- 永煤SCP003票面利率4.39%，看似高于同期AAA短融中位数（约3.0-3.5%）。但这是表面价值。
- 中债隐含评级AA（2020年6月），即将下调至AA-（2020年7月），意味着市场已提前定价风险。AA级短融中位数利差约200-300bp vs AAA，4.39%的票面实际上只是AAA+溢价后的合理水平，并无超额收益。
- 豫能化CP001票面5.50%——这一利率水平对于AAA国企而言已是显著偏高（AAA国企CP通常在3-4%区间）。豫能化作为永煤的母公司（河南能化），其定价本身就反映了市场对河南能源化工体系的风险定价 [来源: dual-track-methodology.md §11]。
- 偏离度分析：2020年7月后出现非市场化发行特征（票面锁定6%，偏离估值127bp），但在6月30日这一信号尚未完全暴露。

**同行业/同评级对比**：
- 煤炭行业AAA国企可比利差（2020年6月）：约100-150bp vs 国债。
- 永煤SCP003若按真实信用质量（隐含评级AA），可比利差应为200-300bp+。
- 4.39%的票面对应AAA评级虽低但不算异常，但对应AA隐含评级则偏低。

**相对价值评分**: 4/10 —— 表面上有AAA高评级提供的"廉价"融资成本，但隐含评级AA揭示的信用风险并未被票面充分补偿。没有超额利差，反而承担了AA级信用风险。

#### M1.2 条款保护评估（权重25%）

**SCP条款特征**：
- SCP（超短期融资券）为纯信用品种，**无担保、无抵押**。
- 无回售条款——持有人无法在信用恶化时提前退出。
- 交叉违约条款存在但门槛较高——通常要求其他债券违约金额超过一定阈值（如5,000万元）且经30日宽限期后方可触发。
- 无财务承诺条款——发行人无义务维持特定财务指标。
- 无资产出售限制——理论上发行人可以在债券存续期内将核心资产划转出去（这正是后续发生的！）。

**条款保护力的现实评估**：
- 在违约发生时，SCP持有人与其他一般无担保债权人处于同等受偿顺序，无优先权。
- 母公司核心资产（子公司股权）被划转时，债券持有人没有任何法律工具阻止 [来源: holding-company-framework.md §2.9]。
- 交叉违约条款在面对"公开市场违约"的场景下有一定保护力，但面对"资产划转掏空"的场景则完全无效。

**条款保护评分**: 1/10 —— 无担保纯信用 + 无回售 + 无财务承诺 + 交叉违约门槛高 + 无法阻止资产划转。条款保护几乎为零。

#### M1.3 流动性评估（权重20%）

**2020年6月流动性状态**：
- AAA国企债券在银行间市场有质押融资便利（标准券质押率通常较高），这是永煤债券的流动性"保护伞"。
- 但这一保护伞完全依赖于主体评级维持AAA。一旦评级下调或市场出现信用担忧，质押资格可能被取消或质押率大幅下调。
- 20永煤SCP003于2020年2月发行，至6月30日仅4个多月，二级市场成交不活跃，缺乏充分的定价发现。
- AAA国企的流动性是"虚假流动性"——貌似可以方便质押融资，但信用质量恶化时流动性会瞬间枯竭（永煤违约后市场验证了这一点）。

**流动性评估**: 5/10 —— 正常状态下有AAA国企质押便利，但这一流动性的前提（AAA评级）本身已经不可靠。在压力情景下，流动性将归零。

#### M1.4 事件日历评估（权重25%）

**未来3个月（2020年7月-9月）可能事件**：

| 时间 | 事件类型 | 事件内容 | 影响级别 | 数据可得性 |
|------|---------|---------|---------|-----------|
| 2020年7月 | 评级事件 | 中债隐含评级或从AA下调至AA- | 高 | 已有前兆信号 [来源: dual-track-methodology.md §11] |
| 2020年7月 | 公司事件 | 河南省国资委资产划转（非公开） | 极高 | 当时不可获取 [来源: 2026-07-07-blackswan-validation-yongmei.html] |
| 2020年8月 | 财务事件 | 2020年半年报发布 | 中 | 可预期 |
| 2020年8-9月 | 市场事件 | 信用债到期高峰，再融资压力检验 | 中高 | 公开信息 |
| 2020年9月 | 行业事件 | 煤炭价格走势 | 中 | 行业数据 |
| 2020年10月 | 公司事件 | 三季度运营数据 | 中 | — |

**年内到期压力**：母公司有息负债764亿中短债占60%（约458亿），滚动续作压力极大。每一期SCP/CP到期都是一次"信任投票"。

**事件日历评分**: 2/10 —— 隐含评级即将下调、再融资压力巨大、而且有未公开但同业间已有传闻的资产划转风险。事件风险集中在未来3个月。

**核心结论：回避。** 不买入，存量持仓建议在二级市场卖出（如果存在流动性的话）。相对价值不合理（4.39%不足以补偿AA级暗含的信用风险），条款保护为零，流动性在压力下会瞬间蒸发，事件日历上多重风险正在逼近。

### 2.3 M4 组合风控（风控总监视角）

**评估框架**: 集中度风险 + 压力测试 + 尾部风险 [来源: multi-stakeholder.md §5.2]
**核心约束**: 集中度限额 + 压力测试通过率 + 尾部风险管理 [来源: multi-stakeholder.md §1.1]

#### 2.3.1 单主体集中度——伪高评级风险

**参照框架**: 5.2.3 评级集中度 / 5.2.5 单主体集中度 [来源: multi-stakeholder.md §5.2]

永煤控股的外部评级AAA/稳定但内部引擎评分应远低于7.5（即实质AA-以下），触发**严重伪高评级**判定：
- 外部AAA / 内部评分<4.0 → 🔴严重伪高评级 —— 永煤/紫光/华晨违约前评级均为AAA但引擎评分均<4 [来源: multi-stakeholder.md §5.2.3]
- 单主体敞口：参照5.2.5单主体集中度规则，对于伪高评级主体，即便外部AAA，敞口占比也必须严格控制。

**规则应用**：
- 对于AAA伪高评级主体（永煤类），单主体敞口应不超过组合净值的**0.5%硬上限**。
- 保险机构单主体敞口不超过总资产的5%是监管要求，但对于永煤这类伪高评级主体，建议内部收紧至**1%以下** [来源: multi-stakeholder.md §5.2.5]。
- **不得为了"AAA评级"而给予AAA级的敞口额度。**

#### 2.3.2 区域集中度——河南区域风险

**参照框架**: 5.2.2 区域集中度 [来源: multi-stakeholder.md §5.2.2]

- 河南属于中部区域，非高风险区域（贵州/云南/辽宁）。但区域信用事件已在发酵：
  - 豫能化CP001票面5.50%（AAA国企偏高）——同一实际控制人体系下的风险已在债券定价中体现。
  - 如果永煤发生信用事件，河南区域信用利差将整体走阔。

**区域风险传导路径** [来源: multi-stakeholder.md §5.2.2]：
```
永煤信用事件
  └──► 河南省其他国企再融资受阻
    └──► 河南区域信用利差整体走阔
      └──► 组合中所有河南省敞口FV变动（即使未违约）
        └──► 组合净值回撤
```

**规则应用**：
- 单一省份敞口<10%为低风险（河南非高风险区域），但建议将河南敞口控制在**<5%**以防范系统性区域信用收缩。
- 若组合中已有豫能化敞口，则永煤+豫能化合计敞口应并入"河南能源化工体系"统一计算关联敞口。

#### 2.3.3 行业集中度——煤炭行业

**参照框架**: 5.2.1 行业集中度 [来源: multi-stakeholder.md §5.2.1]

- 煤炭行业在2016-2019年供给侧改革后盈利改善，但2020年上半年煤价已跌入红色区间 [来源: 2026-07-07-blackswan-validation-yongmei.html]。
- 区域风险>行业风险——永煤的违约更多是母公司层面的资金链断裂问题（关联占款、短债长投），而非煤炭行业周期问题 [来源: holding-company-framework.md §1.2]。
- 但行业下行周期会加剧母公司的现金流压力（子公司盈利下滑导致分红减少）。

#### 2.3.4 信用事件压力测试

**假设信用事件场景（2020年11月10日实际发生）**：

| 压力场景 | 描述 | 利差变动 | 对组合净值影响 |
|---------|------|---------|--------------|
| S1 轻度 | 隐含评级下调至AA-，利差走阔100bp | +100bp | 持仓市值下降约0.8-1.2%（久期0.8年） |
| S2 中度 | 评级下调至A，公开市场出现负面新闻 | +200bp | 持仓市值下降约1.6-2.4% |
| **S3 严重** | **实质性违约，回收率不确定** | **违约** | **本金损失敞口，回收率不确定** |

**对组合层面的影响**：
- 如果永煤敞口占比0.5%，S3场景下对组合净值的直接冲击为0.5% × (1 - 回收率)。假设回收率30%，则冲击为0.35%。
- 如果加上区域传染（河南其他债券利差走阔100bp），总冲击可能达到1-2%的组合净值。
- 对于持有永煤敞口超过0.5%的组合，冲击将被放大。

#### M4评估结论

| 维度 | 风险等级 | 判定 |
|------|---------|------|
| 单主体集中度 | 🔴 严重伪高评级 | 敞口硬上限0.5%，不得因AAA评级而放宽 |
| 区域集中度 | 🟡 关注 | 河南已出现信用分化，建议区域敞口<5% |
| 行业集中度 | 🟢 低-中 | 煤炭行业非主要风险来源，但下行周期加剧母公司压力 |
| 压力测试（S3） | 🔴 违约场景损失可控 | 若敞口控制在0.5%以内，冲击在容忍范围内 |

**组合风控结论**：
1. **永煤属于严重伪高评级主体**——外部AAA但实际信用质量在CCC级，必须按CCC而非AAA来设定敞口限额 [来源: multi-stakeholder.md §5.2.3]。
2. **单主体敞口硬上限0.5%**——对于任何信用评分<4.0（CCC及以下）的主体，敞口不得超过组合净值0.5%。
3. **区域敞口限制**——河南区域总敞口不超过5%，若河南能源化工体系已出现定价异常，则敞口应进一步压缩。
4. **建议行动**：检查组合中是否存在永煤敞口，若有则启动减仓至0.5%以下。河南区域其他国企敞口同步排查。

## 六、综合输出

### 6.1 核心判断

**永煤控股在2020年6月30日（违约前133天）已被公开数据充分识别为高风险主体。**

- 三个身份（M0信贷审批、M1债券投资、M4组合风控）独立评估后**高度一致**地指向负面结论。
- 不存在任何身份做出"可接受/建议参与"的判断——分歧仅存在于"拒绝方式"（M0: 拒绝贷款 / M1: 回避买入 / M4: 控制敞口），而非"是否应该拒绝"。
- **这一点比华晨案例更清晰**：华晨案例中M0有条件同意（仅限抵押贷款5亿上限1年），而永煤的M0直接拒绝无担保信用贷款。

### 6.2 预警窗口

| 指标 | 数值 |
|------|------|
| 分析时点 | 2020-06-30 |
| 首次违约日 | 2020-11-10（20永煤SCP003） |
| 从本分析到违约 | **133天（约4.5个月）** |
| 最早预警时点 | 2019-06-30（T-17个月，母公司负债率94.87%等四红一黄信号）[来源: dual-track-methodology.md §11] |
| 数据源限制 | 100%当时已公开数据 |

**即使在最保守的估计下（2020年6月30日才启动分析），预警窗口仍有133天，足以完成减仓/调整敞口。**

### 6.3 核心教训

1. **合并报表是最大的噪音源**。合并报表货币资金486亿看似充裕，但母公司现金/短债比仅0.1——只看合并报表会被完全误导。这是"子强母弱"结构最危险的陷阱 [来源: holding-company-framework.md §5.2]。

2. **AAA评级在"子强母弱"结构下不仅是滞后指标，更是误导指标**。中诚信AAA/稳定在违约前从未下调，而中债隐含评级早在违约前半年就从AA下调至AA-。外部评级在控股集团类主体面前彻底失效 [来源: dual-track-methodology.md §11]。

3. **SCP/CP等短期限品种的条款保护几乎为零**。无担保、无回售、无财务承诺、交叉违约门槛高——再加上资产划转条款缺失，SCP持有人在信用恶化时完全被动。

4. **国企信仰是三重噪音中最难穿透的一层**。评级噪音（AAA/稳定）+ 合并报表噪音（486亿货币资金）+ 国企信仰（河南国资委实控），三层噪音叠加使大量机构在违约前一周还在买入永煤债券 [来源: 2026-07-07-blackswan-validation-yongmei.html]。

5. **"伪高评级"是组合风控的最大盲区**。AAA评级在组合管理中通常对应较高的敞口权限，但对永煤类主体（外部AAA/内部CCC），必须切断评级与敞口限额之间的自动关联 [来源: multi-stakeholder.md §5.2.3]。

6. **多身份并行评估的价值**：M0的母公司报表穿透、M1的隐含评级偏离分析、M4的伪高评级规则——三个身份从不同角度切入，最终指向同一结论。单一身份的盲区被其他身份补偿，整体判断的鲁棒性显著高于任何单一分析。

### 6.4 分身份行动建议

| 身份 | 行动建议 |
|------|---------|
| **M0 信贷审批** | 拒绝永煤控股任何新增无担保信用贷款。如有政府协调压力，仅接受：① 全额抵押（土地/采矿权）；② 期限<1年；③ 基准+300bp；④ 永煤股份担保。存量贷款要求追加担保或压缩敞口。 |
| **M1 债券投资** | 不买入任何永煤控股存续或新发债券。如已持有，在二级市场流动性尚可时择机卖出。2020年7月隐含评级下调窗口是最后的减仓机会。 |
