## 一、行业金字塔框架概述

半导体行业采用 **五层金字塔结构**（政策驱动型 — 地缘政治-技术混合型/国家安全型），区别于标准的四层结构。行业分类依据引擎框架的十维评分体系（D1-D10）判定为政策驱动型（D3=5>=4, D4=3>=3），同时因外部依赖度极高（D7=5）被标注为跨越政策驱动型与技术壁垒型边界的特殊类别。

### 十维行业评分

| 维度 | 评分 | 说明 |
|------|------|------|
| D1 市场容量 | 5 | 全球4400亿美元+，万亿美元级生态 |
| D2 增量空间 | 5 | AI/HPC/IoT持续驱动增长 |
| D3 政策支持力度 | 5 | 国家级战略，大基金明确持续支持 |
| D4 政策波动性 | 3 | 持续性较强但校产改革等有结构性变动 |
| D5 资本可持续性 | 4 | 大基金+地方政府+科创板多元化融资 |
| D6 民生关联度 | 4 | 手机/汽车/数据中心/通信等广泛关联 |
| D7 外部依赖度 | 5 | 核心环节严重受制于人（光刻机/EDA/IP） |
| D8 供应链权力集中度 | 3 | 极少数链主（台积电/ASML/Applied Materials）垄断 |
| D9 产业生命周期 | 4 | 成长成熟期（进口替代空间大） |
| D10 周期波动性 | 3 | 半导体内存/代工有周期，设计公司相对平稳 |

### 五层金字塔结构

```
L1 地缘政治与供应链安全（30-35%） ← 最重层
    ↓ 通过
L2 技术壁垒与创新能力（25-30%）
    ↓ 通过
L3 市场位势与客户生态（15-20%）
    ↓ 通过
L4 政策准入与资本支持（10-15%）
    ↓ 通过
L5 财务质量与债务结构（5-10%；并购驱动型上调至15-20%） ← 验证层
    ↓
综合评级
```

**核心约束**:
1. 不可跳跃：每层结论是下一层的输入
2. 一票否决优先：任何层触发终止条件，整体上限CCC
3. L5只做验证不做颠覆：L5差但前四层好→标记风险但不直接降级；L5好但前四层差→更危险

## 二、验证案例：紫光集团回溯验证（T-17个月）

### 2.1 案例概况

| 项目 | 内容 |
|------|------|
| **验证标的** | 紫光集团（清华控股旗下半导体航母） |
| **回溯基准时点** | 2019年6月30日 |
| **违约事件** | 2020年11月16日，17紫光PPN005实质性违约 |

**框架有效性**: 通过。T-17个月即可判定高风险。

```

→ 扣除利息 65亿
→ 剩余 58亿
→ 远不足以覆盖资本开支 200亿+
```

## 五、关键发现

### 发现一：L5财务层在并购驱动型企业权重应上调

半导体行业标准模板中L5权重为5-10%，但对紫光集团这类通过大规模并购扩张的企业，财务风险在信用质量中的权重应提升至15-20%。商誉修正后的负债率（扣除商誉后负债率97.59%）是比原始资产负债率（73.42%）更具洞察力的核心指标。

### 发现二：L4政策层区分"产业支持"与"信用背书"是关键

大基金表态"全额支持"容易被市场误读为国家兜底，但实际上大基金注资进入子公司层面（长江存储、紫光展锐），无法用于偿还母公司1,652亿债务。L4框架区分了产业支持和信用背书——国家保了产业，没保集团的债券。

### 发现三：一票否决条件在此案例中全面触发

三个独立的一票否决条件同时触发：（1）商誉/总资产>20%且利息覆盖<2x；（2）扣除商誉后负债率>95%；（3）控股股东被迫剥离且转让失败。每一条件单独即可将评级上限锁定为CCC，三重触发形成"不可能误判"的信号。

### 发现四：评级机构失职——公开数据完整但未被正确解读

中诚信2019年6月18日维持AAA/稳定评级，同一份报告中列出了7大风险——资产负债水平上升较快、商誉减值风险、实控人变更风险、整合管理压力——全部列出，然后选择不下调。框架的核心优势在于以结构化方式强制对公开数据进行审计级审视，不受叙事干扰。

### 发现五：框架局限需诚实标注

（1）无法预测违约的具体时点——能判断"不可持续"，不能判断"什么时候断裂"；（2）商誉减值测试涉及管理层重大判断，外部分析无法验证其合理性；（3）无法自动捕获非标准收入项目异常（如中诚信报告中132亿顾问服务收入异常）。

## 六、框架修正建议

基于紫光集团回溯验证，对半导体行业分析框架提出以下修正：

| 修正项 | 原设定 | 修正后 | 依据 |
|--------|--------|--------|------|
| 并购驱动型企业L5权重 | 5-10% | 15-20% | 商誉调整后负债率是关键指标，财务风险在并购驱动型企业中权重应大幅提升 |
| 半导体行业特殊规则 | 无 | 新增"M&A驱动型企业L5权重提升至15-20%"规则 | 已纳入 `engine/industry-framework.md` v0.3.0 |
| L4一票否决条件 | 无 | 新增"控股股东被迫剥离且连续转让失败"条件 | 紫光案例推导 |
| L5一票否决条件 | 仅商誉/总资产>20%且利息覆盖<2x | 新增"扣除商誉后负债率>95%"条件 | 紫光案例推导 |

## 附录：推理链摘要

```
Step 1: 行业识别 → 半导体 → 加载半导体权重模板（30/25/15/15/15，并购驱动型修正）

Step 2: 数据采集
  → 2018年年报（2019年4月公开）→ 提取财务数据、商誉、利息覆盖
  → 校企改革政策（2018年5月）→ 清华剥离要求
  → 股权转让公告 → 两次失败
  → 大基金公开表态 → 注资子公司而非母公司

Step 3: 逐层分析
  L1（地缘政治）→ 中美科技战已升级，中兴已被制裁，YMTC设备依赖大 → 60分
  L2（技术壁垒）→ YMTC未量产，距3D NAND头部2-3代差距，但有技术积累 → 65分
  L3（市场位势）→ 新华三市场第一，ESL/FIT等子公司细分领先 → 75分
  L4（政策准入）→ 校企改革强制剥离；两次转让失败 → 40分
  L5（财务）→ 扣除商誉负债率98%；利息覆盖1.61x；扣非巨亏68亿 → 20分

Step 4: 综合判定
  最重两层（L5+L4）同时红灯 → 高风险
  三项一票否决条件触发 → 具体评级：CCC

Step 5: 建议
  → 降低或清零敞口
  → 持续监控：资产出售进度、大基金注资节奏、债务到期结构
```
