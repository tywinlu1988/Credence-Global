## 4. 方法论层面尚未覆盖的内容

排除"未编码"和"无用户反馈"后，以下为方法论概念框架层面的缺口：

### 4.1 评级体系的结构性缺口

| 缺口 | 影响 | 当前状态 |
|---|---|---|
| PD/LGD未分离 | 同一发行人不同债项（有担保vs无担保vs次级）共享同一评级，无法反映预期损失差异 | 12档方案已写入dual-track.md v0.4.0但未正式采用；LGD模块独立但未与PD整合 |
| 量化PD缺失 | 评级为排序而非量化违约概率；无法计算EL=PD×LGD，Basel IRB核心输出缺位 | 迁移矩阵作为方向参考标注，非精确PD |
| 无主权天花板 | 跨国/多国发行人分析失效；所有评级隐含假设中国主权评级为上限但未明确嵌入 | 完全未覆盖 |
| 无评级边界缓冲区 | 7.4分=BB、7.5分=BBB，0.1分差可导致评级跳跃 | 评级机构有缓冲区，引擎无 |

### 4.2 行业覆盖缺口

| 缺口 | 影响 | 当前状态 |
|---|---|---|
| 金融债空白（~15%市场） | 银行/券商/保险债信用逻辑（监管资本/Basel框架/MLF传导）与产业债完全不同，当前完全不可用 | 完全未覆盖，十维评分未扩展到金融行业 |
| 跨行业控股集团缺独立轨道 | 北大方正/清华紫光等只能通过"类比+迁移"方式分析，无形式化检测模板库 | 华晨"子强母弱"模式已验证可迁移但未建立模板 |
| 城投债框架未经验证 | LGV方法论骨架完成，但零案例验证；缺31省财政数据库、隐性债务估算模型、政策传导跟踪器 | v0.1.0骨架，距离可运行还需2-3轮迭代 |

### 4.3 风险覆盖面缺口

| 缺口 | 影响 | 当前状态 |
|---|---|---|
| 操作风险仅覆盖欺诈部分 | 内部流程缺陷、系统故障、法律合规风险均未定义检测方法论 | governance-fraud-risk.md仅覆盖财务欺诈和治理红旗 |
| 声誉风险和战略风险未嵌入评级调整 | 信号清单有（non-credit-risk-overlay.md）但无强制嵌入评级调整的规则 | 叠加层设计完成但调整规则为可选，非强制 |
| ESG因子未嵌入 | Moody's/S&P已全面嵌入ESG信用分析；环境违规/社会争议/治理风险是中国信用风险的重要前导指标 | 完全未覆盖 |
| 市场风险组合级方法缺失 | 组合级VaR/ES/经济资本无方法论定义；利率风险仅基础久期近似；汇率/商品风险传导已定义但组合层面无聚合 | M4组合风控规划为P2，尚未启动 |

### 4.4 统计与验证层面的方法论缺陷

| 缺口 | 影响 | 当前状态 |
|---|---|---|
| 幸存者偏差/数据挖掘偏差 | 3个验证案例均事后选择已知正确答案，未在已覆盖行业的已违约/未违约企业中做系统性回测 | 从未修复（T-1/T-2在统计审计中标记为高优先级但文档层也无法完全修复） |
| 情景概率无校准依据 | 压力测试概率（轻度20%/中度10%/严重3%/极端1%）标注为排序指标，无实证校准来源 | 标注了"排序指标"但未定义校准方法 |
| 展望预测准确率零验证展望 | 方法论文档完善（24条正面+22条负面触发因子），但展望方向正确性从未在案例中验证 | 仅在validation-methodology.md中规划了验证框架（4.5节） |
| 缓释建议未验证 | 10条缓释路径+发现→缓释映射规则（9.3节）文档完整，但零真实场景验证有效性和可操作性 | dual-track.md §9设计完成但零验证 |
| 多因子模型简化 | 24观测值对3-4因子OLS回归统计效力极低；单因子回归无法分解多因子共线效应；面板数据不可行 | 统计审计已有修正方案但代码未实现，小样本是根本性硬约束 |
